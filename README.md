# PACK-cattle-science

> **Тип репозитория:** `Pack`

**Pack (паспорт области) для предметной области «Наука о содержании и кормлении крупного рогатого скота»**

Source-of-truth для знаний о кормлении, воспроизводстве, экономике и здоровье КРС.

---

## Предметная область

**«Наука о содержании и кормлении КРС»** — инженерное описание принципов, методов и норм содержания молочного скота.

### Входит в область:
- Кормление (рационы, нормы, БМВД)
- Воспроизводство (осеменение, сухостой, отёл)
- Экономика (затраты, рентабельность, модели)
- Профилактика здоровья (метаболические заболевания)

### НЕ входит:
- Лечебная ветеринария
- Генетика и селекция
- Строительство ферм
