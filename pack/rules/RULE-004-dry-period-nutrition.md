# RULE-004: Dry Period Nutrition — Controlled Energy Approach

> **Тип:** nutrition-protocol  
> **Уверенность:** medium — основано на Litherland 2025 + Drackley 1999  
> **Источники:** CS.SOTA.036 (Litherland), CS.SOTA.055 (Drackley), CS.SOTA.029 (Aschenbach)

## Правило

### IF [Условия]

```
ALL of:
1. Период: Дальний сухостойный период (FO, день -60 до -25)
2. Порода: Голштины (Holstein)
3. BCS при сухостое: 3.0-3.5 (по 5-балльной шкале)
4. Ожидаемая продуктивность: >30 л/сут

OPTIONAL (факторы риска метаболических проблем):
- История метаболических заболеваний в прошлой лактации
- Возраст: мультипары (не первотёлки)
- Сезон: зима/весна (выше риск)
```

### THEN [Действие]

```
1. Рацион дальнего сухостойного периода (FO):
   - Ограничение энергии: 80-100% NRC NEL
   - Рекомендуется: 80NRC (ограниченный высокоэнергетический)
   - НЕ рекомендуется: 150NRC (ad libitum высокоэнергетический)

2. Контрольные точки:
   - BCS при вводе в сухостой: 3.0-3.25
   - BCS к отёлу: 3.25-3.5 (не выше!)
   - Прирост BCS в FO: не более 0.25 единицы

3. Физическая наполняемость:
   - Обеспечить объём корма (NDF >40%)
   - Длина волокна >2.5 см
   - Животные должны "работать" за корм

4. Рацион ближнего сухостойного периода (CU):
   - Можно ad libitum (влияние минимально по Litherland)
   - Постепенный переход к ДО рациону
   - Мониторинг DMI (цель: >13 кг/сут к отёлу)
```

### BECAUSE [Механизм]

```
Litherland 2025 (JDS, n=71, факторный дизайн 3×2):

Ключевые находки:
1. 80NRC (ограниченный FO) → глюконеогенез выше:
   - Пропионат: +71%, +37%, +73% vs аланин в день 1
   - Глюконеогенная способность выше в дни -14 и 28
   
2. Перекорм в FO (150NRC ad libitum) → снижение ферментов:
   - Пируваткарбоксилаза (PC) — ключевой фермент
   - Фосфоенолпируваткарбоксикиназа (PEPCK)
   
3. CU рацион (ближний сухостой) — минимальное влияние:
   - Эффекты FO доминируют
   - Можно ad libitum без негативных последствий

Drackley 1999 (биология):
- Накопление жира в печени в FO → проблемы в начале лактации
- Ограничение энергии → лучшая адаптация метаболизма

Aschenbach 2010 (глюконеогенез):
- Пропионат = 60-74% субстрата
- Способность печени к глюконеогенезу критична
- Ограничение в FO "тренирует" эту способность

Механизм:
Ограничение энергии в FO → 
  → небольшой энергетический дефицит → 
  → адаптация метаболических путей → 
  → сохранение/увеличение глюконеогенной способности → 
  → лучшая адаптация к лактации
```

### LIMITS [Границы применимости]

**Применимо (✅):**
- Мультипарные голштины
- Ожидаемая продуктивность >30 л
- BCS при сухостое 3.0-3.5
- Привязное/беспривязное содержание

**НЕ применимо / Требует адаптации (❌):**
- Первотёлки — другая физиология, BCS должен быть выше (3.25-3.5)
- Джерси — более чувствительны к энергетическому дефициту
- Низкопродуктивные стада (<25 л) — низкий риск метаболизма
- Экстремальные климатические условия — требуется корректировка

**Требует осторожности (⚠️):**
- BCS <2.75 — животное худое, ограничение рискованно
- BCS >3.75 — животное тучное, требуется строгое ограничение
- Двойня в анамнезе — повышенный риск

## Валидация

### На чём основано

| Источник | Дизайн | Ключевой вывод |
|----------|--------|----------------|
| Litherland 2025 | Field study, n=71 | 80NRC → лучший глюконеогенез |
| Dann 2006 | RCT (предшественник) | Основа для дизайна Litherland |
| Drackley 1999 | Review | Теоретическая основа |
| Aiello 1984 | Lab study | Глюконеогенез выше в начале лактации |

### Конфликт данных

**Традиционный подход:**
- "Корми максимально в сухостой, набирай BCS"
- Не подтверждено исследованиями

**Evidence-based (Litherland 2025):**
- Ограничение в FO → лучшие результаты
- Перекорм → снижение метаболической функции

**Практический компромисс:**
- 80-100% NEL в FO (не ad libitum высокоэнергетические)
- Контроль BCS (не перекармливать)

## Практическое применение

**Что делать сегодня:**
1. Оценить текущий BCS при сухостое
2. Если средний BCS >3.5 → пересмотреть рацион FO
3. Обеспечить физическую наполняемость (объём!)
4. Мониторинг: BCS при вводе в сухостой, при отёле

**Метрики успеха:**
- BCS при отёле 3.25-3.5 (не выше)
- BHB <1.2 mmol/L на 7 DIM (>80% коров)
- NEFA <0.7 mEq/L на 7 DIM
