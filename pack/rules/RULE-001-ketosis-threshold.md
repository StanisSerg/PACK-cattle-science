# RULE-001: Ketosis-Threshold-Invalidation

> **Тип:** threshold + action  
> **Уверенность:** medium — основано на 1 валидированном кейсе  
> **Источник:** [DL-001](../decisions/DL-001-bhb-threshold.md)

## Правило

### IF [Условия]

```
ALL of:
1. BHB > 1.2 mmol/L (плазменный, постпандриальный)
2. DMI < 80% от нормы (факт/норма по NRC)
3. Потеря BCS > 0.5 с момента отёла
4. Дни отёла < 30 (ранний постпартум)
5. Клинические признаки минимальны (нет ацетона в дыхании, нет атаксии)

OPTIONAL (усиливает уверенность):
- АСТ повышен (но <300 U/L)
- Глюкоза <3.0 mmol/L
- NEFA >0.7 mmol/L
```

### THEN [Действие]

```
1. НЕ применять гепатопротекторы как основное или единственное решение
   (Essentiale, Liv-52 и др. — могут использоваться как вспомогательные)

2. НЕ использовать симптоматическую терапию в изоляции
   (глюкоза в/в, пропиленгликоль без системных изменений)

3. Приоритет: коррекция системы
   а. Разделить ДО и сухостойных по разным рационам (если смешаны)
   б. Снизить энергетическую плотность рациона ДО
   в. Увеличить объём/частоту раздачи
   г. Обеспечить постоянный доступ к корму (не перерывы >2ч)

4. Мониторинг: повторный BHB через 7-10 дней

5. Если через 14 дней нет улучшения — пересмотреть диагноз
   (возможно, не метаболический дефицит, а инфекция/первичная патология)
```

### BECAUSE [Механизм]

```
При BHB >1.2 + низком DMI организм находится в состоянии 
метаболического дефицита:

1. Энергетические субстраты (глюкоза, жирные кислоты) 
   направляются на:
   - Базальный метаболизм (поддержание жизни)
   - Лактационный синтез (молоко)
   - Компенсацию дефицита

2. Печень не получает достаточно субстратов для:
   - Глюконеогенеза (синтез глюкозы)
   - Уреогенеза (детоксикация аммиака)
   - Желчного синтеза
   - Детоксикации общей

3. Гепатопротекторы работают только при наличии:
   - Энергетических субстратов (ATP для реакций)
   - Функциональных кофакторов (витамины, микроэлементы)
   - Нормальной клеточной мембраны (требует фосфолипидов)

При метаболическом дефиците все эти компоненты недоступны —
гепатопротекторы становятся «пустыми обещаниями».

Аналогия: 
Двигатель не заводится не потому что плохой бензин, 
а потому что нет бензина вообще. 
Присадки в бензин не помогут — нужно залить топливо.
```

### LIMITS [Границы применимости]

**Применимо (✅):**
- Голштины в привязном содержании
- Стандартные рационы постпартуриентных
- Ранний постпартум (7-30 дней отёла)
- Субклинический кетоз без осложнений

**НЕ применимо (❌):**
- Клинический кетоз с ацетоном в дыхании → требуется экстренная терапия
- Атаксия, неврологические признаки → интенсивная терапия
- АСТ >500 U/L → первичное поражение печени, требуется диагностика
- Поздний постпартум (>60 дней) → другой механизм (не метаболический дефицит)
- Сопутствующие инфекции (мастит, метрит) → комплексное лечение

**Требует осторожности (⚠️):**
- Джерси и другие мелкие породы (более чувствительны)
- Экстремальные климатические условия (>30°C или <-20°C)
- Роботизированное доение (другой паттерн поведения)
- Проблемы с кормом (плесень, загрязнение)

