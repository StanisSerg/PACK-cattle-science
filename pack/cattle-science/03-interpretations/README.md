# Интерпретации (Interpretations)

## Назначение

Папка `03-interpretations/` содержит агрегированные интерпретации сущностей на основе анализа множественных SoTA (State of the Art). В отличие от сущностей в `02-domain-entities/`, которые описывают конкретные понятия, интерпретации представляют собой **cross-SoTA анализ** — обобщение знаний из различных источников с выявлением паттернов, противоречий и интегративных инсайтов.

## Структура

### Категории интерпретаций

| Категория | Префикс | Содержание |
|-----------|---------|------------|
| **Метаболические** | CS.INT.MET.XXX | Интерпретации энергетического, липидного, углеводного метаболизма |
| **Репродуктивные** | CS.INT.REP.XXX | Интерпретации репродуктивной физиологии и управления |
| **Иммунные/Воспалительные** | CS.INT.IMM.XXX | Интерпретации иммунитета, воспаления, стресса |
| **Поведенческие** | CS.INT.BEH.XXX | Интерпретации поведения, благополучия, мониторинга |
| **Экономические** | CS.INT.ECO.XXX | Интерпретации экономической эффективности, оптимизации |
| **Технологические** | CS.INT.TEC.XXX | Интерпретации цифровых технологий, AI, автоматизации |

## Принципы создания интерпретаций

### 1. Cross-SoTA подход
- Минимум 2-3 SoTA для каждой интерпретации
- Сравнение методологий и выводов
- Выявление консенсуса и противоречий

### 2. Иерархия уверенности
| Уровень | Обозначение | Критерии |
|---------|-------------|----------|
| **Высокий** | ⭐⭐⭐ | Мета-анализ, RCT, консенсус экспертов |
| **Средний** | ⭐⭐ | Несколько независимых исследований |
| **Начальный** | ⭐ | Ограниченные данные, пилотные исследования |

### 3. Практическая применимость
- Каждая интерпретация должна содержать actionable insights
- Алгоритмы принятия решений
- Количественные пороги и критерии
