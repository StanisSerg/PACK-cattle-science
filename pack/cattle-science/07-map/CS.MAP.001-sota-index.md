## Применение в практике

### Для консультации по репродукции
1. Оценить текущий ВОС → CS.SOTA.001 для IEP
2. Оценить ED/точность → CS.SOTA.005 для выбора программы
3. Проверить здоровье свежих → CS.SOTA.004, 010-13 для рисков

### Для экономического анализа
1. Baseline: CS.SOTA.001 (современные данные)
2. Validation: CS.SOTA.002, CS.SOTA.003 (методология)
3. Sensitivity: CS.SOTA.005 (диапазон сценариев)

### Для кормления
1. Расчёт рациона: CS.SOTA.010-10 (CNCPS/AMTS)
2. Энергия: CS.SOTA.010-04 (NEL требования)
3. Протеин: CS.SOTA.010-05 (MP система)
4. Минералы: CS.SOTA.010-06 (DCAD, Ca, Mg)
5. Переходный период: CS.SOTA.010-13 (DCAD протоколы)

### Для обучения
- CS.SOTA.004, 010-04 — механизмы (почему)
- CS.SOTA.001, 010-10 — методология (как считать)
- CS.SOTA.005, 010-13 — практика (как делать)
