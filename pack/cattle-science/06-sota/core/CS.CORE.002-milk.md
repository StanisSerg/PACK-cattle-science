# CS.CORE.002: Молоко — фундаментальная сущность

> **Определение:** Молоко — питательная жидкость, продуцируемая молочной железой самок млекопитающих для питания потомства. У дойного скота — основной продукт.

---

## 1. Состав молока

### 1.1 Основные компоненты

| Компонент | % | Диапазон | Факторы вариации |
|-----------|---|----------|------------------|
| **Вода** | 87-88 | 85-89 | Гидратация, порода |
| **Сухое вещество (СВ)** | 12-13 | 11-15 | Обратно воде |
| **Жир** | 3.5-4.5 | 2.5-6.0 | Порода, стадия лактации, питание |
| **Белок** | 3.0-3.5 | 2.5-4.0 | Генетика, питание |
| **Лактоза** | 4.5-5.0 | 4.0-5.5 | Генетика, стадия лактации |
| **Минералы** | 0.7-0.9 | 0.6-1.0 | Питание, метаболизм |

## 2. Синтез молока

### 2.1 Анатомия вымени

| Структура | Описание | Функция |
|-----------|----------|---------|
| **Дольки (lobules)** | ~150-200 | Секреторные единицы |
| **Альвеолы** | 100-200 мкм | Синтез молока |
| **Протоки** | Сеть | Транспорт молока |
| **Цистерны** | Резервуары | Хранение |
| **Сосковые каналы** | 2-3 см | Выпуск молока |

### 2.2 Клетки альвеол

| Тип | Функция | Продукт |
|-----|---------|---------|
| **Секреторные эпителиоциты** | Синтез | Молоко |
| **Миоэпителиальные** | Сокращение | Экспульсия |
| **Эндотелиальные** | Барьер | Кровь ↔ молоко |

### 2.3 Механизмы синтеза

```
Субстраты из крови:
├── Глюкоза → Лактоза
├── Жирные кислоты → Триглицериды
├── Аминокислоты → Белки
├── Минералы → Ионный состав
└── Вода → Основа
```

**Энергетические затраты:**
- 1 кг молока (3.5% жира) = 0.75 Mcal NEL
- 1 г лактозы = 3.2 ккал
- 1 г белка = 4.0 ккал
- 1 г жира = 9.0 ккал

## 4. Качество молока

### 4.1 Показатели качества

| Показатель | Норма | Проблема |
|------------|-------|----------|
| **SCC** | < 100 тыс/мл | > 200 = мастит |
| **Бактерии** | < 10 тыс/мл | > 50 = проблема |
| **Термостабильность** | Положительная | Отрицательная = мастит |
| **Кислотность** | 16-18°T | > 20 = проблема |
| **Плотность** | 1.027-1.033 | Зависит от состава |

### 4.2 Факторы ухудшения

| Фактор | Механизм | Решение |
|--------|----------|---------|
| Мастит | Воспаление, бактерии | Профилактика, лечение |
| Недокорм | Мобилизация, кетоз | Коррекция рациона |
| Стресс | Кортизол, оксидативный | Комфорт, антиоксиданты |
| Неправильное доение | Травма сосков | Техника, оборудование |

## 7. Ключевые выводы

- Молоко — сложная биологическая жидкость с ~13% СВ
- Состав варьируется по стадии лактации, породе, питанию
- Синтез требует значительных энергетических затрат (0.75 Mcal/кг)
