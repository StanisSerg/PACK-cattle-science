# CS.CORE.004: Метаболизм — фундаментальная сущность

> **Определение:** Метаболизм — совокупность всех химических реакций в организме коровы, обеспечивающих поддержание жизни, рост, продуктивность и воспроизводство.

---

## 1. Энергетический метаболизм

### 1.1 Иерархия энергетических систем

```
GE (Gross Energy) — общая энергия корма
    ↓ минус фекалии
DE (Digestible Energy)
    ↓ минус моча + газы
ME (Metabolizable Energy)
    ↓ минус тепло
NE (Net Energy)
    ├── NEL — для лактации
    ├── NEM — для поддержания
    └── NEg — для прироста
```

### 1.2 Энергетический баланс

| Параметр | Формула | Норма |
|----------|---------|-------|
| **Поступление** | DMI × NEL рациона | Зависит от фазы |
| **Расход** | NEM + NEL milk + другие | Зависит от продуктивности |
| **Баланс** | Поступление − Расход | ±0 (средняя), −5..-10 (ранняя) |

### 1.3 Негативный энергетический баланс (НЭБ)

**Физиология НЭБ:**
```
Высокие требования (молоко)
    ↓
Недостаточное поступление (DMI лагирует)
    ↓
Мобилизация резервов (NEFA из жира)
    ↓
Либо: Адаптация (нормально)
Либо: Кетоз (патология)
```

**Стадии НЭБ:**

| Стадия | NEFA, mmol/L | BHBA, mmol/L | Состояние |
|--------|--------------|--------------|-----------|
| Норма | < 0.3 | < 0.6 | Нет мобилизации |
| Адаптация | 0.3-0.6 | 0.6-1.0 | Физиологическая |
| Риск | 0.6-1.0 | 1.0-1.2 | Мониторинг |
| Кетоз | > 1.0 | > 1.2 | Лечение |

## 2. Белковый метаболизм

### 2.1 Потоки протеина

```
Кормовой протеин (CP)
    ↓
├── RDP → Микробный протеин (в рубце)
└── RUP → Кишечник
    ↓
Микробный протеин + RUP = MP (Metabolizable Protein)
    ↓
Аминокислоты в крови
    ↓
├── Молоко
├── Ткани
└── Энергия (катаболизм)
```

### 2.2 Ключевые аминокислоты

| АК | Роль | Источник | Требование |
|----|------|----------|------------|
| **Lysine** | Молочный белок | Микробный, RUP | 7.2% MP |
| **Methionine** | Молочный белок, метилирование | Микробный, RUP | 2.4% MP |
| **Histidine** | Гистамин, рост | Микробный | 2.4% MP |
| **Leucine** | mTOR сигналинг | Микробный, RUP | Не ограничивает |

### 2.3 Баланс азота

| Параметр | Формула | Контроль |
|----------|---------|----------|
| **MUN** | Мочевина в молоке | 10-14 мг/дл |
| **BUN** | Мочевина в крови | 10-20 мг/дл |
| **Высокая** | > 18 | Избыток RDP |
| **Низкая** | < 8 | Недостаток RDP |

## 3. Минеральный метаболизм

### 3.1 Кальциевая регуляция

```
Низкий Ca в крови
    ↓
Паратиреоидная железа → PTH
    ↓
├── Кости: мобилизация Ca
├── Почки: реабсорбция Ca
└── Кишечник: активация витамина D → абсорбция Ca
    ↓
Нормализация Ca в крови
```

**Гормоны кальциевого метаболизма:**

| Гормон | Источник | Стимул | Действие |
|--------|----------|--------|----------|
| **PTH** | Паращитовидка | ↓Ca | ↑Ca (кости, почки, кишечник) |
| **Кальцитонин** | Щитовидка | ↑Ca | ↓Ca (кости) |
| **Витамин D** | Кожа/почки | PTH | ↑Абсорбция Ca |

### 3.2 Другие минералы

| Минерал | Регуляция | Функция |
|---------|-----------|---------|
| **Phosphorus** | PTH, FGF23 | Энергия, кости |
| **Magnesium** | Почки | Кофермент, PTH-секреция |
| **Sodium** | Альдостерон | Осмос, объём |
| **Potassium** | Инсулин, альдостерон | Осмос, сердце |

## 4. Гормональная регуляция

### 4.1 Гормоны метаболизма

| Гормон | Источник | Стимул | Действие |
|--------|----------|--------|----------|
| **Инсулин** | Поджелудочная | ↑Глюкоза | Анаболизм, липогенез |
| **Глюкагон** | Поджелудочная | ↓Глюкоза | Катаболизм, глюконеогенез |
| **GH** | Гипофиз | НЭБ | Липолиз, молоко |
| **IGF-1** | Печень | GH | Рост, анаболизм |
| **Лептин** | Жировая ткань | Жир | Сытость, репродукция |
| **Грелин** | Желудок | Голод | Голод, GH релиз |

### 4.2 Интеграция метаболизма

```
Ранняя лактация:
    НЭБ → ↓Инсулин, ↑GH, ↑Глюкагон
    → Липолиз (NEFA)
    → Глюконеогенез (пропионат)
    → Кетогенез (риск)

Средняя лактация:
    Баланс → Нормальные гормоны
    → Равновесие

Поздняя лактация:
    Позитивный баланс → ↑Инсулин
    → Липогенез (восстановление BCS)
```

## 5. Метаболические заболевания

### 5.1 Кетоз

**Механизм:**
```
НЭБ → Высокие NEFA → Печень
    ↓
    ├── β-окисление → Ацетил-КоА → Кетоновые тела (BHBA)
    └── Неполная окисление → Триглицериды → Жировой гепатоз
```

**Диагностика:**

| Метод | Порог | Применение |
|-------|-------|------------|
| BHBA кровь | > 1.2 mmol/L | Золотой стандарт |
| BHBA моча | > 15 мг/дл | Скрининг |
| Кетостик | +++ | Быстрый тест |
| Ацетон дыхание | Запах | Клинический |

### 5.2 Гипокальция

**Стадии:**

| Стадия | Ca, mmol/L | Симптомы |
|--------|------------|----------|
| Норма | 2.1-2.5 | Нет |
| Субклиническая | 1.8-2.0 | Вялость, мастит |
| Клиническая I | 1.4-1.8 | Стоячая, тремор |
| Клиническая II | < 1.4 | Лежащая, кома |

### 5.3 Жировой гепатоз

| Параметр | Норма | Гепатоз |
|----------|-------|---------|
| Триглицериды печени | < 5% | > 10% |
| Печёночная функция | Норма | Повышены ферменты |
| Связь | — | Часто с кетозом |
