# CS.CORE.003: Рубец — фундаментальная сущность

> **Определение:** Рубец (rumen) — первый отдел желудка жвачных, представляющий собой огромную ферментационную камеру с сложной микробной экосистемой. Центр пищеварения и метаболизма коровы.

---

## 1. Анатомия и физиология

### 1.1 Структура

| Параметр | Значение | Примечание |
|----------|----------|------------|
| **Объём** | 100-250 л | Зависит от размера коровы |
| **pH** | 6.0-6.8 | Критичен для микробов |
| **Температура** | 38-40°C | Оптимум ферментации |
| **Время прохождения** | 24-48 часов | Зависит от корма |
| **Площадь папилл** | 4-7 м² | Абсорбция VFA |

### 1.2 Отделы желудка жвачных

```
Корм входит:
    ↓
┌─────────────┐
│   РУБЕЦ     │ ← Ферментация (основной объём)
│  (Rumen)    │
└──────┬──────┘
       ↓
┌─────────────┐
│   СЕТКА     │ ← Фильтрация
│ (Reticulum) │
└──────┬──────┘
       ↓
┌─────────────┐
│   КНИЖКА    │ ← Дробление
│  (Omasum)   │
└──────┬──────┘
       ↓
┌─────────────┐
│   СЫЧУГ     │ ← Ферментативное пищеварение
│ (Abomasum)  │
└──────┬──────┘
       ↓
Тонкий кишечник
```

### 1.3 Жвачка и эруктация

| Процесс | Частота | Функция |
|---------|---------|---------|
| **Жвачка** | 50-70 раз/мин | Измельчение, смешивание со слюной |
| **Пары (boluses)** | 60-80/мин | Пережёвывание |
| **Время жвачки** | 6-10 часов/день | Зависит от NDF |
| **Эруктация** | 20-40/час | Удаление газов |

## 2. Микробиом рубца

### 2.1 Состав микробной популяции

| Группа | Количество/г | Функция |
|--------|--------------|---------|
| **Бактерии** | 10^10-10^11 | Основные ферментаторы |
| **Протозои** | 10^4-10^6 | Потребляют крахмал, бактерии |
| **Грибы** | 10^3-10^5 | Разложение целлюлозы |
| **Археи** | 10^8-10^9 | Метаногенез |

### 2.2 Ключевые бактерии

| Род | Субстрат | Продукт |
|-----|----------|---------|
| *Fibrobacter* | Целлюлоза | Ацетат |
| *Ruminococcus* | Целлюлоза | Ацетат, формиат |
| *Prevotella* | Крахмал, белок | Пропионат, аммиак |
| *Streptococcus* | Крахмал, сахара | Лактат |
| *Lactobacillus* | Лактат | ? (ацидоз) |
| *Methanobrevibacter* | H2 + CO2 | Метан |

### 2.3 Факторы, влияющие на микробиом

| Фактор | Оптимум | Эффект отклонения |
|--------|---------|-------------------|
| **pH** | 6.0-6.8 | < 6.0 = сдвиг популяции |
| **Температура** | 38-40°C | < 35°C = замедление |
| **Осмолярность** | 300-400 | > 500 = осмотический стресс |
| **Redox-потенциал** | -150 to -250 mV | Анаэробные условия |

## 3. Ферментация и продукты

### 3.1 Распад углеводов

```
Целлюлоза (NDF)
    ↓ Cellulases
Олигосахариды
    ↓
Глюкоза
    ↓ Glycolysis
Пируват
    ↓
    ├──→ Ацетат + CO2 + H2
    ├──→ Пропионат
    ├──→ Бутират
    └──→ Молочная кислота (при ацидозе)
```

### 3.2 Летучие жирные кислоты (VFA)

| VFA | % от общих | Энергия | Функция |
|-----|------------|---------|---------|
| **Ацетат** | 60-70 | 209 ккал/моль | Молоко (жир), энергия |
| **Пропионат** | 15-25 | 367 ккал/моль | Глюкоза (печень) |
| **Бутират** | 10-15 | 524 ккал/моль | Энергия эпителия |

**Соотношение VFA и питание:**

| Рацион | Ацетат | Пропионат | Бутират |
|--------|--------|-----------|---------|
| Высокий NDF (сено) | 70% | 18% | 12% |
| Сбалансированный | 65% | 22% | 13% |
| Высокий крахмал | 55% | 35% | 10% |

### 3.3 Микробный протеин

| Параметет | Значение |
|-----------|----------|
| Синтез | 130 г/кг переваримого СВ |
| Эффективность | 60-70% |
| Состав | 70-80% белок, 20-30% нуклеиновые кислоты |
| Профиль АК | Отличный (Lys 7.8%, Met 2.4%) |

**Условия для максимального синтеза:**
- Достаточно RDP (9-11% СВ)
- Достаточно NFC (36-40% СВ)
- pH 6.0-6.8
- Нет токсинов

## 4. Поглощение питательных веществ

### 4.1 Механизмы абсорбции

| Питательное вещество | Место абсорбции | Механизм |
|----------------------|-----------------|----------|
| **VFA** | Рубец (70%), кишечник | Пассивная диффузия |
| **Аммиак** | Рубец | Диффузия, транспортеры |
| **Минералы** | Рубец, кишечник | Активный транспорт |
| **Вода** | Рубец, кишечник | Осмос |

### 4.2 Папиллы рубца

| Характеристика | Значение |
|----------------|----------|
| Плотность | 20-40/см² |
| Длина | 2-10 мм |
| Площадь | 4-7 м² всего |
| Кровоснабжение | Интенсивное |

**Факторы, влияющие на папиллы:**
- Высокий VFA → рост папилл
- Низкий VFA (сухостой) → атрофия
- Резкие изменения рациона → повреждение

## 5. Нарушения функции рубца

### 5.1 Ацидоз (SARA - Subacute Ruminal Acidosis)

| Параметр | Норма | SARA |
|----------|-------|------|
| pH | 6.0-6.8 | 5.2-5.8 |
| Лактат | < 5 мМ | > 10 мМ |
| Пропионат | 20% | 30% |
| Ацетат | 65% | 50% |

**Причины:**
- Резкое увеличение концентрата
- Недостаток NDF
- Нерегулярное кормление

**Последствия:**
- Снижение DMI
- Ламинит
- Печёночный абсцесс
- Метаболические проблемы

### 5.2 Дисбактериоз

| Тип | Причина | Симптомы |
|-----|---------|----------|
| **Перегрузка крахмалом** | Быстрые углеводы | Вздутие, ацидоз |
| **Недостаток белка** | Низкий RDP | Снижение микробов |
| **Токсины** | Плесень, нитраты | Отравление |
