## 2. Потребление воды

### 2.1 Базовое правило

```
Вода (л/день) = (3-5) × Молоко (кг/день)
```

**Уточнение:**
- Низкая продукция (< 20 кг): 4-5 л/кг
- Средняя (20-40 кг): 3-4 л/кг
- Высокая (> 40 кг): 3 л/кг

### 2.2 Факторы, влияющие на потребление

| Фактор | Влияние | Механизм |
|--------|---------|----------|
| **Молоко** | +3-5 л/кг | Основной компонент молока |
| **DMI** | +3-5 л/кг СВ | Сопутствует потреблению корма |
| **Температура** | +20-30% при >25°C | Терморегуляция |
| **Na в рационе** | Пропорционально | Осмотический баланс |
| **K в рационе** | Пропорционально | Осмотический баланс |
| **Белок** | +10-15% | Выведение мочевины |
| **Соль** | +20-30% | Осмотический баланс |

### 2.3 Потребление по стадиям

| Стадия | Молоко (кг) | Вода (л/день) | Примечание |
|--------|-------------|---------------|------------|
| Сухостойные | 0 | 40-60 | Базовое |
| Свежие (7 DIM) | 20 | 80-100 | Начало лактации |
| Ранняя лактация | 35 | 120-150 | Пиковое |
| Средняя лактация | 40 | 130-160 | Максимум |
| Поздняя лактация | 25 | 90-110 | Снижение |

### 2.4 Потребление по массе тела

| Масса тела (кг) | Поддержание (л/день) | На 1 кг прироста |
|-----------------|----------------------|------------------|
| 100 (телята) | 8-12 | 6-8 |
| 300 (хеферы) | 25-35 | 4-6 |
| 500 | 40-50 | 3-4 |
| 650 | 50-60 | 2-3 |
| 800 | 60-75 | 2-3 |

## 3. Источники воды

### 3.1 Питьевая вода

**Основной источник:** 80-90% общего потребления

**Требования к доступу:**
| Параметр | Рекомендация |
|----------|--------------|
| Длина поилки | 7-10 см на голову |
| Поток | 10-15 л/мин |
| Высота | 70-90 см (взрослые) |
| Чистота | Ежедневная проверка |

### 3.2 Вода в кормах

| Корм | Влажность | Вода (кг/кг СВ) |
|------|-----------|-----------------|
| Сено | 15% | 0.18 |
| Силос | 65% | 1.86 |
| Силос кукурузный | 70% | 2.33 |
| Зелёная масса | 80% | 4.00 |
| Корнеплоды | 90% | 9.00 |

**Расчёт:**
```
Вода из кормов = DMI (кг СВ) × (Влажность% / (100 - Влажность%))
```

**Пример:** Корова потребляет 20 кг СВ силоса (65% влаги):
```
Вода = 20 × (65/35) = 37 л
```

### 3.3 Метаболическая вода

**Образование:** В результате метаболизма

| Субстрат | Вода (г/100 г) |
|----------|----------------|
| Крахмал | 60 |
| Жир | 110 |
| Протеин | 40 |

**Вклад:** 5-10% общей воды

## 4. Качество воды

### 4.1 Физические параметры

| Параметр | Оптимум | Допустимое | Проблема |
|----------|---------|------------|----------|
| **Цвет** | Прозрачная | Слабо окрашенная | Загрязнение |
| **Запах** | Отсутствует | Слабый | Бактерии, органика |
| **Вкус** | Нейтральный | Слабый | Минералы |
| **Температура** | 10-15°C | 5-25°C | Снижение потребления |

### 4.2 Химические параметры

| Параметр | Максимум | Риск при превышении |
|----------|----------|---------------------|
| **Нитраты (NO₃)** | 100 мг/л | Метгемоглобинемия |
| **Нитриты (NO₂)** | 10 мг/л | Острая токсичность |
| **Сульфаты (SO₄)** | 500 мг/л | Слабительный эффект |
| **Хлориды (Cl)** | 300 мг/л | Высокое потребление |
| **Жёсткость** | 300 мг/л CaCO₃ | Не критично |
| **pH** | 6.5-8.5 | Нарушение пищеварения |
| **TDS** | 3,000 мг/л | Снижение потребления |

### 4.3 Биологические параметры

| Показатель | Максимум | Риск |
|------------|----------|------|
| Общие колiformы | 200 КОЕ/мл | Инфекции |
| Fecal coliforms | 0 | Патогены |
| E. coli | 0 | Диарея |
| Протозои | 0 | Паразиты |

### 4.4 Токсичные вещества

| Вещество | Максимум | Симптомы отравления |
|----------|----------|---------------------|
| Синильная кислота | 0.01 мг/л | Асфиксия |
| Мышьяк | 0.05 мг/л | Отравление |
| Кадмий | 0.005 мг/л | Нефротоксичность |
| Свинец | 0.05 мг/л | Невротоксичность |
| Фтор | 2.0 мг/л | Флюороз |

## 5. Температура воды

### 5.1 Влияние на потребление

| Температура | Потребление | Комментарий |
|-------------|-------------|-------------|
| < 5°C | -30-40% | Ограничение |
| 5-10°C | -10-20% | Холодная |
| **10-15°C** | **100% (база)** | **Оптимум** |
| 15-20°C | -5-10% | Приемлемо |
| 20-25°C | -15-25% | Тёплая |
| > 25°C | -30-50% | Горячая |

### 5.2 Сезонные эффекты

| Сезон | Температура воды | Действие |
|-------|------------------|----------|
| Зима | 2-5°C | Подогрев до 10-15°C |
| Весна | 10-15°C | Оптимально |
| Лето | 20-25°C | Охлаждение, тень |
| Осень | 10-15°C | Оптимально |

### 5.3 Практические решения

| Проблема | Решение |
|----------|---------|
| Холодная вода (зима) | Подогрев до 10-15°C |
| Горячая вода (лето) | Трубы в тени, изоляция |
| Застойная вода | Циркуляция, ежедневная смена |

## 6. Нарушения водного баланса

### 6.1 Дегидратация

| Степень | Потеря веса | Симптомы |
|---------|-------------|----------|
| Лёгкая | 4-6% | Жажда, снижение молока |
| Умеренная | 6-10% | Сухие слизистые, вялость |
| Тяжёлая | > 10% | Коллапс, смерть |

**Признаки:**
- Сухость носа
- Втягивание глаз
- Снижение тургора кожи
- Тёмная моча

### 6.2 Перегидратация

**Причины:**
- Избыток Na без достаточной воды
- Почечная недостаточность
- Сердечная недостаточность

**Симптомы:**
- Отёки
- Диарея
- Нервозность

## 7. Практические рекомендации

### 7.1 Управление водой

**Ежедневно:**
- [ ] Проверка чистоты поилок
- [ ] Проверка температуры
- [ ] Мониторинг потребления

**Еженедельно:**
- [ ] Чистка резервуаров
- [ ] Проверка потока
- [ ] Тест на запах/вкус

**Ежемесячно:**
- [ ] Анализ качества воды
- [ ] Проверка системы подогрева/охлаждения
- [ ] Инспекция труб

### 7.2 Расчёт потребности

**Формула NASEM:**
```
Вода (л/день) = 14.3 + (1.47 × DMI, кг) + (0.58 × Молоко, кг) + 
                (0.91 × Na, г/день) + (1.05 × K, г/день) + 
                (0.33 × CP, г/день) + (0.11 × (T - 15°C))
```

Где T — температура окружающей среды

### 7.3 Примеры расчёта

**Пример 1: Ранняя лактация**
- DMI = 22 кг
- Молоко = 40 кг
- Na = 50 г
- K = 150 г
- CP = 3,800 г
- T = 20°C

```
Вода = 14.3 + (1.47×22) + (0.58×40) + (0.91×50) + 
       (1.05×150) + (0.33×3800/1000) + (0.11×5)
     = 14.3 + 32.3 + 23.2 + 45.5 + 157.5 + 1.25 + 0.55
     = 274 л/день
```

**Пример 2: Сухостойная**
- DMI = 13 кг
- Молоко = 0
- Na = 30 г
- K = 100 г
- CP = 1,800 г
- T = 10°C

```
Вода = 14.3 + 19.1 + 0 + 27.3 + 105 + 0.59 + 0
     = 166 л/день
```
