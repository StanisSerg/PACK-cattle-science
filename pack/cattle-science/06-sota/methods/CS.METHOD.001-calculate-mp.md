# CS.METHOD.001: Расчёт требования метаболического протеина (MP)

> **Назначение:** Расчёт дневной потребности коровы в метаболическом протеине (MP) для составления рациона.
>
> **Основан на:** [CS.SOTA.010-05 NASEM Protein](../feeding/CS.SOTA.010-05-nasem-protein.md)

---

## Цель

Определить количество MP (метаболического протеина), которое корова должна получать с рационом для поддержания продуктивности и здоровья.

---

## Необходимое оборудование

| Предмет | Назначение |
|---------|------------|
| Калькулятор или Excel | Расчёты |
| Данные по молоку | Удой, % белка |
| Весы для коровы | Живой вес |
| Таблица NASEM или приложение | Проверка норм |

---

## Входные данные

| Параметр | Обозначение | Единицы | Как измерить |
|----------|-------------|---------|--------------|
| Живой вес коровы | BW | кг | Весы или оценка |
| Удой молока | Milk | кг/день | Система учёта |
| Белок в молоке | MilkProt | % | Анализ молока |
| День лактации | DIM | дни | Записи |
| Прирост веса (если есть) | Gain | кг/день | Оценка |

---

## Формула расчёта

```
MP total = MP maintenance + MP milk + MP gain + MP pregnancy

Где:

MP maintenance = 2.2 × BW^0.75 (г/день)

MP milk = Milk × MilkProt × 0.68 (г/день)
    # 0.68 = эффективность конверсии протеина молока

MP gain = Gain × 320 (г/день)
    # Только для растущих животных

MP pregnancy = 0 (до 190 дня) → растёт до 400 г (последние 2 недели)
```

---

## Пошаговая инструкция

### Шаг 1: Расчёт MP для поддержания

```
MP maintenance = 2.2 × BW^0.75

Пример для коровы 650 кг:
MP maintenance = 2.2 × 650^0.75
               = 2.2 × 121.2
               = 267 г/день
```

### Шаг 2: Расчёт MP для молока

```
MP milk = Milk × MilkProt × 0.68

Пример: 35 кг молока, 3.2% белка
MP milk = 35 × 3.2 × 0.68
        = 35 × 2.176
        = 762 г/день
```

### Шаг 3: Расчёт MP для прироста (если применимо)

```
MP gain = Gain × 320

Пример: прирост 0.1 кг/день
MP gain = 0.1 × 320 = 32 г/день
```

### Шаг 4: Расчёт MP для беременности (если применимо)

| Дни беременности | MP pregnancy, г/день |
|------------------|----------------------|
| < 190 | 0 |
| 190-220 | 100 |
| 220-250 | 200 |
| > 250 | 300-400 |

### Шаг 5: Суммирование

```
MP total = 267 + 762 + 0 + 0 = 1029 г/день
```

## Пример полного расчёта

**Дано:**
- Корова: 650 кг, 80 DIM
- Удой: 38 кг/день
- Белок молока: 3.3%
- Прирост: 0 (взрослая)
- Беременность: нет

**Расчёт:**
```
MP maintenance = 2.2 × 650^0.75 = 283 г
MP milk = 38 × 3.3 × 0.68 = 853 г
MP gain = 0
MP pregnancy = 0

MP total = 283 + 853 = 1136 г/день
```

**Проверка по NASEM:**
- Норма для 650 кг, 38 кг молока: 1100-1200 г
- Расчёт в норме ✅

## Типичные ошибки

| Ошибка | Последствие | Исправление |
|--------|-------------|-------------|
| Заниженный вес | Недостаток MP | Взвесить корову |
| Пиковый удой вместо среднего | Избыток MP | Брать среднее за неделю |
| Забыть про прирост (нетели) | Недостаток MP | Учитывать для молодых |
| Не учесть беременность | Недостаток MP | Добавлять с 190 дня |
