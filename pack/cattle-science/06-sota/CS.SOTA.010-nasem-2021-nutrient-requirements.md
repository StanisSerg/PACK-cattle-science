## Ключевые разделы

### 1. Энергетические требования

| Параметр | Единицы | Применение |
|----------|---------|------------|
| NEL (Net Energy Lactation) | Mcal/kg | Основная энергетическая система |
| NEM (Net Energy Maintenance) | Mcal/kg | Для сухих коров и телок |
| NEg (Net Energy Gain) | Mcal/kg | Для растущих животных |

**Ключевые факторы:**
- Уровень молочной продуктивности
- Стадия лактации
- Масса тела
- Изменение ВМТ (BCS)

### 2. Протеиновые требования

| Параметр | Описание |
|----------|----------|
| MP (Metabolizable Protein) | Основная система для молочных коров |
| RUP (Rumen Undegradable Protein) | Бypass протеин |
| RDP (Rumen Degradable Protein) | Для микробиома рубца |

**Аминокислоты:**
- Лизин — первый лимитирующий
- Метионин — второй лимитирующий
- Гистидин — для высокопродуктивных

### 3. Макроминералы

| Минерал | Единица | Критичность |
|---------|---------|-------------|
| Кальций (Ca) | г/кг СВ | Переходный период |
| Фосфор (P) | г/кг СВ | Репродукция, метаболизм |
| Магний (Mg) | г/кг СВ | Профилактика гипомагниемии |
| Калий (K) | г/кг СВ | Баланс катионов-анионов |
| Натрий (Na) | г/кг СВ | Осмотический баланс |
| Хлор (Cl) | г/кг СВ | DCAD |
| Сера (S) | г/кг СВ | Синтез метионина |

### 4. Микроминералы

| Минерал | Функция | Риски дефицита/избытка |
|---------|---------|------------------------|
| Железо (Fe) | Транспорт кислорода | Редко дефицит у взрослых |
| Марганец (Mn) | Репродукция, кости | Недостаток → нерегулярные циклы |
| Цинк (Zn) | Иммунитет, кожа | Важен для свежих коров |
| Медь (Cu) | Ферменты | Взаимодействие с Mo, S |
| Йод (I) | Щитовидная железа | Критичен для телят |
| Селен (Se) | Антиоксидантная защита | Репродукция, мастит |
| Кобальт (Co) | Синтез B12 | Зависит от микробиома |

### 5. Витамины

| Витамин | Функция | Особенности |
|---------|---------|-------------|
| A | Зрение, иммунитет | Жирорастворимый |
| D | Кальциевый метаболизм | D3 для животных |
| E | Антиоксидант | Важен для свежих коров |
| K | Свертывание крови | Синтезируется микробиомом |
| B12 | Метаболизм | Зависит от Co |

### 6. Вода

- **Требование:** 3-5 кг воды на 1 кг молока
- **Качество:** Ограничения по нитратам, сульфатам, бактериям
- **Температура:** Влияет на потребление

## Связь с PACK-cattle-science

### Для CS.METHOD.001 (Protein norm calculation)

**Основа:**
- MP система (Metabolizable Protein)
- Расчёт RUP/RDP баланса
- Аминокислотная оптимизация

### Для CS.METHOD.002 (Energy requirement assessment)

**Основа:**
- NEL система
- Факторы корректировки
- Прогнозирование потребления

### Для переходного периода

**Критично:**
- Ca требования (профилактика гипокальции)
- DCAD (Dietary Cation-Anion Difference)
- Энергетическая плотность

### Для репродукции

**Связи:**
- BCS и энергетический баланс
- Минералы (Se, Zn) для фертильности
- Профилактика метаболических нарушений
