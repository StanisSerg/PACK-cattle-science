## 1. Body Condition Score (BCS)

### 1.1 Шкала BCS (1-5)

| BCS | Описание | Физическое состояние |
|-----|----------|---------------------|
| 1.0 | Очень худой | Кости видны, нет жира |
| 1.5 | Худой | Рёбра видны, минимум жира |
| 2.0 | Ниже среднего | Рёбра легко прощупываются |
| 2.5 | Слегка худой | Рёбра прощупываются |
| **3.0** | **Оптимум** | **Рёбра чувствуются с усилием** |
| 3.5 | Слегка полный | Рёбра едва чувствуются |
| 4.0 | Полный | Рёбра не чувствуются |
| 4.5 | Ожирение | Жировые отложения явные |
| 5.0 | Тяжёлое ожирение | Массивные отложения |

### 1.2 Методика оценки

**Зоны оценки:**
1. **Хвостовое седло** (tail head) — жировые отложения
2. **Крестец** (hooks/pins) — выступание костей
3. **Рёбра** — легкость прощупывания
4. **Поясница** (loin) — мышечная масса

**Процедура:**
- Оценка визуальная + пальпация
- Округление до 0.25 (например, 3.25, 3.5)
- Повторная оценка другим оператором для валидации

### 1.3 Целевые значения BCS

| Стадия | Целевой BCS | Допустимый диапазон |
|--------|-------------|---------------------|
| Закрытие | 3.25-3.50 | 3.0-3.75 |
| Отёл | 3.25-3.50 | 3.0-3.75 |
| 30 DIM | 2.75-3.25 | 2.5-3.5 |
| 100 DIM | 2.75-3.25 | 2.5-3.5 |
| 200 DIM | 3.0-3.5 | 2.75-3.75 |
| Сухостой | 3.25-3.75 | 3.0-4.0 |

## 2. Мобилизация жировых резервов

### 2.1 Физиология

**Триггеры мобилизации:**
- Негативный энергетический баланс (NEB)
- Низкий инсулин
- Высокий глюкагон
- Кортизол (стресс)

**Процесс:**
```
Триглицериды (жировая ткань)
    ↓ hormone-sensitive lipase
Глицерол + NEFA (свободные жирные кислоты)
    ↓ кровоток
Печень
    ↓
- β-окисление → энергия (ATP)
- Реэтерификация → триглицериды → VLDL
- Кетогенез → кетоновые тела (BHBA)
```

### 2.2 NEFA (Non-Esterified Fatty Acids)

**Нормы NEFA в сыворотке:**

| Период | Норма | Умеренная мобилизация | Тяжёлая мобилизация |
|--------|-------|----------------------|---------------------|
| Сухостой | < 0.3 мEq/л | 0.3-0.5 | > 0.5 |
| 0-7 DIM | < 0.7 мEq/л | 0.7-1.0 | > 1.0 |
| 7-30 DIM | < 0.5 мEq/л | 0.5-0.7 | > 0.7 |
| > 30 DIM | < 0.3 мEq/л | 0.3-0.5 | > 0.5 |

**Интерпретация:**
| NEFA | Риск метаболических | Действие |
|------|---------------------|----------|
| < 0.3 | Низкий | Поддерживать |
| 0.3-0.5 | Умеренный | Мониторинг |
| 0.5-0.7 | Высокий | Корректировка рациона |
| > 0.7 | Критический | Немедленное вмешательство |

### 2.3 BHBA (β-Hydroxybutyrate)

**Нормы BHBA:**

| Категория | BHBA (ммоль/л) | Интерпретация |
|-----------|----------------|---------------|
| Норма | < 1.0 | Оптимально |
| Граница | 1.0-1.2 | Внимание |
| Субклинический кетоз | 1.2-3.0 | Лечение |
| Клинический кетоз | > 3.0 | Срочное лечение |

**Связь NEFA → BHBA:**
- NEFA ↑ в печени → кетогенез ↑
- BHBA — основной кетоновый орган
- BHBA > 1.2 → риск для репродукции

## 3. Динамика BCS по лактации

### 3.1 Нормальная кривая

| DIM | BCS | Изменение | Комментарий |
|-----|-----|-----------|-------------|
| Отёл | 3.25-3.50 | — | Оптимум |
| 14 | 3.0-3.25 | -0.25 | Начало мобилизации |
| 30 | 2.75-3.0 | -0.5 | Пик мобилизации |
| 60 | 2.75-3.0 | -0.5 | Стабилизация |
| 100 | 2.75-3.25 | -0.25 | Восстановление |
| 200 | 3.0-3.5 | 0 | Норма |
| 305 | 3.25-3.75 | +0.25 | Подготовка к сухостою |
| Сухой | 3.25-3.75 | +0.25 | Поддержание |

### 3.2 Допустимые потери

| Период | Допустимая потеря BCS | Критическая потеря |
|--------|----------------------|-------------------|
| 0-30 DIM | 0.5-0.75 | > 1.0 |
| 0-60 DIM | 0.75-1.0 | > 1.25 |
| 0-100 DIM | 1.0 | > 1.5 |

**Последствия чрезмерной потери:**
- Репродуктивные проблемы
- Иммуносупрессия
- Мастит
- Метрит
- Задержка зачатия

### 3.3 Восстановление резервов

| Период | Целевое восстановление | Стратегия |
|--------|----------------------|-----------|
| 100-200 DIM | +0.25 BCS | Увеличение энергии |
| 200-305 DIM | +0.25-0.5 BCS | Позитивный баланс |
| Сухостой | +0.25 BCS | Поддержание |

**Скорость восстановления:**
- Оптимум: 0.25 BCS за 60-80 дней
- Максимум: 0.5 BCS за 100 дней
- Быстрее → риск ожирения

## 4. BCS и репродукция

### 4.1 Влияние BCS на фертильность

| BCS при отёле | Риск проблем | Рекомендация |
|---------------|--------------|--------------|
| < 2.75 | Высокий | Задержка осеменения |
| 2.75-3.0 | Умеренный | Внимание |
| **3.0-3.75** | **Оптимум** | Стандартный протокол |
| 3.75-4.0 | Умеренный | Контроль потерь |
| > 4.0 | Высокий | Риск гипокальции |

### 4.2 Потеря BCS и зачатие

| Потеря BCS (0-30 DIM) | CR1 | Дни открытых |
|----------------------|-----|--------------|
| < 0.5 | 45-50% | 110-120 |
| 0.5-0.75 | 40-45% | 120-130 |
| 0.75-1.0 | 35-40% | 130-150 |
| > 1.0 | < 35% | > 150 |

**Вывод:** Чем меньше потеря BCS, тем лучше репродукция.

### 4.3 NEFA и репродукция

| NEFA (0-7 DIM) | CR1 | Эмбриональная смертность |
|----------------|-----|-------------------------|
| < 0.3 | 45-50% | 15-20% |
| 0.3-0.5 | 40-45% | 20-25% |
| 0.5-0.7 | 35-40% | 25-30% |
| > 0.7 | < 35% | > 30% |

**Механизм:**
- Высокие NEFA → иммуносупрессия
- Кетоновые тела → токсичность для эмбриона
- Недостаток глюкозы → недостаток энергии для яйцеклетки

## 5. Управление BCS

### 5.1 Стратегия закрытия

**Цель:** Достичь BCS 3.25-3.50 к отёлу

| Текущий BCS (200 DIM) | Целевой BCS (отёл) | Стратегия |
|----------------------|-------------------|-----------|
| < 2.75 | 3.0-3.25 | Агрессивное кормление |
| 2.75-3.0 | 3.25-3.50 | Стандартное |
| 3.0-3.5 | 3.25-3.50 | Поддержание |
| 3.5-4.0 | 3.25-3.50 | Ограничение |
| > 4.0 | 3.25-3.50 | Строгое ограничение |

### 5.2 Стратегия сухостойного периода

**Принцип:** BCS не должен меняться в сухостой

| BCS при закрытии | Стратегия | Рацион |
|------------------|-----------|--------|
| < 3.0 | Небольшой gain | NEL 1.40-1.45 |
| 3.0-3.75 | Поддержание | NEL 1.30-1.40 |
| > 3.75 | Лёгкое ограничение | NEL 1.25-1.30 |

### 5.3 Стратегия ранней лактации

**Цель:** Минимизировать потерю BCS

| Фактор | Действие |
|--------|----------|
| DMI | Максимизация с первых дней |
| Энергия | NEL 1.70-1.80 Mcal/кг |
| Протеин | Достаточный MP |
| Минералы | Ca, Mg, DCAD |
| Мониторинг | BHBA, NEFA |

## 7. Мониторинг и диагностика

### 7.1 Протокол мониторинга

**Групповой уровень:**
| Показатель | Частота | Порог действия |
|------------|---------|----------------|
| Средний BCS при отёле | Ежемесячно | < 3.0 или > 3.75 |
| % с BCS < 2.75 | Ежемесячно | > 10% |
| % с BCS > 4.0 | Ежемесячно | > 10% |
| Средняя потеря BCS (0-30) | Ежемесячно | > 0.75 |
| Средний NEFA (0-7) | Квартал | > 0.5 |
| % BHBA > 1.2 (7-14) | Месяц | > 15% |

**Индивидуальный уровень:**
| Показатель | Когда | Действие |
|------------|-------|----------|
| BCS | Отёл, 30, 100, 200 DIM | Корректировка |
| NEFA | 0-7 DIM при риске | Лечение при > 0.7 |
| BHBA | 7-14 DIM всех | Лечение при > 1.2 |

### 7.2 Риск-факторы для мониторинга

| Фактор | Риск | Мониторинг |
|--------|------|------------|
| BCS > 3.75 при отёле | Гипокальция | Ca сыворотки |
| BCS < 2.75 при отёле | Кетоз, метрит | BHBA, NEFA |
| Первотёлки | Высокий кетоз | BHBA обязательно |
| Близнецы | Тяжёлый NEB | Усиленный |
| Зима | Кетоз | BHBA |
| Высокая продуктивность | NEB | NEFA, BHBA |
