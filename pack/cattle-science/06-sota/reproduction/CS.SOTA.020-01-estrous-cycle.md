# CS.SOTA.020-01: Эстральный цикл и репродуктивная физиология

> **Для чего эта глава:** Понимание эстрального цикла — основа управления репродукцией. Без знания гормональных механизмов невозможно эффективное осеменение и борьба с бесплодием.
>
> **Ключевая цифра:** Нормальный эстральный цикл длится 18-24 дня (среднее 21 день).

---

## Быстрый старт: что нужно знать

### Три главных правила цикла

| Правило | Почему важно | Что проверить |
|---------|--------------|---------------|
| **1. Овуляция через 24-32 часа после начала охоты** | Оптимальное время осеменения | Охота → осеменение через 12-24ч |
| **2. Цикл возобновляется через 15-20 дней после отёла** | Задержка = проблема | Мониторинг с 14 DIM |
| **3. НЭБ блокирует цикл** | Метаболизм важнее репродукции | BCS, NEFA при отёле |

### Ключевые цифры на память

```
Длительность цикла:     18-24 дня (среднее 21)
Длительность охоты:     12-18 часов
Время овуляции:         24-32 часа после начала охоты
Постpartum ановуляция:  10-20 дней
Целевой сервис период:  80-100 дней
Концепция:              40-50% (одно осеменение)
```

## Часть 1. Анатомия репродуктивной системы

### 1.1 Основные органы

| Орган | Функция | Ключевые характеристики |
|-------|---------|------------------------|
| **Яичники** | Продукция яйцеклеток, гормонов | 3-5 см, расположены краниально |
| **Матка** | Развитие эмбриона | Двурогая, тело 25-30 см |
| **Шейка матки** | Барьер, канал для спермы | Слизистая пробка при беременности |
| **Влагалище** | Копуляция, родовой канал | 25-30 см, pH 6.5-7.0 |
| **Вульва** | Наружный отверстие | Индикатор охоты (отёк, слизь) |

### 1.2 Яичники и фолликулярный аппарат

| Структура | Описание | Функция |
|-----------|----------|---------|
| **Фолликулы** | Пузырьки с жидкостью | Содержат ооциты |
| **Доминантный фолликул** | Крупнейший (> 10 мм) | Секреция эстрогена |
| **Тело жёлтое** | Регрессирующий фолликул | Секреция прогестерона |
| **Тело белое** | Регрессирующее желтое тело | Неактивно |

## Часть 2. Эстральный цикл

### 2.1 Фазы цикла

```
День 0 (Овуляция)
    ↓
Фолликулярная фаза (0-3 дня)
    ↓ Рост фолликула → Эстроген
День 18-20 (Охота/Эструс)
    ↓ LH пик → Овуляция
Лютеиновая фаза (4-17 дня)
    ↓ Тело жёлтое → Прогестерон
День 17-18 (Лизис желтого тела)
    ↓ Падение прогестерона
День 0 (Овуляция) — цикл повторяется
```

### 2.2 Гормональная регуляция

| Гормон | Источник | Функция | Пик |
|--------|----------|---------|-----|
| **GnRH** | Гипоталамус | Стимулирует LH/FSH | Каждые 1-2 часа |
| **LH** | Гипофиз | Овуляция, лютеинизация | Пик перед овуляцией |
| **FSH** | Гипофиз | Рост фолликулов | Базальный + пики |
| **Эстрадиол** | Фолликулы | Охота, LH пик | Пик перед охотой |
| **Прогестерон** | Жёлтое тело | Поддержка беременности | Лютеальная фаза |
| **Простагландин F2α** | Матка | Лизис желтого тела | День 17-18 |

### 2.3 Детальная хронология

| День | Событие | Гормоны | Признаки |
|------|---------|---------|----------|
| 0 | Овуляция | LH пик | — |
| 1-3 | Формирование желтого тела | ↑ Прогестерон | — |
| 4-16 | Лютеальная фаза | Высокий прогестерон | Нет охоты |
| 17-18 | Лизис желтого тела | ↓ Прогестерон | — |
| 18-20 | Фолликулярный рост | ↑ Эстрадиол | Признаки охоты |
| 20-21 | Охота/Эструс | Пик эстрадиола | Принятие самца |
| 21 | Овуляция | LH пик | — |

## Часть 3. Охота (Эструс)

### 3.1 Признаки охоты

| Признак | Интенсивность | Надёжность |
|---------|---------------|------------|
| **Принятие самца** | Высокая | 95%+ |
| **Отёк вульвы** | Средняя | 70% |
| **Слизистые выделения** | Средняя | 80% |
| **Беспокойство** | Средняя | 60% |
| **Прыжки на других коров** | Высокая | 85% |
| **Потеря аппетита** | Низкая | 40% |
| **Снижение удоя** | Низкая | 50% |

### 3.2 Время осеменения

| Протокол | Время | Эффективность |
|----------|-------|---------------|
| **AM-PM** | Утренняя охота → вечернее ОС, вечерняя → утреннее | 65-70% |
| **12-24 часа** | Через 12-24 часа после начала охоты | 60-65% |
| **Тimed AI** | По протоколу синхронизации | 50-60% |
| **Осмотр** | По размеру фолликула (УЗИ) | 70-75% |

### 3.3 Проблемы с обнаружением охоты

| Проблема | Частота | Решение |
|----------|---------|---------|
| **Тихая охота** | 10-15% | Хвостовые мелки, активити мониторы |
| **Ночная охота** | 30-40% | 2-3 проверки/день |
| **Короткая охота** | 5-10% | Частый мониторинг |
| **Ановуляция** | 15-25% postpartum | Прогестероновые протоколы |

## Часть 4. Постpartum период

### 4.1 Восстановление репродуктивной функции

| Процесс | Время | Факторы |
|---------|-------|---------|
| **Инволюция матки** | 30-40 дней | Отёл, метрит, мастит |
| **Первый фолликул** | 10-15 дней | НЭБ, BCS |
| **Первый цикл** | 20-40 дней | Питание, доение |
| **Первая охота** | 30-60 дней | BCS при отёле |
| **Первое осеменение** | 60-90 дней | Управленческое решение |

### 4.2 Факторы, задерживающие цикл

| Фактор | Механизм | Порог |
|--------|----------|-------|
| **НЭБ** | Глюкокортикоиды блокируют GnRH | NEFA > 0.7 mmol/L |
| **Низкий BCS** | Недостаток резервов | BCS < 2.5 |
| **Кетоз** | Метаболический стресс | BHBA > 1.2 mmol/L |
| **Метрит** | Воспаление, токсины | Любой метрит |
| **Мастит** | Системное воспаление | Клинический |
| **Ламинит** | Боль, стресс | Хромота |

## Часть 5. Практические кейсы

### Кейс 1: Задержка первой охоты

**СИТУАЦИЯ:**
- Ферма: 180 дойных
- Проблема: Средняя день первого осеменения 95 DIM (цель 75)
- Много коров без охоты до 60 DIM

**АНАЛИЗ:**
- BCS при отёле: 3.0 (норма)
- NEFA 7 DIM: 0.8 mmol/L (высокий)
- BHBA: 1.5 mmol/L у 20% (субклинический кетоз)
- Метриты: 15%

**Причина:** Сильный НЭБ + метаболические проблемы блокируют цикл

**РЕШЕНИЕ:**

**Шаг 1:** Улучшение питания transition
- DCAD для профилактики гипокальции
- Контроль BCS при отёле
- Мониторинг метаболитов

**Шаг 2:** Лечение метаболических проблем
- Кетоз: пропиленгликоль
- Метриты: раннее выявление и лечение

**Шаг 3:** Экспресс-осеменение
- Протокол ovsynch для коров без охоты к 70 DIM

**РЕЗУЛЬТАТ:**
- День первого ОС: 95 → 78 DIM
- Концепция: +12%

**УРОК:**
> Репродукция не работает при метаболическом стрессе. Начинайте с transition period.

---

### Кейс 2: Плохая концепция

**СИТУАЦИЯ:**
- Ферма: 250 дойных
- Проблема: Концепция 35% (норма 45-50%)
- Много повторных осеменений

**АНАЛИЗ:**
- Время осеменения: не оптимальное (через 8-36 часов)
- BCS при осеменении: 2.4 (низкий)
- Техника ОС: не проверялась

**РЕШЕНИЕ:**

**Шаг 1:** Обучение персонала
- AM-PM правило
- Признаки охоты
- Техника осеменения

**Шаг 2:** Улучшение BCS
- Цель: 2.75-3.0 при осеменении
- Корректировка рациона

**Шаг 3:** Мониторинг
- Хвостовые мелки
- Проверка техники ОС

**РЕЗУЛЬТАТ:**
- Концепция: 35% → 48%
- Дни открытые: 145 → 115

**УРОК:**
> Концепция — результат многих факторов. Проверяйте всё: технику, время, BCS.

## Часть 6. Чек-листы

### Чек-лист 1: Оценка охоты

| № | Признак | Наблюдается | Интенсивность |
|---|---------|-------------|---------------|
| 1 | Принятие самца | ☐ Да / ☐ Нет | 1-5 |
| 2 | Отёк вульвы | ☐ Да / ☐ Нет | 1-5 |
| 3 | Слизистые выделения | ☐ Да / ☐ Нет | Прозрачная/Мутная |
| 4 | Прыжки на других | ☐ Да / ☐ Нет | Часто/Редко |
| 5 | Беспокойство | ☐ Да / ☐ Нет | Высокое/Низкое |

**Интерпретация:**
- 4-5 признаков + принятие = Яркая охота → ОС через 12-24ч
- 2-3 признака = Слабая охота → ОС сразу

### Чек-лист 2: Мониторинг postpartum

| DIM | Действие | Цель |
|-----|----------|------|
| 14 | Проверка циклов | > 30% должны показать признаки |
| 30 | Оценка BCS | Не ниже 2.5 |
| 50 | Анализ метаболитов | NEFA < 0.3, BHBA < 1.0 |
| 60 | Первое осеменение | Если охота наблюдалась |
| 70 | Ovsynch | Если нет охоты |

## Ключевые выводы

- Эстральный цикл: 18-24 дня, среднее 21 день
- Овуляция: 24-32 часа после начала охоты
- Осеменение: оптимально через 12-24 часа
- Postpartum: цикл возобновляется через 20-40 дней
- НЭБ и метаболические проблемы блокируют цикл
- BCS при осеменении: 2.75-3.0 оптимально
