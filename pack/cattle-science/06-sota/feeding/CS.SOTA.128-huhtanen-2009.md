# 1. РЕЗЮМЕ (Abstract)

Классический мета-анализ 86 исследований о влиянии концентрации протеина (CP) и его деградабельности (RDP/RUP) на удой молочного протеина и азотную эффективность. Оптимум CP: 16.5-17.5% для максимального удоя протеина.

---

# 2. KEY CLAIMS (ключевые утверждения)

## Claim 1: Оптимум CP для молочного протеина
- **Утверждение:** Молочный протеин достигает максимума при CP 16.5-17.5%; дальнейшее повышение CP не увеличивает удой, но снижает N-эффективность
- **Evidence:** Мета-анализ 86 исследований; нелинейная модель; breakpoint analysis
- **Confidence:** high — большая выборка, консистентный эффект
- **Цитата:** "Milk protein yield was maximized at dietary CP concentrations between 16.5 and 17.5%, with no further response to additional protein"

## Claim 2: Деградабельность важнее концентрации
- **Утверждение:** При фиксированном CP, повышение RUP с 32% до 40% увеличивает молочный протеин на 3.5%, в то время как повышение CP с 15% до 18% дает прирост 8%
- **Evidence:** Meta-regression; разделение эффектов CP vs RDP/RUP
- **Confidence:** high — разделение компонент статистически значимо

## Claim 3: Азотная эффективность снижается при высоком CP
- **Утверждение:** N-эффективность (молочный N / потребленный N) падает с 32% (при 15% CP) до 25% (при 19% CP)
- **Evidence:** Расчеты баланса азота; корреляция с MUN
- **Confidence:** high — линейный тренд (p<0.001)

# 5. ПРАКТИЧЕСКОЕ ПРИМЕНЕНИЕ

## 5.1. Рекомендации по формулированию

```
1. Целевой CP: 16.5-17.0% для высокопродуктивных
2. RUP: 36-38% от CP (достаточно для максимума)
3. MUN: 12-14 мг/дл как индикатор
4. При CP >18%: снижение эффективности без прироста продукции
```

# 6. ВЫВОДЫ

## 6.1. Ключевые выводы

1. Оптимум CP: 16.5-17.5% для молочного протеина
2. RUP важнее RDP (коэффициент 1.87:1)
3. Высокий CP снижает N-эффективность
4. MUN — хороший индикатор баланса протеина

## 6.2. Историческая ценность

- Фундамент для NRC 2012
- База для современных систем (CNCPS)
- Подтверждено Hanigan 2024
