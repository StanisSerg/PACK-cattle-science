## 1. Анатомия желудочно-кишечного тракта

### 1.1 Структура ЖКТ взрослой коровы

| Отдел | Объём (л) | Функция | pH |
|-------|-----------|---------|-----|
| **Рубец** | 100-250 | Ферментация | 5.5-7.0 |
| **Сетка** | 5-20 | Сортировка | 5.5-7.0 |
| **Книжка** | 10-20 | Дегидратация | 6.0-7.0 |
| **Сычуг** | 10-20 | Секреция ферментов | 2.0-3.5 |
| **Тонкий кишечник** | 30-50 | Пищеварение, абсорбция | 7.0-8.0 |
| **Толстый кишечник** | 20-40 | Ферментация, абсорбция воды | 6.5-7.5 |

### 1.2 Рубец (Rumen)

**Анатомия:**
- Вместимость: 100-250 л (зависит от размера коровы)
- Температура: 39-40°C
- Влажность: 85-95%
- Микробная биомасса: 10-15% объёма

**Слои стенки:**
1. **Эпителий** — поглощение VFA
2. **Собственная пластинка** — кровоснабжение
3. **Мышечный слой** — контракции (1-3/мин)
4. **Сероза** — защита

**Папиллы рубца:**
- Увеличивают площадь абсорбции в 10-15 раз
- Развитие зависит от рациона (концентраты → больше папилл)

## 2. Микробиом рубца

### 2.1 Состав микробиома

| Группа | Численность (клеток/мл) | Функция |
|--------|------------------------|---------|
| Бактерии | 10^9 - 10^11 | Основная ферментация |
| Протозои | 10^4 - 10^6 | Утилизация крахмала |
| Грибы | 10^3 - 10^5 | Разрушение целлюлозы |
| Археи | 10^7 - 10^9 | Метаногенез |

### 2.2 Бактерии по функциям

| Группа | Представители | Субстрат | Продукты |
|--------|---------------|----------|----------|
| **Целлюлолитические** | Fibrobacter, Ruminococcus | Целлюлоза | Ацетат, формиат |
| **Амилолитические** | Streptococcus, Prevotella | Крахмал | Пропионат, лактат |
| **Протеолитические** | Prevotella, Ruminobacter | Протеин | Аминокислоты, аммиак |
| **Липолитические** | Anaerovibrio, Butyrivibrio | Липиды | Жирные кислоты |
| **Метаногены** | Methanobrevibacter | H2 + CO2 | Метан |

### 2.3 Факторы, влияющие на микробиом

| Фактор | Оптимум | Влияние при отклонении |
|--------|---------|------------------------|
| pH | 6.0-6.8 | < 6.0 → снижение целлюлолиза |
| Температура | 39°C | < 38°C → замедление |
| Осмолярность | 300-350 mOsm | > 400 → ингибирование |
| Окислительно-восстановительный потенциал | -150 до -250 mV | > -100 → анаэробы погибают |

## 3. Ферментация в рубце

### 3.1 Разрушение углеводов

#### Целлюлоза → VFA

```
Целлюлоза
    ↓ целлюлазы
Глюкоза
    ↓ гликолиз
Пируват
    ↓ ферменты рубца
Ацетат + Propionate + Butyrate + CO2 + CH4
```

**Скорость разрушения:**
| Углевод | Скорость | Пример корма |
|---------|----------|--------------|
| Сахара | Очень быстрая (1-2 ч) | Сахарная свёкла |
| Крахмал | Быстрая (2-4 ч) | Зерно |
| Растворимая клетчатка | Средняя (4-8 ч) | Пектины |
| Нерастворимая клетчатка | Медленная (12-48 ч) | Сено |

### 3.2 Доля VFA в общей энергии

| VFA | % от общих VFA | Энергетическая ценность |
|-----|----------------|------------------------|
| **Ацетат** | 60-70% | 209 kcal/моль |
| **Пропионат** | 15-25% | 367 kcal/моль |
| **Бутират** | 10-15% | 524 kcal/моль |

**Соотношение ацетат:пропионат:**
| Рацион | Соотношение | Эффект |
|--------|-------------|--------|
| Грубые корма | 4:1 | Высокий жир молока |
| Концентраты | 2:1 | Высокий белок, риск ацидоза |
| Оптимум | 2.5-3:1 | Баланс |

### 3.3 Метаногенез

**Реакция:**
```
CO2 + 4H2 → CH4 + 2H2O
```

**Потери энергии:**
- 2-12% валовой энергии рациона
- 25-30% энергии ферментации

**Факторы, влияющие на метан:**
| Фактор | Эффект |
|--------|--------|
| Концентраты | Снижение метана |
| Жиры | Снижение метана |
| Nitrate | Снижение метана |
| 3-NOP | Снижение метана (ингибитор) |

## 4. Протеиновый метаболизм в рубце

### 4.1 Деградация протеина

**Схема:**
```
Протеин корма
    ↓ протеазы
Пептиды
    ↓ пептидазы
Аминокислоты
    ↓ дезаминазы
Аммиак + VFA (углеродный скелет)
```

**Скорость деградации (fraction/hour):**
| Протеин | Класс | Скорость |
|---------|-------|----------|
| Растворимый | A | 0.30-0.40 |
| Быстро деградируемый | B1 | 0.10-0.40 |
| Медленно деградируемый | B2 | 0.02-0.10 |
| Недеградируемый | B3 | 0.001-0.02 |
| Связанный с NDF | C | 0 |

### 4.2 Синтез микробного протеина (MCP)

**Условия для максимального синтеза:**
1. Достаточно RDP (азот)
2. Достаточно энергии (углеводы)
3. Оптимальный pH (6.0-6.8)
4. Достаточно микроэлементов (Co, S)

**Эффективность синтеза:**
```
MCP (г/день) = 0.13 × TDN (кг/день)
```

Или через энергию:
```
MCP (г/день) = 130 × NEL (Mcal/день)
```

**Максимальная эффективность:**
- Теоретическая: 10 г MCP/Mcal NEL
- Фактическая: 8-9 г MCP/Mcal NEL

### 4.3 Аммиак в рубце

| Концентрация | Статус |
|--------------|--------|
| < 5 мг/дл | Дефицит для MCP |
| 5-25 мг/дл | Оптимум |
| > 25 мг/дл | Избыток, риск токсичности |

**Источники аммиака:**
- Деградация RDP (80-90%)
- Слюна (10-20%)
- Рециркуляция мочевины

## 5. Липидный метаболизм

### 5.1 Гидролиз липидов

**Липазы рубца:**
- Бактериальные липазы
- Растительные липазы (активны при pH 5.5-6.5)

**Продукты:**
- Глицерол → ферментация в VFA
- Жирные кислоты → абсорбция или биогидрогенация

### 5.2 Биогидрогенация

**Ненасыщенные → насыщенные жирные кислоты:**
```
Линолевая (C18:2) → Стеариновая (C18:0)
```

**Степень биогидрогенации:** 70-95%

**Последствия:**
- Жир молока более насыщенный
- Менее здоровый профиль для человека

## 6. Абсорбция питательных веществ

### 6.1 VFA (летучие жирные кислоты)

**Место абсорбции:**
- 70-80% — рубец
- 15-20% — оmasum (книжка)
- 5-10% — толстый кишечник

**Механизм:**
- Пассивная диффузия (недиссоциированные)
- Активный транспорт (диссоциированные)

**Эффективность:**
- Ацетат: 70-80%
- Пропионат: 80-90%
- Бутират: 90-95%

### 6.2 Микробный протеин (MCP)

**Место:** Тонкий кишечник

**Усвояемость:** 75-80%

**Механизм:**
- Панкреатические протеазы
- Пептидные транспортёры
- Аминокислотные транспортёры

### 6.3 Минералы

| Минерал | Место абсорбции | Механизм |
|---------|-----------------|----------|
| Na, K | Тонкий кишечник | Активный транспорт |
| Ca | Тонкий кишечник | Активный (D-зависимый) |
| P | Тонкий кишечник | Активный |
| Mg | Тонкий кишечник | Пассивный |
| Fe | Тонкий кишечник | Активный |

## 7. Проходжение пищи по ЖКТ

### 7.1 Время транзита

| Отдел | Время | Комментарий |
|-------|-------|-------------|
| Рубец | 12-48 часов | Зависит от рациона |
| Оmasum | 2-6 часов | Дегидратация |
| Сычуг | 2-6 часов | Кислое пищеварение |
| Тонкий кишечник | 3-6 часов | Основная абсорбция |
| Толстый кишечник | 12-24 часов | Ферментация, вода |

**Общее время:** 2-4 дня

### 7.2 Размер частиц

| Отдел | Максимальный размер | Механизм отбора |
|-------|---------------------|-----------------|
| Рубец | 1-2 мм | Контракции, плотность |
| Сетка | 1-2 мм | Фильтрация |
| Оmasum | 3-5 мм | Сжатие |
