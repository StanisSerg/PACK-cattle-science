## 1. Система Metabolizable Protein (MP)

### 1.1 Определение

**MP** — протеин, доступный для абсорбции в тонком кишечнике:
```
MP = MCP + RUP + END
```

Где:
- **MCP** — Microbial Crude Protein (микробный протеин)
- **RUP** — Rumen Undegradable Protein (непереваримый в рубце)
- **END** — Endogenous protein (эндогенный протеин)

### 1.2 Преимущества MP системы

| Аспект | CP система | MP система |
|--------|------------|------------|
| Учёт микробиома | Нет | Да |
| Разделение RUP/RDP | Нет | Да |
| Аминокислоты | Ограничено | Детально |
| Точность | Низкая | Высокая |

---

## 2. Компоненты MP

### 2.1 RDP — Rumen Degradable Protein

**Определение:** Протеин, разрушаемый микроорганизмами рубца.

**Функции:**
- Субстрат для микробного роста
- Источник азота для MCP синтеза

**Оптимальный уровень:** 60-65% от СП

### 2.2 RUP — Rumen Undegradable Protein

**Определение:** Протеин, проходящий через рубец без разрушения.

**Функции:**
- "Bypass" протеин для тонкого кишечника
- Дополнение к MCP при высокой продукции

**Оптимальный уровень:** 35-40% от СП (ранняя лактация)

### 2.3 MCP — Microbial Crude Protein

**Синтез микробного протеина:**
```
MCP (г/день) = 0.13 × TDN (кг/день)
```

Или через энергию:
```
MCP (г/день) = 130 × NEL (Mcal/день)
```

**Эффективность синтеза:**
| Фактор | Влияние на MCP |
|--------|----------------|
| RDP доступность | Лимитирующий фактор |
| Энергия (NEL) | Лимитирующий фактор |
| pH рубца | Оптимум 6.0-6.8 |
| Температура | Оптимум 39°C |

**Состав MCP:**
| Компонент | Содержание |
|-----------|------------|
| СП | 100% |
| Истинный протеин | 75-80% |
| Нуклеиновые кислоты | 15-20% |
| Лизин | 7.8% от CP |
| Метионин | 2.4% от CP |

## 3. MP требования

### 3.1 MP Maintenance

| Масса тела (кг) | MP maint (г/день) |
|-----------------|-------------------|
| 500 | 350 |
| 600 | 400 |
| 650 | 425 |
| 700 | 450 |

**Формула:** MP_maint = 2.75 × BW^0.5

### 3.2 MP для молока

```
MP_milk (г/кг молока) = True Protein (г/кг) / 0.65
```

Где 0.65 — эффективность использования MP для молока.

**Стандартные значения:**

| Молоко | Белок % | True Protein | MP required (г/кг) |
|--------|---------|--------------|-------------------|
| 3.0% | 3.0 | 26 | 85 |
| 3.3% | 3.3 | 29 | 94 |
| 3.5% | 3.5 | 31 | 100 |
| 3.8% | 3.8 | 33 | 108 |

### 3.3 MP для беременности

| Триместр | MP (г/день) |
|----------|-------------|
| 1-2 | 0 |
| 3 (начало) | 150 |
| 3 (середина) | 300 |
| 3 (конец) | 450-550 |

### 3.4 Сводные MP требования

Корова 650 кг:

| Молоко (кг/день) | MP total (г/день) | MP (г/кг СВ) | СП (% СВ) |
|------------------|-------------------|--------------|-----------|
| 20 | 1,200 | 75 | 15-16 |
| 25 | 1,400 | 82 | 16-17 |
| 30 | 1,600 | 89 | 17-18 |
| 35 | 1,800 | 95 | 18-19 |
| 40 | 2,000 | 100 | 19-20 |
| 45 | 2,200 | 105 | 20-21 |
| 50 | 2,400 | 110 | 21-22 |

**Примечание:** При DMI = 3.2% от массы тела.

## 6. Расчёт MP баланса

### 6.1 Пошаговый алгоритм

**Шаг 1:** Рассчитать MP требование
```
MP_req = MP_maint + MP_milk + MP_preg
```

**Шаг 2:** Рассчитать MCP
```
MCP = 130 × NEL_intake (Mcal)
```

**Шаг 3:** Рассчитать RUP поглощение
```
RUP_intake = Σ (Feed_i × CP_i × RUP_%_i)
```

**Шаг 4:** Рассчитать общее MP
```
MP_supply = MCP × 0.75 + RUP_intake × RUP_digestibility + END
```

**Шаг 5:** Рассчитать баланс
```
MP_balance = MP_supply - MP_req
```

### 6.2 Коэффициенты усвояемости

| Компонент | Усвояемость |
|-----------|-------------|
| MCP | 75-80% |
| RUP (среднее) | 80-85% |
| RUP (экструдер сои) | 90-95% |
| RUP (кровяная мука) | 85-90% |
| END | 100% (фиксировано) |

## 8. Мониторинг и диагностика

### 8.1 Индикаторы статуса протеина

| Показатель | Целевой | Низкий | Высокий |
|------------|---------|--------|---------|
| Мочевина в молоке (MUN) | 12-16 мг/дл | < 10 | > 18 |
| Мочевина в крови (BUN) | 15-20 мг/дл | < 12 | > 25 |
| Белок молока | 3.2-3.5% | < 3.0% | > 3.8% |

### 8.2 Интерпретация MUN

| MUN (мг/дл) | Интерпретация | Действие |
|-------------|---------------|----------|
| < 8 | Дефицит RDP | Увеличить растворимый протеин |
| 8-12 | Низкий | Проверить RDP |
| **12-16** | **Оптимум** | Поддерживать |
| 16-18 | Граница | Мониторинг |
| > 18 | Избыток RDP | Снизить RDP, проверить энергию |
| > 22 | Тяжёлый избыток | Снизить протеин, риск ацидоза |

## 9. Практические примеры

### Пример 1: Ранняя лактация

**Условия:**
- Корова 650 кг
- 30 DIM, 40 кг молока
- DMI = 22 кг
- Рацион: 16.5% СП, 38% RUP

**Расчёт:**
```
MP_req = 425 (maint) + 40 × 100 = 4,425 г/день

MCP = 130 × 38 (NEL) = 4,940 г MCP
MCP_MP = 4,940 × 0.75 = 3,705 г

RUP_intake = 22 × 0.165 × 0.38 = 1,379 г
RUP_MP = 1,379 × 0.85 = 1,172 г

MP_supply = 3,705 + 1,172 = 4,877 г
MP_balance = 4,877 - 4,425 = +452 г (дефицита нет)
```

### Пример 2: Проверка АК

**Условия:**
- MP supply = 2,000 г
- Лизин = 130 г (6.5% MP)
- Метионин = 42 г (2.1% MP)

**Анализ:**
```
Лизин: 6.5% — граница, можно увеличить до 7.0%
Метионин: 2.1% — норма
Соотношение Lys:Met = 3.1:1 — оптимум
```
