# 2. РЕЗЮМЕ (Abstract)

Исследование характеризовало размер частиц кормов и проглоченных болюсов. Образцы включали 6 образцов ржаного сена различной длины частиц, сенаж из травы, кукурузный силос и ТМР. Боли длинного сена, сена на сите 19 мм, сена на сите 8 мм и сена 50 мм имели сопоставимую среднюю длину (ML) 10-11 мм. Размеры частиц болюсов этих образцов не зависели от размеров частиц корма. Боли сена на сите 1.18 мм имели PROP_1.6 0.51 и меньшую ML 8 мм. Мало изменений размера частиц при поедании ТМР или силосов. Размер частиц болюса не связан с размером крупных частиц корма — жвачные пережёвывают корм до постоянного размера, подходящего для глотания. Этот размер может зависеть от химического состава корма.

# 3. ВВЕДЕНИЕ (Introduction)

## 3.1. Полный текст введения [перевод]

**Важность размера частиц:**

Размер частиц корма в диетах молочного скота связан с ПВМ, удоем молока, составом молока и эффектами на здоровье. Однако, несмотря на годы исследований, всё ещё сложно предсказать эффекты размера частиц диеты на продуктивность животных.

**Проблема различий в методах:**

Одна из проблем при разработке рекомендаций по кормлению из опубликованной литературы может быть частично связана с различными методами, используемыми для оценки размера частиц в различных исследованиях. Вариация в методе может способствовать непоследовательности результатов.

**Размер частиц болюса:**

В литературе обычно рассматриваются только эффекты размера частиц корма, но не размеры пережёванных и проглоченных частиц. Только несколько исследований измеряли размер мастиката или размер частиц рубцовой пищеварительной массы вместе с размером частиц корма.

**Критический размер:**

Частицы диеты состоят из частиц с размерами, превышающими критический размер для прохождения из рубца. Частицы длиннее этого критического размера могут быть более ответственными за функциональность рубцового вала.

## 3.2. Ключевые аргументы автора

1. **Размер частиц болюса vs корма:**
   - Размер частиц корма не обязательно отражает размер проглоченного болюса
   - Необходимо учитывать оба параметра

2. **Пороговый размер:**
   - Жвачные пережёвывают корм до постоянного размера (10-11 мм для сена)
   - Независимо от исходного размера частиц

3. **Влияние химического состава:**
   - Размер болюса может зависеть от химического состава
   - Влажность, NDF, лигнин влияют на пережёвывание

4. **Методология:**
   - Горизонтальное мокрое сито
   - Анализ изображений для измерения длины частиц
   - Критический размер 5 мм для анализа

# 6. ПРАКТИЧЕСКОЕ ПРИМЕНЕНИЕ

## 6.1. Алгоритм/рекомендации

**Оценка размера частиц:**

1. **Для длинных грубых кормов (>25 мм):**
   - Размер частиц корма не определяет размер болюса
   - Коровы пережёвывают до постоянного размера (~10-11 мм)
   - Фокус на химическом составе, а не размере частиц

2. **Для мелких частиц (<25 мм):**
   - Размер болюса коррелирует с размером корма
   - Важен размер частиц в корме

3. **Для силосов и ТМР:**
   - Высокая влажность позволяет глотать более длинные частицы
   - Меньшее количество жеваний

## 6.2. Калькулятор/инструменты

**Оценка физической эффективности:**

```
Факторы:
- Размер частиц корма
- Химический состав (aNDF, ADL)
- Влажность
- Ожидаемый размер болюса

Оценка:
- Корм с частицами >25 мм → болюс ~10-11 мм
- Корм с частицами <25 мм → зависит от размера корма
- Сенаж/ТМР с высокой влажностью → более длинные болюсы
```

# 8. ВЫВОДЫ (Conclusions)

## 8.1. Полный текст выводов [перевод]

**Основные выводы:**

1. Размер частиц болюса не обязательно связан с размером частиц корма.

2. Ржаные сена с ML от 25 мм до примерно 600 мм имели одинаковую ML болюса 10-11 мм, независимо от ML корма. ML 10-11 мм ржаных сен может представлять максимальную длину, которую коровы могли проглотить.

3. Только для ржаных сен с ML меньше 25 мм (1.18_PSPS), травяного и кукурузного сенажа, и ТМР распределения длин частиц корма были сильно коррелированы с соответствующими болюсами.

4. Ржаные сена с ML ниже пороговой длины 25 мм могут с большей вероятностью влиять на параметры, такие как время удержания частиц в рубце и потребление.

5. Ситовое оборудование для лучшего описания различных размеров частиц сенокошеной травы, которые могут быть более связаны с прохождением и потреблением, должно, вероятно, содержать дополнительное сито с отверстиями меньше 8 мм.

6. Частицы корма, проходящие через сито 1.18 мм, могут не определять удержание в рубце.

7. Размер частиц болюса не связан с размером крупных частиц корма. Крупные частицы корма пережёвываются до порогового размера, приемлемого для формирования болюса и глотания. Данные текущего исследования предполагают, что этот размер связан с химическим составом корма.

## 8.2. Ключевые выводы (структурировано)

1. **Пороговый размер:**
   - ~10-11 мм для ржаных сен
   - Максимальный размер, который корова может проглотить

2. **Размер корма vs болюса:**
   - Для крупных частиц (>25 мм) — независимы
   - Для мелких частиц (<25 мм) — коррелированы

3. **Химический состав:**
   - Влияет на интенсивность пережёвывания
   - Влажность, NDF, ADL — ключевые факторы

4. **Практические рекомендации:**
   - Необходимость дополнительного сита <8 мм
   - Учёт химического состава при оценке физической эффективности

## 8.3. Ключевые сообщения для лекции

> "Размер частиц болюса не связан с размером крупных частиц корма — жвачные пережёвывают корм до постоянного размера, подходящего для глотания."

> "Пороговый размер для ржаных сен — около 10-11 мм. Это максимальная длина, которую корова способна проглотить."

> "Размер болюса связан с химическим составом корма, а не только с физическими свойствами."

# 10. FAQ

**Q1: Почему размер болюса важнее размера корма?**
A: Потому что именно размер болюса определяет удержание в рубце и прохождение через ретикуло-омазальный отверстие, а не исходный размер корма.

**Q2: Какой пороговый размер частиц для коров?**
A: Для ржаных сен — около 10-11 мм. Это максимальная длина, которую корова может проглотить.

**Q3: Почему сенаж и ТМР имеют более длинные болюсы?**
A: Из-за высокой влажности и низкого содержания клетчатки, что облегчает глотание длинных частиц.

**Q4: Как измерить размер частиц болюса?**
A: Требуется рубцово-фистулированные коровы. Болюсы собираются через канюлу, затем анализируются ситовым методом и анализом изображений.

**Q5: Нужно ли пересматривать рекомендации по размеру частиц?**
A: Да, необходимо учитывать, что для крупных частиц (>25 мм) размер корма не определяет размер болюса.

**Q6: Как химический состав влияет на размер болюса?**
A: Влажность, содержание NDF и лигнина влияют на гибкость частиц и интенсивность пережёвывания.

**Q7: Какое сито рекомендуется для анализа?**
A: Рекомендуется добавить сито с отверстиями между 1.18 и 8 мм для лучшего описания размеров частиц.
