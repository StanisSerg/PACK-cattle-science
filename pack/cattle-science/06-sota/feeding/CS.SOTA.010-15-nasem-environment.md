## 1. Тепловой стресс

### 1.1 Температурно-влажностный индекс (THI)

**Формула THI:**
```
THI = (1.8 × T°C + 32) - [(0.55 - 0.0055 × RH%) × (1.8 × T°C - 26.8)]
```

**Упрощённая формула:**
```
THI = T°C - (0.55 - 0.0055 × RH%) × (T°C - 14.5)
```

**Интерпретация THI:**
| THI | Категория | Влияние на коров |
|-----|-----------|------------------|
| < 68 | Комфорт | Нет стресса |
| 68-72 | Лёгкий стресс | Незначительное снижение DMI |
| 72-80 | Умеренный стресс | DMI ↓ 5-10%, молоко ↓ 5-10% |
| 80-90 | Тяжёлый стресс | DMI ↓ 10-20%, молоко ↓ 10-20% |
| > 90 | Опасный стресс | DMI ↓ > 20%, риск смерти |

### 1.2 Физиологические реакции на жару

| Параметр | Норма | При THI > 72 | При THI > 80 |
|----------|-------|--------------|--------------|
| Температура тела | 38.5°C | 39.0-39.5°C | > 39.5°C |
| Частота дыхания | 30-40/мин | 60-80/мин | > 80/мин |
| Потоотделение | Минимальное | Умеренное | Интенсивное |
| DMI | 100% | 90-95% | 80-90% |
| Молоко | 100% | 90-95% | 80-90% |

### 1.3 Корректировки питания при жаре

| Параметр | Норма | При THI 72-80 | При THI > 80 |
|----------|-------|---------------|--------------|
| NEL, Mcal/кг СВ | 1.65 | 1.70-1.75 | 1.75-1.80 |
| СП, % СВ | 16-17 | 17-18 | 18-19 |
| Жир, % СВ | 4-5 | 5-6 | 6-7 |
| NDF, % СВ | 30-32 | 28-30 | 26-28 |
| K, % СВ | 1.2-1.5 | 1.5-1.8 | 1.8-2.0 |
| Na, % СВ | 0.3-0.4 | 0.4-0.5 | 0.5-0.6 |

### 1.4 Практические меры

**Управление окружением:**
| Мера | Эффективность | Стоимость |
|------|---------------|-----------|
| Теневые навесы | +++ | Низкая |
| Вентиляция | +++ | Средняя |
| Распыление воды | ++++ | Средняя |
| Охлаждение стойла | +++++ | Высокая |
| Пастбище ночью | +++ | Низкая |

**Кормовые стратегии:**
| Стратегия | Механизм |
|-----------|----------|
| Увеличение энергетической плотности | Компенсация снижения DMI |
| Добавка жиров | Высокая плотность, меньше ТЭП |
| Увеличение СП | Компенсация снижения микробного протеина |
| Электролиты | Компенсация потерь с потом |

## 2. Холодовой стресс

### 2.1 Нижняя критическая температура (LCT)

| Категория | LCT (°C) | Факторы снижения LCT |
|-----------|----------|----------------------|
| Телёнок (< 3 недель) | 15-20 | Сухая шерсть, укрытие |
| Телёнок (> 3 недель) | 5-10 | Увеличенный рацион |
| Нетель | -5 до 0 | BCS > 3.0, укрытие |
| Взрослая корова | -15 до -10 | Сухая шерсть, BCS > 3.0 |
| Свежая корова | -5 до 0 | Метаболизм, укрытие |

### 2.2 Корректировки при холоде

**Увеличение энергии:**
| Температура | Увеличение NEM | Практическое применение |
|-------------|----------------|-------------------------|
| 0°C | +10% | +0.5 кг концентрата |
| -10°C | +20% | +1.0 кг концентрата |
| -20°C | +30% | +1.5 кг концентрата |
| -30°C | +40% | +2.0 кг концентрата |

**Формула корректировки:**
```
NE increase = 0.0007 × (LCT - T)^2 × BW^0.75
```

### 2.3 Управление в холодный период

| Мера | Эффект |
|------|--------|
| Увеличение энергии рациона | Компенсация теплопотерь |
| Увеличение количества корма | Поддержание DMI |
| Укрытие от ветра | Снижение ощущаемой температуры |
| Сухая подстилка | Изоляция от холода |
| Доступ к воде (не ледяной) | Поддержание DMI |

## 3. Выбросы метана

### 3.1 Источники метана

| Источник | Доля выбросов | Механизм |
|----------|---------------|----------|
| Рубцовая ферментация | 90-95% | Метаногенез |
| Толстый кишечник | 4-8% | Ферментация |
| Почва навоза | 1-5% | Анаэробное разложение |

### 3.2 Факторы, влияющие на метан

| Фактор | Влияние на CH4 | Механизм |
|--------|----------------|----------|
| Концентраты | ↓↓↓ | Снижение pH, быстрая ферментация |
| Жиры | ↓↓ | Ингибирование метаногенов |
| Линолевая кислота | ↓↓↓ | Токсичность для метаногенов |
| Nitrate | ↓↓↓ | Альтернативный акцептор электронов |
| 3-NOP | ↓↓↓↓ | Ингибитор метаногенов |
| Ionophores | ↓ | Сдвиг к пропионату |

### 3.3 Стратегии снижения метана

**Питательные:**
| Стратегия | Снижение CH4 | Побочные эффекты |
|-----------|--------------|------------------|
| Увеличение концентратов | 10-20% | Риск ацидоза |
| Добавка жиров | 10-15% | Снижение DMI при > 6% |
| Linseed oil | 15-20% | Снижение DMI |
| Nitrate (3% СВ) | 20-30% | Риск токсичности |
| 3-NOP (60-80 мг/кг) | 30-50% | Высокая стоимость |

**Управленческие:**
| Стратегия | Снижение CH4 | Примечание |
|-----------|--------------|------------|
| Улучшение продуктивности | 10-15% | Меньше коров для того же молока |
| Селекция | 5-10% | Генетика метана |
| Раннее отлучение | 5-10% | Сокращение периода содержания |

## 4. Азотистые выбросы

### 4.1 Баланс азота в организме

```
Азот с кормом (N intake)
    ↓
┌─────────────────────────────────────┐
│  Молоко (15-25%)                    │
│  Тело (рост, беременность) (5-15%)  │
│  Моча (40-55%)                      │
│  Фекалии (30-40%)                   │
└─────────────────────────────────────┘
```

### 4.2 Факторы, влияющие на выделение азота

| Фактор | Влияние | Механизм |
|--------|---------|----------|
| Избыток СП | ↑↑↑ | Неусвоенный азот |
| Небаланс RDP/RUP | ↑↑ | Неэффективное использование |
| Низкая энергия | ↑↑ | Ограничение MCP синтеза |
| MUN > 18 | ↑↑ | Избыток аммиака |

### 4.3 Стратегии снижения азотистых выбросов

**Питательные:**
| Стратегия | Снижение N выбросов | Эффективность |
|-----------|---------------------|---------------|
| MP система vs CP | 15-20% | Высокая |
| Баланс аминокислот | 10-15% | Высокая |
| Оптимальный MUN (12-16) | 10-15% | Средняя |
| Фазное кормление | 5-10% | Средняя |

**Управление навозом:**
| Стратегия | Снижение NH3 | Снижение N2O |
|-----------|--------------|--------------|
| Немедленное внесение | +++ | — |
| Инъекция в почву | ++++ | ++ |
| Покрытие навозохранилища | +++ | + |
| Компостирование | + | ++ |
| Анаэробная переработка | ++ | +++ |

## 5. Экологический след молочного производства

### 5.1 Показатели углеродного следа

| Показатель | Типичное значение | Перспективное |
|------------|-------------------|---------------|
| CO2-eq/кг молока | 1.0-1.5 кг | < 0.8 кг |
| CO2-eq/кг FPCM | 0.9-1.3 кг | < 0.7 кг |
| CO2-eq/кг живого веса/год | 3.0-4.5 т | < 3.0 т |

### 5.2 Распределение выбросов

| Источник | Доля выбросов | Стратегии снижения |
|----------|---------------|-------------------|
| Корм (производство) | 35-45% | Эффективность, местные корма |
| Метан рубца | 25-35% | Ингибиторы, концентраты |
| Навоз (хранение) | 10-15% | Быстрое внесение, переработка |
| Энергия (ферма) | 5-10% | Возобновляемые источники |
| Другое | 10-15% | Разное |

### 5.3 Эффективность использования корма

| Показатель | Типичный | Эффективный |
|------------|----------|-------------|
| FCE (кг молока/кг СВ) | 1.2-1.4 | > 1.5 |
| N use efficiency | 25-30% | > 35% |
| P use efficiency | 30-40% | > 45% |

## 6. Практические рекомендации

### 6.1 Мониторинг стресса

| Показатель | Метод | Частота |
|------------|-------|---------|
| THI | Термометр + гигрометр | Непрерывно |
| Температура тела | Ректальная/логгеры | При подозрении |
| Частота дыхания | Наблюдение | Ежедневно в жару |
| DMI | Измерение | Ежедневно |

### 6.2 План действий при жаре

| THI | Действия |
|-----|----------|
| 68-72 | Увеличить наблюдение, проверить вентиляцию |
| 72-80 | Активное охлаждение, корректировка рациона |
| 80-90 | Интенсивное охлаждение, ночное кормление |
| > 90 | Экстренные меры, минимизация стресса |

### 6.3 Устойчивое производство

| Направление | Действия |
|-------------|----------|
| Снижение метана | 3-NOP, жиры, оптимизация рациона |
| Снижение азота | MP система, баланс АК |
| Эффективность | Селекция, здоровье, управление |
| Возобновляемая энергия | Солнечные панели, биогаз |
