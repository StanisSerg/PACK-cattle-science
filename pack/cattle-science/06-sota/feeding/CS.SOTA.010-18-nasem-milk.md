## 1. Компоненты молока

### 1.1 Средний состав молока

| Компонент | Содержание | Диапазон | Вариабельность |
|-----------|------------|----------|----------------|
| Вода | 87.0% | 85-89% | Низкая |
| Общее сухое вещество | 13.0% | 11-15% | Средняя |
| Жир | 4.0% | 2.5-5.5% | Высокая |
| Белок | 3.3% | 2.5-4.0% | Средняя |
| Лактоза | 4.8% | 4.5-5.0% | Низкая |
| Минералы (зола) | 0.7% | 0.6-0.8% | Низкая |

### 1.2 Энергетическая ценность молока

**Формула:**
```
NEL (Mcal/кг) = 0.0929 × Fat% + 0.0547 × CP% + 0.0395 × Lactose%
```

**Упрощённая формула:**
```
NEL (Mcal/кг) ≈ 0.74 + 0.096 × Fat% + 0.047 × Protein%
```

| Жир % | Белок % | NEL (Mcal/кг) |
|-------|---------|---------------|
| 3.5 | 3.0 | 0.72 |
| 4.0 | 3.3 | 0.77 |
| 4.5 | 3.5 | 0.82 |
| 5.0 | 3.8 | 0.88 |

## 2. Синтез жира молока

### 2.1 Источники молочного жира

| Источник | Доля | Механизм |
|----------|------|----------|
| Де novo (в молочной железе) | 40-50% | Синтез из ацетата, β-butyrate |
| Преformed (из крови) | 50-60% | Жиры рациона, мобилизация |

### 2.2 Де novo синтез

**Субстраты:**
| Субстрат | Источник | Доля в жире |
|----------|----------|-------------|
| Ацетат | Рубцовая ферментация | 35-45% |
| β-hydroxybutyrate | Кровь | 5-10% |
| Глюкоза | Кровь | 5-10% |

**Жирные кислоты де novo:**
| Кислота | Длина цепи | Доля в молочном жире |
|---------|------------|----------------------|
| C4:0 (Butyric) | 4C | 3-4% |
| C6:0 (Caproic) | 6C | 2-3% |
| C8:0 (Caprylic) | 8C | 1-2% |
| C10:0 (Capric) | 10C | 3-4% |
| C12:0 (Lauric) | 12C | 3-5% |
| C14:0 (Myristic) | 14C | 10-12% |
| C16:0 (Palmitic) | 16C | 25-30% |

### 2.3 Факторы, влияющие на жирность

| Фактор | Влияние | Механизм |
|--------|---------|----------|
| **NDF / Forage** | +++ | Ацетат → де novo синтез |
| **Концентраты** | — | Пропионат, низкий ацетат |
| **Жиры рациона** | + (переменное) | Преformed ЖК |
| **Ненасыщенные ЖК** | — | Ингибирование де novo |
| **pH рубца** | — | < 6.0 снижает жирность |
| **Частота кормления** | + | Стабильность рубца |

### 2.4 Жир:Белок соотношение

| Соотношение | Интерпретация | Действие |
|-------------|---------------|----------|
| < 1.10 | Низкий жир | Проверить NDF, ацидоз |
| 1.10-1.25 | Норма | Оптимально |
| 1.25-1.40 | Хороший жир | Приемлемо |
| **> 1.40** | **Высокий жир** | **Риск кетоза** |
| > 1.50 | Критический | Проверить энергетический баланс |

## 3. Синтез белка молока

### 3.1 Источники молочного белка

| Источник | Доля | Компоненты |
|----------|------|------------|
| Кровь (аминокислоты) | 80-90% | Все эссенциальные АК |
| Молочная железа (синтез) | 10-20% | Неэссенциальные АК |

### 3.2 Белковые фракции молока

| Фракция | % от общего белка | Функция |
|---------|-------------------|---------|
| Казеины | 78-82% | Сычужный, пищевая ценность |
| Сывороточные белки | 18-20% | Иммунитет, питание |
| Жирнокислот-связанный | < 1% | Транспорт |

**Казеины:**
| Тип | % от казеина | Особенности |
|-----|--------------|-------------|
| αs1-casein | 38-40% | Структура, вкус |
| αs2-casein | 10-12% | Минеральное связывание |
| β-casein | 35-38% | Гибкость мицелл |
| κ-casein | 12-15% | Стабильность мицелл |

### 3.3 Аминокислоты и белок молока

| Аминокислота | Лимитация | Действие при дефиците |
|--------------|-----------|----------------------|
| **Лизин** | 1-я | Снижение белка |
| **Метионин** | 2-я | Снижение белка, жира |
| **Гистидин** | 3-я | Снижение белка |
| Лейцин | Редко | Сигналинг |
| Фенилаланин | Редко | Тирозин прекурсор |

### 3.4 Факторы, влияющие на белок

| Фактор | Влияние | Механизм |
|--------|---------|----------|
| **Энергия (NEL)** | +++ | Доступность АК, синтез |
| **MP (метаболизируемый протеин)** | +++ | Субстрат для синтеза |
| **Аминокислоты** | +++ | Лимитирующие факторы |
| **Генетика** | ++ | Потенциал синтеза |
| **Стадия лактации** | ++ | DIM эффект |
| **Мастит** | — | Повреждение ткани |

## 4. Лактоза

### 4.1 Синтез лактозы

**Механизм:**
```
Глюкоза + UDP-галактоза → Лактоза
```

**Особенности:**
- Осмотический регулятор молока
- Концентрация стабильна (4.5-5.0%)
- Прямо пропорциональна объёму молока

### 4.2 Факторы, влияющие на лактозу

| Фактор | Влияние | Примечание |
|--------|---------|------------|
| Генетика | + | Породные различия |
| Стадия лактации | + | Незначительно |
| Мастит | — | Снижение (использование лактозы) |
| Голод | — | Снижение объёма → лактоза сохраняется |

## 5. Питание и состав молока

### 5.1 Влияние на жирность

| Кормовой фактор | Изменение жирности | Механизм |
|-----------------|-------------------|----------|
| NDF > 30% СВ | +0.2-0.4% | Больше ацетата |
| NDF < 26% СВ | -0.2-0.4% | Меньше ацетата |
| Добавка жиров | ±0.1-0.3% | Зависит от типа жира |
| Насыщенные жиры | +0.1-0.2% | Не ингибируют де novo |
| Ненасыщенные жиры | -0.2-0.4% | Ингибируют де novo |
| Буферы | +0.1-0.2% | Стабильность pH |
| Частое кормление | +0.1-0.2% | Стабильность рубца |

### 5.2 Влияние на белок

| Кормовой фактор | Изменение белка | Механизм |
|-----------------|-----------------|----------|
| MP > требования | +0.1-0.2% | Достаточно АК |
| MP < требования | -0.1-0.3% | Дефицит АК |
| Лизин ↑ | +0.05-0.1% | Лимитирующий АК |
| Метионин ↑ | +0.05-0.1% | Лимитирующий АК |
| Энергия ↑ | +0.1-0.2% | Лучшее использование АК |
| Энергия ↓ | -0.1-0.2% | Мобилизация |

### 5.3 MUN — Milk Urea Nitrogen

**Нормативы:**
| MUN (мг/дл) | Интерпретация | Действие |
|-------------|---------------|----------|
| < 8 | Дефицит RDP | Увеличить растворимый протеин |
| 8-12 | Низкий | Проверить RDP |
| **12-16** | **Оптимум** | **Поддерживать** |
| 16-18 | Граница | Мониторинг |
| > 18 | Избыток | Снизить протеин |
| > 22 | Тяжёлый избыток | Риск ацидоза, выбросов |

**Связь с BUN:**
```
MUN ≈ BUN × 0.85
BUN ≈ MUN × 1.15
```

### 5.4 Влияние на профиль жирных кислот

| Кормовой фактор | C16:0 | C18:0 | Ненасыщенные |
|-----------------|-------|-------|--------------|
| Высокий концентрат | ↑ | ↓ | ↓ |
| Высокое сено | ↓ | ↑ | ↑ |
| Масличные добавки | ↓ | ↓ | ↑↑ |
| Желудочный жир (bypass) | ↑↑ | — | — |

## 6. Качество молока

### 6.1 Показатели качества

| Показатель | Целевое | Действие при отклонении |
|------------|---------|-------------------------|
| Соматические клетки | < 200,000/мл | > 250,000 → мастит |
| Общее число бактерий | < 10,000/мл | > 50,000 → гигиена |
| Ингибиторы | Отрицательно | Положительно → карантин |
| Температура | < 4°C | > 6°C → порча |

### 6.2 Факторы, влияющие на качество

| Фактор | Влияние на СС | Влияние на ТБС |
|--------|---------------|----------------|
| Гигиена доения | + | +++ |
| Охлаждение | — | +++ |
| Мастит | ++++ | + |
| Кормление | + | — |
| Стресс | ++ | — |

### 6.3 Питание и мастит

| Питательный фактор | Влияние | Механизм |
|--------------------|---------|----------|
| Витамин E + Se | Снижение риска | Антиоксиданты |
| Zn (органический) | Снижение риска | Иммунитет |
| Витамин A | Снижение риска | Эпителий |
| Перекорм энергией | Повышение риска | Иммуносупрессия |
| Дефицит Se | Повышение риска | Окислительный стресс |

## 7. Практические рекомендации

### 7.1 Оптимизация жирности

| Стратегия | Действие | Ожидаемый эффект |
|-----------|----------|------------------|
| NDF 28-32% СВ | Увеличить грубые корма | +0.2-0.3% |
| peNDF 21-23% | Обеспечить физическую форму | +0.1-0.2% |
| Буферы | Стабилизировать pH | +0.1-0.2% |
| Частое кормление | 3+ раза в день | +0.1-0.2% |
| Ограничить ненасыщенные | < 3% СВ | +0.1-0.3% |

### 7.2 Оптимизация белка

| Стратегия | Действие | Ожидаемый эффект |
|-----------|----------|------------------|
| MP баланс | Обеспечить 100-105% | +0.1-0.2% |
| Лизин 7.0% MP | Добавить защищённый | +0.05-0.1% |
| Метионин 2.4% MP | Добавить защищённый | +0.05-0.1% |
| Энергетический баланс | Избегать дефицита | +0.1-0.2% |
| MUN 12-16 | Оптимизировать RDP | Стабильность |

### 7.3 Мониторинг состава

| Показатель | Частота | Действие |
|------------|---------|----------|
| Жир, белок, лактоза | Ежемесячно (пул) | Тренды |
| MUN | Ежемесячно | Баланс протеина |
| Жир:Белок | Еженедельно | Энергетический статус |
| СС | Ежемесячно | Здоровье вымени |

### 7.4 Действия при отклонениях

| Проблема | Возможная причина | Решение |
|----------|-------------------|---------|
| Жир < 3.5% | Низкий NDF, ацидоз | Увеличить грубые корма, буферы |
| Жир > 5.0% | Кетоз, мобилизация | Проверить NEFA, энергию |
| Белок < 3.0% | Дефицит MP, АК | Увеличить протеин, баланс АК |
| MUN > 18 | Избыток RDP | Снизить растворимый протеин |
| MUN < 10 | Дефицит RDP | Увеличить RDP |
