## 1. Ионофоры (Ionophores)

### 1.1 Механизм действия

**Действие на микробиом:**
```
Ионофоры (монензин, ласалоцид)
    ↓
Избирательное подавление грам(+) бактерий
    ↓
Снижение ацетат/пропионат соотношения
    ↓
Увеличение пропионата (глюконеогенез)
    ↓
Снижение метана, контроль ацидоза
```

### 1.2 Монензин (Monensin)

**Дозировки:**
| Форма | Дозировка | Применение |
|-------|-----------|------------|
| Premix (Rumensin) | 11-33 г/т СВ | Стандарт |
| Controlled release | 300-400 мг/голова/день | Длительное действие |

**Эффекты:**
| Параметр | Изменение | Механизм |
|----------|-----------|----------|
| Пропионат | +10-15% | Сдвиг ферментации |
| Метан | -10-15% | Снижение H2 |
| DMI | -2-5% | Контроль аппетита |
| Молоко | +1-2 кг | Энергетическая эффективность |
| Кетоз | -30-50% | Больше пропионата |
| Ацидоз | -20-30% | Контроль молочнокислых |

### 1.3 Ласалоцид (Lasalocid)

**Дозировка:** 100-300 мг/голова/день

**Сравнение с монензином:**
| Параметр | Монензин | Ласалоцид |
|----------|----------|-----------|
| Сдвиг к пропионату | +++ | ++ |
| Контроль кокцидиоза | — | +++ |
| Применение у телят | Ограничено | Широкое |
| Эффект на молоко | +++ | ++ |

### 1.4 Применение и ограничения

| Категория | Рекомендация |
|-----------|--------------|
| Ранняя лактация | +++ (контроль кетоза) |
| Средняя лактация | ++ |
| Сухостойные | + (подготовка) |
| Телята | + (ласалоцид) |
| Запрещено | В органическом производстве |

## 2. Буферы и щелочи

### 2.1 Натрий бикарбонат (Sodium Bicarbonate)

**Характеристики:**
| Параметр | Значение |
|----------|----------|
| Формула | NaHCO₃ |
| pH раствора | 8.3 (1% раствор) |
| Буферная ёмкость | Высокая |

**Дозировки:**
| Условия | Дозировка | % рациона |
|---------|-----------|-----------|
| Стандарт | 100-150 г/голова/день | 0.5-0.8% |
| Высокие концентраты | 150-250 г/голова/день | 0.8-1.2% |
| Риск ацидоза | 200-300 г/голова/день | 1.0-1.5% |

**Эффекты:**
| Параметр | Изменение |
|----------|-----------|
| pH рубца | +0.2-0.4 единицы |
| DMI | +5-10% |
| Молоко | +1-2 кг |
| Жир молока | +0.1-0.2% |

### 2.2 Карбонат кальция-магния

| Параметр | Значение |
|----------|----------|
| Ca:Mg соотношение | 2:1 до 5:1 |
| Буферная ёмкость | Средняя |
| Дополнительный эффект | Ca и Mg источник |

**Дозировка:** 50-150 г/голова/день

### 2.3 Другие буферы

| Буфер | Дозировка | Особенности |
|-------|-----------|-------------|
| Поташ (K₂CO₃) | 100-200 г/день | DCAD эффект |
| Магнезия (MgO) | 50-100 г/день | Mg + буфер |
| Сода кальцинированная | 100-150 г/день | Высокая щёлочность |

### 2.4 Когда применять буферы

| Условие | Приоритет |
|---------|-----------|
| Концентраты > 50% рациона | +++ |
| Жир молока < 3.5% | +++ |
| pH рубца < 6.0 | +++ |
| Силос с высокой кислотностью | ++ |
| Fine particle size кормов | ++ |

## 3. Микробиальные добавки

### 3.1 Дрожжевые культуры (Yeast)

**Механизм действия:**
```
Saccharomyces cerevisiae
    ↓
Потребление кислорода (анаэробная среда)
    ↓
Стабилизация pH
    ↓
Стимуляция целлюлолитических бактерий
    ↓
Улучшение NDF усвояемости
```

**Формы:**
| Форма | Дозировка | Стабильность |
|-------|-----------|--------------|
| Живая (active dry) | 10-20 г/день | Средняя |
| Autolyzed | 15-25 г/день | Высокая |
| Cell wall | 10-15 г/день | Высокая |

**Эффекты:**
| Параметр | Изменение | Уверенность |
|----------|-----------|-------------|
| DMI | +3-5% | Средняя |
| Молоко | +1-2 кг | Средняя |
| pH рубца | +0.1-0.2 | Высокая |
| NDF усвояемость | +5-10% | Средняя |
| Стабильность pH | +++ | Высокая |

### 3.2 Пробиотики (Direct-Fed Microbials)

**Виды микроорганизмов:**
| Вид | Дозировка (КОЕ/день) | Эффект |
|-----|----------------------|--------|
| Lactobacillus plantarum | 10^9 - 10^11 | Стабилизация pH |
| Enterococcus faecium | 10^9 - 10^11 | Иммунитет |
| Propionibacterium | 10^9 - 10^10 | Контроль кетоза |
| Bacillus subtilis | 10^9 - 10^11 | Протеолиз |

**Применение у телят:**
| Возраст | Пробиотик | Эффект |
|---------|-----------|--------|
| 0-14 дней | Lactobacillus + Bifidobacterium | Профилактика диареи |
| 14-56 дней | Multi-strain | Улучшение роста |

### 3.3 Премиксы с дрожжами

| Комбинация | Синергизм |
|------------|-----------|
| Дрожжи + буферы | +++ |
| Дрожжи + ферменты | ++ |
| Дрожжи + пробиотики | ++ |

## 4. Ферменты

### 4.1 Фибролитические ферменты

**Типы ферментов:**
| Фермент | Субстрат | Эффективность |
|---------|----------|---------------|
| Cellulase | Целлюлоза | Средняя |
| Xylanase | Гемицеллюлоза | Высокая |
| β-glucanase | β-глюканы | Высокая |
| Pectinase | Пектин | Средняя |

**Применение:**
| Корм | Фермент | Результат |
|------|---------|-----------|
| Ячмень | β-glucanase | Снижение вязкости |
| Травяной силос | Cellulase + Xylanase | +5-10% NDFD |
| Пшеница | Xylanase | +3-5% усвояемости |

### 4.2 Амилолитические ферменты

| Фермент | Применение | Эффект |
|---------|------------|--------|
| α-amylase | Кукуруза, зерно | +5-10% крахмала |
| Amyloglucosidase | Высокий крахмал | Стабилизация pH |

### 4.3 Протеолитические ферменты

| Фермент | Цель | Применение |
|---------|------|------------|
| Protease | RUP модификация | Экспериментально |
| Keratinase | Перья, шерсть | Специализированно |

## 5. Другие добавки

### 5.1 Органические кислоты

| Кислота | Дозировка | Применение |
|---------|-----------|------------|
| Пропионовая | 50-100 г/день | Контроль кетоза |
| Бензойная | 20-30 г/день | Контроль кетоза |
| Сорбиновая | 10-20 г/т | Консервант |
| Молочная | 20-40 г/т | Консервант |

**Пропиленгликоль (Propylene Glycol):**
| Параметр | Значение |
|----------|----------|
| Дозировка (лечение) | 300-500 мл/день |
| Дозировка (профилактика) | 150-300 мл/день |
| Длительность | 3-7 дней |
| Механизм | Глюконеогенез |

### 5.2 Хелатированные минералы

| Минерал | Форма | Биодоступность |
|---------|-------|----------------|
| Zn | Zn-methionine | 150-200% |
| Cu | Cu-lysine | 150-200% |
| Mn | Mn-methionine | 150-200% |
| Se | Se-yeast | 200-300% |

**Применение:**
| Ситуация | Рекомендация |
|----------|--------------|
| Проблемы с копытами | Zn-methionine 2-4 г/день |
| Иммунитет (свежие) | Se-yeast 3-4 мг/день |
| Репродукция | Cu, Zn, Mn органические формы |

### 5.3 Нуклеотиды и пребиотики

| Добавка | Механизм | Эффект |
|---------|----------|--------|
| MOS (Mannan-oligosaccharides) | Связывание патогенов | Иммунитет |
| β-glucans | Иммуностимуляция | Иммунитет |
| Нуклеотиды | Клеточное деление | Рост телят |
| Galacto-oligosaccharides | Пребиотик | Микробиом |

## 6. Ингибиторы метана

### 6.1 3-NOP (3-Nitrooxypropanol)

**Характеристики:**
| Параметр | Значение |
|----------|----------|
| Дозировка | 60-80 мг/кг СВ |
| Механизм | Ингибирование метил-CoM редуктазы |
| Снижение CH4 | 30-50% |
| Влияние на DMI | Нейтральное или +2-3% |
| Влияние на молоко | Нейтральное или +1-2 кг |

**Статус:** Регистрация в разных странах варьируется

### 6.2 Другие ингибиторы

| Добавка | Дозировка | Снижение CH4 | Статус |
|---------|-----------|--------------|--------|
| Nitrate | 1-3% СВ | 20-30% | Исследования |
| Льняное масло | 3-5% СВ | 15-20% | Практика |
| Seaweed (Asparagopsis) | 0.5-2% СВ | 50-80% | Исследования |

## 7. Практические рекомендации

### 7.1 Выбор добавок по стадиям

| Стадия | Приоритетные добавки | Цель |
|--------|---------------------|------|
| Переходный период | Монензин, буферы, дрожжи | Контроль кетоза, ацидоза |
| Ранняя лактация | Буферы, дрожжи, пропионовая | Стабильность рубца |
| Средняя лактация | Монензин, дрожжи | Эффективность |
| Сухостойные | Монензин (подготовка) | Подготовка к лактации |
| Телята | Пробиотики, ласалоцид | Здоровье, рост |

### 7.2 Экономика добавок

| Добавка | Стоимость/день | ROI | Уверенность |
|---------|----------------|-----|-------------|
| Монензин | $0.05-0.10 | 3:1 - 5:1 | Высокая |
| Бикарбонат | $0.10-0.20 | 2:1 - 4:1 | Высокая |
| Дрожжи | $0.15-0.30 | 2:1 - 3:1 | Средняя |
| Пробиотики | $0.10-0.25 | 1.5:1 - 3:1 | Средняя |
