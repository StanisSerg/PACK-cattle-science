# CS.SOTA.010-13: Transition Period

## Общая информация

| Параметр | Значение |
|----------|----------|
| **Глава** | 13 |
| **Страницы** | ~280-320 |
| **Период** | -21 дней до +30 дней от отёла |
| **Ключевые концепции** | DCAD, энергия, протеин, мониторинг |

**Важность:** 80% метаболических проблем начинаются в переходный период.

---

## 1. Физиология переходного периода

### 1.1 Ключевые изменения

| Период | Основные изменения |
|--------|-------------------|
| -21 до -7 | Подготовка к лактации, рост ВМЖ |
| -7 до 0 | Пик стресса, иммунодепрессия |
| 0 до +7 | Отёл, начало лактации, энергетический дефицит |
| +7 до +30 | Адаптация, восстановление |

### 1.2 Энергетический баланс

| DIM | Энергетический баланс | Комментарий |
|-----|----------------------|-------------|
| -21 | Нейтральный | Подготовка |
| -7 | Небольшой позитив | Оптимум |
| 0 | Нейтральный | Отёл |
| +7 | -10 до -20 Mcal | Дефицит |
| +14 | -15 до -25 Mcal | Пик дефицита |
| +30 | -5 до -15 Mcal | Восстановление |

## 2. Периодизация переходного периода

### 2.1 Ранний сухостой (-60 до -21 дней)

**Цели:**
- Восстановление ВМТ (BCS 3.25-3.75)
- Рост плода
- Подготовка молочной железы

**Рекомендации питания:**

| Параметр | Целевое значение |
|----------|------------------|
| DMI | 12-14 кг (2% от ВМТ) |
| NEL | 1.30-1.40 Mcal/кг СВ |
| СП | 12-14% |
| RUP | 30-35% |
| Ca | 3.0-3.5 г/кг СВ |
| P | 2.5-3.0 г/кг СВ |
| Mg | 4.0-4.5 г/кг СВ |
| DCAD | +50 до +150 mEq/кг |

**Стратегия:** Поддержание, не максимизация.

### 2.2 Поздний сухостой (-21 до 0 дней) — КРИТИЧНО

**Цели:**
- Профилактика гипокальции
- Подготовка к энергетическому дефициту
- Адаптация рубца к лактационному рациону

#### Вариант A: DCAD стратегия (рекомендуется)

| Параметр | Целевое значение |
|----------|------------------|
| DMI | 12-14 кг |
| NEL | 1.50-1.60 Mcal/кг СВ |
| СП | 14-15% |
| RUP | 35-38% |
| Ca | 4.0-4.5 г/кг СВ |
| P | 2.8-3.2 г/кг СВ |
| Mg | 4.0-4.5 г/кг СВ |
| **DCAD** | **-50 до -150 mEq/кг** |
| K | < 1.2% |

**Анионные соли:**
- CaCl₂: 100-150 г/голова/день
- MgSO₄: 50-100 г/голова/день
- Или коммерческие продукты

**Мониторинг:**
| Показатель | Целевое |
|------------|---------|
| pH мочи | 6.0-6.5 |
| Ca сыворотки | > 8.0 мг/дл |

#### Вариант B: Низкий Ca стратегия

| Параметр | Целевое значение |
|----------|------------------|
| Ca | < 15 г/день (< 1.2 г/кг СВ) |
| P | Нормальный (30-40 г/день) |
| DCAD | Положительный (+100 до +200) |

**Примечание:** Менее эффективна, требует строгого контроля.

### 2.3 Ранний послеродовый (0 до +30 дней)

**Цели:**
- Максимизация DMI
- Минимизация энергетического дефицита
- Профилактика метаболических
- Восстановление BCS

| Параметр | 0-7 дней | 7-14 дней | 14-30 дней |
|----------|----------|-----------|------------|
| DMI (кг) | 10-14 | 16-20 | 20-24 |
| NEL (Mcal/кг) | 1.70-1.80 | 1.75-1.85 | 1.70-1.80 |
| СП (%) | 17-18 | 17-19 | 17-19 |
| RUP (%) | 36-38 | 37-39 | 38-40 |
| Ca (г/кг) | 6.0-7.0 | 6.0-7.0 | 5.5-6.5 |
| P (г/кг) | 3.5-4.0 | 3.5-4.0 | 3.5-4.0 |
| Mg (г/кг) | 3.5-4.0 | 3.5-4.0 | 3.0-3.5 |
| DCAD (mEq) | +200/+300 | +250/+350 | +250/+400 |

## 3. DCAD протоколы

### 3.1 Стандартный DCAD протокол

**Фаза 1: Подготовка (-21 до -14)**
- Переход на анионную диету
- Начальная доза анионов: 50% от целевой
- Мониторинг DMI

**Фаза 2: Активация (-14 до -7)**
- Полная доза анионов
- Целевой pH мочи: 6.0-6.5
- Корректировка при отклонениях

**Фаза 3: Поддержание (-7 до 0)**
- Стабильный pH мочи
- Постоянный мониторинг
- Подготовка к отёлу

**Фаза 4: Прекращение (день отёла)**
- Отмена анионов
- Переход на лактационный рацион

### 3.2 Практическое применение анионов

| Проблема | Решение |
|----------|---------|
| Низкая палатабельность | Гранулированные продукты, постепенное введение |
| pH мочи > 7.0 | Увеличить дозу анионов на 25% |
| pH мочи < 5.5 | Снизить дозу анионов на 25% |
| Снижение DMI | Проверить дозировку, использовать более палатабельные формы |
| Диарея | Снизить дозу, проверить Mg уровень |

## 4. Метаболические заболевания

### 4.1 Гипокальция (молочная лихорадка)

**Этиология:** Неспособность поддерживать Ca в крови при начале лактации.

**Факторы риска:**
| Фактор | Риск |
|--------|------|
| Возраст | Первотёлки < Многолетки |
| Сезон | Зима > Лето |
| Порода | Jersey > Holstein |
| BCS | > 3.75 или < 3.0 |
| Предшествующая гипокальция | Высокий риск |
| Отсутствие DCAD | Высокий риск |

**Профилактика:**
1. DCAD диета (-50 до -150 mEq/кг) — 80% эффективности
2. Низкий Ca (< 15 г/день) — 50% эффективности
3. Витамин D3 (20 млн IU) — за 2-7 дней до отёла
4. Oral Ca bolus — при отёле

**Лечение:**
- IV Ca (борглюконат): 500-1000 мл 23%
- Мониторинг сердца
- Повторение при необходимости

### 4.2 Кетоз

**Этиология:** Недостаток энергии → мобилизация жира → кетоновые тела.

**Типы:**
| Тип | Причина | Частота |
|-----|---------|---------|
| Тип I (токсическая) | Глубокий энергетический дефицит | Редко |
| Тип II (жировая) | Избыточная мобилизация | Часто |
| Субклинический | Умеренный дефицит | 40-60% стада |

**Диагностика:**
| Показатель | Клинический | Субклинический | Норма |
|------------|-------------|----------------|-------|
| BHBA (ммоль/л) | > 3.0 | 1.2-3.0 | < 1.2 |
| NEFA (мEq/l) | > 1.0 | 0.7-1.0 | < 0.7 |
| Глюкоза (мг/дл) | < 30 | 30-50 | > 50 |
| Ацетон в молоке | +++ | + | - |

**Профилактика:**
- Максимизация DMI
- Энергетическая плотность 1.70-1.80 Mcal/кг
- Мониторинг свежих (BHBA в 7-14 DIM)
- Пропиленгликоль при риске

**Лечение:**
- IV глюкоза: 500 мл 50%
- Пропиленгликоль: 300 мл 2×/день × 3-5 дней
- Кортикостероиды (после 5-7 DIM)
- Витамин B12

### 4.3 Гипомагниемия (тетания)

**Этиология:** Низкий Mg в крови → нервная возбудимость.

**Факторы риска:**
- Высокий K (> 2.5%)
- Низкий Mg (< 3.0 г/кг)
- K:Mg > 20:1
- Зимние месяцы

**Симптомы:**
- Гиперexcitability
- Тетания мышц
- Convulsions
- Смерть при отсутствии лечения

**Профилактика:**
- Mg 4.0-4.5 г/кг СВ
- MgSO₄ или MgCl₂ (высокая биодоступность)
- K < 1.5% в рационе

**Лечение:**
- IV MgSO₄: 200-400 мл 25% (медленно!)
- Подкожно: 200-400 мл 20%
- Перорально: 200-400 г MgO

### 4.4 Диспепсия (переварительные нарушения)

**Причины:**
- Резкая смена рациона
- Низкая кислотность рубца
- Избыток концентратов
- Недостаток грубого корма

**Профилактика:**
- Постепенный переход (7-10 дней)
- Минимум 25-30% NDF от грубого корма
- Физически эффективное волокно
- Мониторинг pH рубца

## 5. Мониторинг переходного периода

### 5.1 Ключевые показатели

| Показатель | Частота | Целевое | Действие |
|------------|---------|---------|----------|
| pH мочи | Ежедневно (сухие) | 6.0-6.5 | Корректировка DCAD |
| DMI | Ежедневно | > 95% от нормы | Проверка рациона |
| BCS | При отёле, 30 DIM | 3.0-3.5 | Корректировка |
| BHBA | 7-14 DIM | < 1.2 ммоль/л | Лечение при > 1.2 |
| NEFA | 7-14 DIM | < 0.7 мEq/л | Мониторинг |
| Ca сыворотки | При отёле | > 8.0 мг/дл | Лечение при < 8.0 |

### 5.2 Групповой мониторинг

| Метрика | Целевое | Действие |
|---------|---------|----------|
| % гипокальции | < 2% | Проверить DCAD |
| % кетоза | < 5% | Проверить энергию |
| % дислокаций | < 8% | Проверить рацион |
| % мастита (0-30 DIM) | < 10% | Проверить иммунитет |
| % метрита | < 10% | Проверить гигиену |

### 5.3 Пороговые значения для вмешательства

| Проблема | Порог | Действие |
|----------|-------|----------|
| Групповая гипокальция | > 5% | Пересмотреть DCAD |
| Субклинический кетоз | > 15% | Увеличить энергию |
| Низкий DMI | < 80% от нормы | Проверить палатабельность |
| Потеря BCS | > 0.5 за 30 дней | Увеличить энергию |

## 6. Практические протоколы

### 6.1 Протокол введения свежих коров

**День отёла (0):**
- Отмена анионов
- IV Ca (при риске)
- Мониторинг DMI

**Дни 1-3:**
- Постепенное увеличение концентратов
- Минимум 30% грубого корма
- Мониторинг BHBA

**Дни 4-7:**
- Достижение полного рациона
- Мониторинг DMI
- Проверка pH рубца

**Дни 7-14:**
- Скрининг BHBA
- Корректировка по результатам
- Мониторинг BCS

**Дни 14-30:**
- Восстановление BCS
- Подготовка к осеменению
- Мониторинг репродукции

### 6.2 Чек-лист переходного периода

**-21 день:**
- [ ] Перевод в группу сухостойных
- [ ] Начало DCAD (если используется)
- [ ] BCS оценка

**-14 день:**
- [ ] Проверка pH мочи
- [ ] Корректировка DCAD
- [ ] Мониторинг DMI

**-7 день:**
- [ ] Финальная корректировка DCAD
- [ ] Подготовка к отёлу
- [ ] Проверка BCS

**День отёла:**
- [ ] Отмена анионов
- [ ] IV Ca (при необходимости)
- [ ] Перевод в свежих

**+7 день:**
- [ ] BHBA тестирование
- [ ] Оценка DMI
- [ ] Мониторинг здоровья

**+14 день:**
- [ ] Повторное BHBA
- [ ] Корректировка рациона
- [ ] Планирование осеменения
