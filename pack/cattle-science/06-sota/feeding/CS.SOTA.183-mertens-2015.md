# 2. РЕЗЮМЕ (Abstract)

Доклад Дэвида Мертенса объясняет переход от метода aNDF (нейтрально-детергентная клетчатка с амилазой) к методу aNDFom (беззольная нейтрально-детергентная клетчатка с амилазой) в анализе кормов. Ключевые моменты: метод aNDFom исключает загрязнение золой, что критически важно для точной оценки энергетической ценности кормов; коррекция на зольное загрязнение обычно превышает коррекцию на белковое загрязнение; использование aNDFom требует применения суммативных уравнений для расчёта TDN и NE вместо регрессионных уравнений, разработанных для NDF.

# 3. ВВЕДЕНИЕ (Introduction)

## 3.1. Полный текст введения [перевод]

**Контекст:**
Клетчатка — это питательная сущность, а не химическая. Она представляет собой неперевариваемую или медленно перевариваемую фракцию корма, занимающую пространство в желудочно-кишечном тракте. Различия в типах животных определяют различия в клетчатке.

**Проблема методологии:**
Методы анализа клетчатки являются эмпирическими — метод определяет измеряемую фракцию клетчатки. Существуют компромиссы для повышения эффективности и точности методов.

**Для жвачных животных:**
Клетчатка в растениях должна содержать целлюлозу, гемицеллюлозу и лигнин в клеточных стенках. Пектин в клеточных стенках легко ферментируется. Перевариваемые целлюлоза и гемицеллюлоза — это органические структурные углеводы, которые обеспечивают энергию жвачным через ЖКК.

**Нейтрально-детергентная клетчатка (NDF)** — лучший показатель общей нерастворимой клетчатки для жвачных.

## 3.2. Ключевые аргументы автора

1. **Различия в методах:**
   - NDF — базовый метод
   - NDR — с остаточной золой
   - aNDF — с амилазой и сульфитом, без зольной коррекции
   - aNDFom — с амилазой и сульфитом, беззольная

2. **Загрязнение клетчатки:**
   - В зависимости от метода, NDF содержит различные загрязнители
   - aNDF (AOAC Official Method) использует амилазу и сульфит, изолируется без зольной обработки
   - Загрязнён меньшим количеством СР и некоторой золой
   - Быстрый метод, так как нет этапа зольной обработки

3. **aNDFom (AOAC Official Method):**
   - Использует амилазу и сульфит
   - Беззольная (ash-free)
   - Загрязнена некоторым количеством СР
   - Только органическое вещество в клетчатке даёт энергию животному

# 6. ПРАКТИЧЕСКОЕ ПРИМЕНЕНИЕ

## 6.1. Алгоритм/рекомендации

**Переход на aNDFom — пошаговый алгоритм:**

1. **Анализ кормов:**
   - Использовать метод aNDFom вместо aNDF
   - Требуется коррекция зольных тиглей после озоления
   - Отдельные NIR-калибровки для aNDF и aNDFom

2. **Расчёт NFC:**
   - NFC = 100 – ash – CP – Fat – aNDFom
   - Более точное определение NFC

3. **Расчёт энергии:**
   - Использовать СУММАТИВНЫЕ уравнения для TDN/NE
   - НЕ использовать регрессионные уравнения, разработанные для NDF
   - TDN1X = tdCP + tdFA×2.25 + tdNDF + tdNFC – 7

4. **Для прогнозирования dNDFom:**
   - Использовать IVNDFomD (in vitro NDFom digestibility)
   - Лигниновое уравнение неприменимо для aNDFom

## 6.2. Калькулятор/инструменты

**Excel-структура для расчёта:**
```
Колонка A: Показатель
Колонка B: Значение

Показатели:
- aNDF (%)
- aNDF ash (%)
- aNDFom (= aNDF - aNDF ash)
- NDICP (%)
- IVNDFomD (%)
- tdNDF = IVNDFomD × aNDFom
- NFC = 100 - ash - CP - Fat - aNDFom
- TDN (суммативное уравнение)
```

# 8. ВЫВОДЫ (Conclusions)

## 8.1. Полный текст выводов [перевод]

**Основные выводы из доклада:**

1. **aNDFom** — наиболее точное измерение клетчатки, которое может генерировать энергию для животного

2. При использовании aNDFom необходимо использовать суммативные уравнения для оценки TDN или NE

3. При использовании aNDFom в суммативных уравнениях, необходимо использовать IVNDFomD для оценки dNDFom

4. aNDFom не следует подставлять в регрессионные уравнения, разработанные для прогнозирования TDN или NE с использованием NDF

5. Рекомендации по оптимальному NDFom и минимальному peNDFom должны быть примерно на 0.95 от текущих рекомендаций для оптимального NDF и минимального peNDF

## 8.2. Ключевые выводы (структурировано)

1. **Методологический вывод:**
   - aNDFom исключает зольное загрязнение, что критично для точной оценки энергии

2. **Практический вывод:**
   - Коррекция на золу обычно превышает коррекцию на белок
   - Особенно важно для кормов с загрязнением почвой

3. **Вывод для расчётов:**
   - Необходим переход на суммативные уравнения
   - Регрессионные уравнения дают ошибочные результаты с aNDFom

4. **Вывод для рекомендаций:**
   - Требуется корректировка рекомендаций по NDF (×0.95)

## 8.3. Ключевые сообщения для лекции

> "Если мы должны корректировать aNDF на NDICP, то мы также должны корректировать aNDF на зольное загрязнение для более точного измерения клетчатки."

> "Правильное измерение и распределение органического вещества критично, и загрязнение почвой aNDF может быть значительным."

> "Парадоксально, aNDF даёт лучшие оценки NE при использовании регрессионных уравнений, но по неправильным причинам — потому что они разработаны для проб с типичной собственной золой."

# 10. FAQ

**Q1: Что такое aNDFom и чем он отличается от aNDF?**
A: aNDFom (amylase-treated Neutral Detergent Fiber organic matter) — это беззольная клетчатка, обработанная амилазой. Отличается от aNDF отсутствием зольной коррекции, что даёт более точную оценку органической части клетчатки.

**Q2: Почему коррекция на золу важнее коррекции на белок?**
A: Потому что корма, содержащие клетчатку, обычно даются для обеспечения энергией, которая получается только из органического вещества. Зольное загрязнение может достигать 2-7% для различных кормов.

**Q3: Как перейти с aNDF на aNDFom в лаборатории?**
A: Необходимо: 1) Добавить этап зольной коррекции, 2) Разработать новые NIR-калибровки, 3) Обновить методические указания.

**Q4: Почему нельзя использовать регрессионные уравнения с aNDFom?**
A: Регрессионные уравнения были разработаны для NDF и aNDF, которые содержат зольное загрязнение. Использование aNDFom (без золы) даёт завышенные значения NE.

**Q5: Как скорректировать рекомендации по NDF для aNDFom?**
A: Рекомендуется умножить текущие рекомендации по оптимальному NDF и минимальному peNDF на 0.95.

**Q6: Влияет ли метод на прогнозирование дайджестибильности клетчатки?**
A: Да, для aNDFom необходимо использовать IVNDFomD (in vitro NDFom digestibility) вместо лигниновых уравнений.

**Q7: Как aNDFom влияет на расчёт NFC?**
A: NFC = 100 – ash – CP – Fat – aNDFom. Использование aNDFom даёт более высокие значения NFC, что более точно отражает реальное содержание неструктурных углеводов.
