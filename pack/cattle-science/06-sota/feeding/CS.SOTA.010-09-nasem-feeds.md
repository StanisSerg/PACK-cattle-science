## 1. Методы анализа кормов

### 1.1 Van Soest анализ клетчатки

**Фракции клетчатки:**

| Фракция | Метод                             | Содержание                         |
| ------- | --------------------------------- | ---------------------------------- |
| **NDF** | Нейтрально-детергентная клетчатка | Целлюлоза + гемицеллюлоза + лигнин |
| **ADF** | Кислотно-детергентная клетчатка   | Целлюлоза + лигнин                 |
| **ADL** | Лигнин (кислотный)                | Лигнин                             |

**Расчётные фракции:**

```
Гемицеллюлоза = NDF - ADF
Целлюлоза = ADF - ADL
```

### 1.2 NDFD — NDF Digestibility

**Определение:** Усвояемость NDF в рубце (in vitro или in situ)

| Показатель | Описание                  | Типичный диапазон |
| ---------- | ------------------------- | ----------------- |
| NDFD 30h   | Усвояемость за 30 часов   | 40-60%            |
| NDFD 240h  | Потенциальная усвояемость | 60-80%            |
| kd (NDF)   | Скорость разрушения       | 2-6 %/час         |

**Интерпретация NDFD:**

| NDFD 30h | Качество | Применение       |
| -------- | -------- | ---------------- |
| < 40%    | Низкое   | Сухостойные      |
| 40-50%   | Среднее  | Поздняя лактация |
| 50-60%   | Хорошее  | Средняя лактация |
| > 60%    | Отличное | Ранняя лактация  |

### 1.3 NFC — Non-Fiber Carbohydrates

**Формула расчёта:**

```
NFC (% СВ) = 100 - (NDF + CP + EE + Ash)
```

**Или с корректировкой:**

```
NFC (% СВ) = 100 - (NDF - NDF-CP + CP + EE + Ash)
```

**Состав NFC:**

| Компонент | Доля в NFC | Скорость ферментации |
| --------- | ---------- | -------------------- |
| Сахара    | 10-30%     | Очень быстрая        |
| Крахмал   | 40-70%     | Быстрая              |
| Пектины   | 10-30%     | Средняя              |
| β-глюканы | 5-15%      | Средняя              |

### 2.3 Физическая эффективность NDF (peNDF)

**Определение:** Способность корма стимулировать жвачку и выработку слюны.

| Фактор         | Влияние на peNDF   |
| -------------- | ------------------ |
| Длина частиц   | + (длиннее = выше) |
| Твёрдость      | + (твёрже = выше)  |
| NDF содержание | + (больше = выше)  |
| Увлажнённость  | - (влажнее = ниже) |

**Целевые значения peNDF:**

| Стадия           | peNDF (% СВ) | Диапазон |
| ---------------- | ------------ | -------- |
| Ранняя лактация  | 21-23        | 19-25    |
| Средняя лактация | 23-25        | 21-27    |
| Сухостойные      | 25-28        | 23-30    |

## 6. Практические рекомендации

### 6.1 Выбор грубых кормов

| Стадия           | Рекомендуемые корма                        | Ограничения     |
| ---------------- | ------------------------------------------ | --------------- |
| Ранняя лактация  | Высокое качество люцерны, кукурузный силос | NDF < 28-30% СВ |
| Средняя лактация | Смесь люцерны и трав, силос                | NDF 30-32% СВ   |
| Поздняя лактация | Травяное сено, зерновой силос              | NDF 32-35% СВ   |
| Сухостойные      | Грубое сено, солома                        | NDF 35-40% СВ   |

### 6.2 Контроль качества силоса

| Параметр         | Целевой    | Действие при отклонении    |
| ---------------- | ---------- | -------------------------- |
| pH               | 3.7-4.2    | > 4.5 → риск порчи         |
| NH3-N            | < 10% от N | > 12% → плохая консервация |
| Уксусная кислота | < 2%       | > 3% → риск ацидоза        |
| Масляная кислота | < 0.5%     | > 1% → клостридии          |
| Сухое вещество   | 30-35%     | < 28% → вытекание сока     |

### 6.3 Формирование рациона

**Правило 60:40:**

- 60% рациона — грубые корма (NDF источник)
- 40% рациона — концентраты (энергия, протеин)

**Максимумы концентратов:**

| Стадия           | Макс. концентраты | Причина              |
| ---------------- | ----------------- | -------------------- |
| Ранняя лактация  | 55-60%            | Пик молока           |
| Средняя лактация | 50-55%            | Баланс               |
| Сухостойные      | 20-25%            | Профилактика ацидоза |
