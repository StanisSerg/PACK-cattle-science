## 1. Энергетические системы

### 1.1 NEL — Net Energy Lactation

**Применение:** Молочные коровы (основная система)

**Определение:** Энергия, доступная для поддержания жизнедеятельности и продукции молока после вычета тепловых потерь.

**Формула расчёта требований:**
```
NEL_total = NEL_maintenance + NEL_lactation + NEL_pregnancy + NEL_activity + NEL_reserves
```

### 1.2 NEM — Net Energy Maintenance

**Применение:** Сухие коровы, телки, быки

**Формула:**
```
NEM (Mcal/день) = 0.08 × BW^0.75
```
Где BW — масса тела в кг

### 1.3 NEg — Net Energy Gain

**Применение:** Растущий молодняк

**Формула:**
```
NEg (Mcal/кг прироста) = 0.037 × BW^0.75 × ADG^1.097
```
Где ADG — average daily gain, кг/день

## 2. NEL требования для молочных коров

### 2.1 NEL Maintenance

| Масса тела (кг) | NEL maint (Mcal/день) |
|-----------------|----------------------|
| 500 | 8.4 |
| 550 | 9.0 |
| 600 | 9.6 |
| 650 | 10.2 |
| 700 | 10.8 |
| 750 | 11.4 |

**Формула:** NEL_maint = 0.08 × BW^0.75 × 1.2

Коэффициент 1.2 — корректировка на активность в стойле.

### 2.2 NEL для молока

**Базовая формула:**
```
NEL_milk (Mcal/кг) = 0.0929 × Fat% + 0.0547 × CP% + 0.0395 × Lactose%
```

**Упрощённая формула:**
```
NEL_milk (Mcal/кг) ≈ 0.74 + 0.096 × Fat% + 0.047 × Protein%
```

**Стандартные значения:**

| Компонент | NEL (Mcal/кг) |
|-----------|---------------|
| Жир 4.0%, белок 3.3% | 0.72 |
| Жир 4.5%, белок 3.5% | 0.77 |
| Жир 5.0%, белок 3.8% | 0.82 |

### 2.3 NEL для беременности

| Триместр | NEL (Mcal/день) |
|----------|-----------------|
| 1-2 | 0 |
| 3 (начало) | 2.0 |
| 3 (середина) | 4.0 |
| 3 (конец) | 6.0-8.0 |
| Близнецы | × 1.5 |

**Формула (последний триместр):**
```
NEL_preg = 0.025 × exp(0.035 × DaysPregnant - 6.5)
```

## 4. Корректировки NEL

### 4.1 Температура окружающей среды

#### Холод (< 5°C)

| Температура | Корректировка |
|-------------|---------------|
| 0°C | +10% |
| -10°C | +20% |
| -20°C | +30% |

**Формула:**
```
Correction = 0.0007 × (5 - T)^2
```
Где T — температура в °C

#### Жара (> 25°C)

| Температура | Корректировка |
|-------------|---------------|
| 25°C | 0% |
| 30°C | -5% |
| 35°C | -15% |
| 40°C | -30% |

**Формула:**
```
Correction = -0.0065 × (T - 25)^2
```

### 4.2 Body Condition Score (BCS)

| BCS | Статус | Корректировка |
|-----|--------|---------------|
| < 2.0 | Слишком худой | +10% |
| 2.0-2.5 | Худой | +5% |
| 2.5-3.5 | Оптимальный | 0% |
| 3.5-4.0 | Полный | -5% |
| > 4.0 | Ожирение | -10% |

### 4.3 Активность

| Тип активности | Добавка к NEL |
|----------------|---------------|
| Ходьба (1 км) | +0.57 Mcal |
| Пастбище (8 часов) | +15-20% к maintenance |
| Холмистая местность | +10% |
| Грязь/снег | +10-15% |

### 4.4 Генетический потенциал

| Уровень продуктивности | Корректировка |
|------------------------|---------------|
| Низкий (< 25 кг) | -5% |
| Средний (25-40 кг) | 0% |
| Высокий (40-50 кг) | +5% |
| Элитный (> 50 кг) | +10% |

## 5. Потребление сухого вещества (DMI)

### 5.1 Факторы, влияющие на DMI

| Фактор | Влияние |
|--------|---------|
| Масса тела | +0.5% на 10 кг |
| Продукция | +0.4 кг СВ на кг молока |
| Жирность | +0.1 кг СВ на 0.1% жира |
| Недоедание | -10-20% при ограничении |
| Жара (> 25°C) | -10-30% |
| Кислотность | -5-15% при pH < 6.0 |

### 5.2 Прогнозирование DMI

**Формула NASEM 2021:**
```
DMI (кг/день) = (0.0185 × BW + 0.305 × FCM) × corr
```

Где:
- BW — масса тела, кг
- FCM — 4% fat-corrected milk, кг
- corr — корректировочный коэффициент

**FCM формула:**
```
FCM (кг) = 0.4 × Milk (кг) + 15 × Fat (кг)
```

### 5.3 DMI по стадиям (корова 650 кг)

| DIM | DMI (кг/день) | % от BW | Примечание |
|-----|---------------|---------|------------|
| 0-7 | 10-12 | 1.5-1.8 | Постпартум |
| 7-14 | 14-16 | 2.2-2.5 | Восстановление |
| 14-30 | 18-22 | 2.8-3.4 | Рост потребления |
| 30-100 | 22-26 | 3.4-4.0 | Пиковое |
| 100-200 | 24-28 | 3.7-4.3 | Стабильное |
| 200-305 | 22-26 | 3.4-4.0 | Снижение |
| > 305 | 18-22 | 2.8-3.4 | Поздняя лактация |
| Сухие (-60 до -21) | 12-14 | 1.8-2.2 | Ранний сухостой |
| Сухие (-21 до 0) | 12-14 | 1.8-2.2 | Поздний сухостой |

### 5.4 Пиковое потребление

| Параметр | Значение |
|----------|----------|
| Время пика | 10-14 недель после отёла |
| Максимум DMI | 3.5-4.5% от массы тела |
| Лаг от пика молока | 4-8 недель |
| Недоедание в пике | 5-10% энергетический дефицит |

## 7. Энергетический баланс

### 7.1 Расчёт баланса

```
Energy Balance (Mcal/день) = NEL intake - NEL required
```

### 7.2 Интерпретация

| Баланс (Mcal/день) | Статус | Действие |
|-------------------|--------|----------|
| > +5 | Позитивный | Восстановление BCS |
| 0 до +5 | Нейтральный | Оптимально |
| -5 до 0 | Лёгкий дефицит | Приемлемо в ранней лактации |
| -10 до -5 | Умеренный дефицит | Мониторинг NEFA |
| < -10 | Тяжёлый дефицит | Риск метаболических |

### 7.3 NEFA и энергетический статус

| NEFA (mEq/L) | Интерпретация |
|--------------|---------------|
| < 0.3 | Норма |
| 0.3-0.5 | Лёгкая мобилизация |
| 0.5-0.7 | Умеренная мобилизация |
| > 0.7 | Тяжёлая мобилизация, риск |

## 8. Практические примеры

### Пример 1: Ранняя лактация

**Условия:**
- Корова 650 кг
- 30 DIM
- 40 кг молока, 4.2% жира
- DMI = 22 кг

**Расчёт:**
```
NEL_maint = 10.2 Mcal
NEL_milk = 40 × (0.74 + 0.096×4.2 + 0.047×3.3) = 40 × 0.82 = 32.8 Mcal
NEL_total = 10.2 + 32.8 = 43.0 Mcal

NEL рациона = 43.0 / 22 = 1.95 Mcal/кг СВ
```

**Вывод:** Требуется высокоэнергетический рацион (кукуруза, жиры).

### Пример 2: Поздний сухостой

**Условия:**
- Корова 700 кг
- -14 дней до отёла
- Беременность: 3 триместр
- DMI = 13 кг

**Расчёт:**
```
NEL_maint = 10.8 Mcal
NEL_preg = 6.0 Mcal
NEL_total = 16.8 Mcal

NEL рациона = 16.8 / 13 = 1.29 Mcal/кг СВ
```

**Вывод:** Стандартный рацион с умеренной энергией.
