## 6. Качество и вариабельность

### 6.1 Факторы вариабельности

| Фактор | Влияние | Мониторинг |
|--------|---------|------------|
| Исходное сырьё | ++++ | Сертификат анализа |
| Производственный процесс | +++ | Поставщик, спецификация |
| Сезонность | ++ | История партий |
| Хранение | +++ | Влажность, температура |

### 6.2 Контроль качества

| Параметр | Частота анализа | Критичность |
|----------|-----------------|-------------|
| Сухое вещество | Каждая партия | ++++ |
| СП | Каждая партия | +++ |
| Жир (DDGS) | Каждая партия | ++++
| NDF | Еженедельно | ++ |
| Минералы | Ежемесячно | ++ |
| Токсины | При подозрении | ++++ |

### 6.3 Экономика

| Фактор | Влияние на стоимость |
|--------|----------------------|
| Цена на энергию | +++ |
| Цена на протеин | +++ |
| Транспорт | ++ |
| Хранение | ++ |
| Вариабельность | + (риск) |

**Расчёт ценности:**
```
Ценность ($/т) = (NEL × $/Mcal) + (CP × $/кг CP) + (RUP × $/кг RUP)
```

## 7. Практические рекомендации

### 7.1 Введение побочных продуктов

| Этап | Действие | Длительность |
|------|----------|--------------|
| 1 | 5-10% рациона | 7-10 дней |
| 2 | 10-15% рациона | 7-10 дней |
| 3 | Целевой уровень | Постоянно |

### 7.2 Ограничения по категориям

| Побочный продукт | Ограничивающий фактор | Максимум |
|------------------|----------------------|----------|
| DDGS | Жир, сера | 20-25% СВ |
| Пивная дробина | Влажность, NDF | 30-40 кг/день |
| Свёкловичный жом | Влажность, сахар | 25-30 кг/день |
| Рапсовый шрот | Glucosinolates | 20-25% СВ |
| Хлопковый шрот | Gossypol | 15% СВ |

### 7.3 Проблемы и решения

| Проблема | Причина | Решение |
|----------|---------|---------|
| Снижение DMI | Высокий уровень DDGS | Снизить до 15% |
| Мягкий навоз | Избыток жира | Обезжиренные DDGS |
| Низкий жир молока | Высокий NFC | Увеличить NDF |
| Высокая MUN | Избыток RDP | Сбалансировать протеин |
