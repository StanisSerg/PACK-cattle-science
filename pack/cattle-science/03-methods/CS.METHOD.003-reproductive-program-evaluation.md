# CS.METHOD.003: Оценка и проектирование репродуктивной программы

> **Назначение:** Систематическая оценка текущей репродуктивной программы и проектирование оптимальной стратегии на основе экономического моделирования.
>
> **Целевая аудитория:** Консультанты, технологи, ветеринарные специалисты
>
> **Уровень:** Средний (требует понимания основ экономики и репродукции)
>
> **Время применения:** 2-4 часа на полную оценку

---

## 1. Описание метода

### 1.1. Что оценивает метод

Метод оценивает репродуктивную программу по трём измерениям:

| Измерение | Что оценивается | Ключевые метрики |
|-----------|-----------------|------------------|
| **Эффективность** | Насколько хорошо работает текущая программа | 21-d PR, P/AI, Service Rate |
| **Экономика** | Прибыльность программы | NR, IOFC, RC, RMC |
| **Устойчивость** | Обеспеченность сменой | HSR, оборачиваемость, достаточность замен |

### 1.2. Основа метода

Метод интегрирует две проверенные модели:
- **Cabrera 2012** — оценка ценности коровы и оптимальная политика замены
- **Lauber 2025** — оптимальный период осеменения (IEP) для разных стратегий

### 1.3. Результат применения

- Оценка текущей программы (сильные/слабые стороны)
- Сравнение с эталонными сценариями
- Рекомендации по оптимизации
- Экономический эффект от изменений

## 2. Алгоритм метода

### Этап 0: Подготовка (15 мин)

**Необходимые данные:**

```

├── Общее количество коров
├── Средний удой (кг/корова/год)
├── 21-дневная стельность (%)
├── Концепция на осеменение (%)
├── Доля осеменённых коров (%)
├── Потери стельности (%)
└── Средние дни до зачатия

Параметры стада:
├── Оборачиваемость (%/год)
├── Выживаемость телочек (%)
├── Распределение по лактациям
└── Средний живой вес

Экономические параметры:
├── Цена молока ($/кг)
├── Затраты на корм ($/кг СВ)
├── Стоимость замещения ($)
├── Стоимость семени ($/доза)

├── Стратегия распределения спермы

```

### Этап 1: Расчёт ключевых метрик (20 мин)

#### 1.1. 21-дневная стельность (21-d PR)

```
21-d PR = Service Rate × P/AI

Где:
- Service Rate = (осеменённые за 21 дня / доступные для осеменения) × 100%
- P/AI = (подтверждённые стельности / осеменения) × 100%
```

**Интерпретация:**

| 21-d PR | Категория | Действие |
|---------|-----------|----------|
| <20% | Low | Требует немедленного вмешательства |
| 20-25% | Below average | Значительный потенциал улучшения |
| 25-30% | Average | Умеренный потенциал |
| 30-35% | Above average | Небольшие улучшения |
| ≥35% | High | Поддержание текущего уровня |

#### 1.2. Экономические показатели (упрощённые)

```
IOFC (Income Over Feed Cost) = Доход от молока − Затраты на корм

Примерная оценка NR (Net Return):
NR ≈ IOFC + Доход от телят − Затраты на замену − Затраты на репродукцию
```

**Быстрая оценка (для консультанта):**

| Показатель | Формула | Норма |
|------------|---------|-------|
| IOFC | $/корова/год | >$2,000 |
| Затраты на репродукцию | $/корова/год | <$150 |
| Затраты на замену | $/корова/год | <$400 |

#### 1.3. Оценка смены

```
Проверка достаточности замен:

Требуемая смена = Оборачиваемость стада (%)

Доступная смена = HSR × (количество рождённых телочек / количество коров)

Если Доступная смена < Требуемая смена → необходима покупка замен
```

### Этап 5: Разработка рекомендаций (30 мин)

#### 5.1. Матрица решений

**Если 21-d PR < 25% (Low):**

| Приоритет | Действие | Ожидаемый эффект | Затраты | ROI |

| 3 | Улучшение диагностики | +2-3 п.п. 21-d PR | $1000-2000 | Средний |

**Если 25% ≤ 21-d PR < 30% (Average):**

| Приоритет | Действие | Ожидаемый эффект | Затраты | ROI |
|-----------|----------|------------------|---------|-----|
| 1 | Оптимизация тайминга осеменения | +2-3 п.п. 21-d PR | $500-1000 | Высокий |
| 2 | Анализ и коррекция P/AI | +2-4 п.п. 21-d PR | $1000-2000 | Высокий |

| Приоритет | Действие | Ожидаемый эффект | Затраты | ROI |
|-----------|----------|------------------|---------|-----|

| 2 | Переход на sexed+beef при подходящих условиях | +$50/корова/год | $3000-5000 | Средний |
| 3 | Поддержание текущего уровня | — | $500-1000 | — |

#### 5.2. Расчёт экономического эффекта

```
Формула расчёта прироста NR от повышения 21-d PR:

Прирост NR ($/корова/год) = Δ21-d PR (п.п.) × $3-7

Где:
- $3-7 — эффект от 1 п.п. 21-d PR (зависит от текущего уровня и типа спермы)
- Для низких 21-d PR эффект выше ($5-7)
- Для высоких 21-d PR эффект ниже ($2-3)

Общий эффект для стада:
Общий прирост = Прирост NR × Количество коров

Пример:
- Стадо 500 коров
- Текущая 21-d PR: 22%
- Целевая 21-d PR: 28%
- Δ21-d PR: 6 п.п.
- Прирост NR: 6 × $5 = $30/корова/год
- Общий прирост: $30 × 500 = $15,000/год
```

## 4. Примеры применения

### Пример 1: Стадо с низкой 21-d PR

**Исходные данные:**
- Коров: 300
- 21-d PR: 18%
- Service Rate: 45%
- P/AI: 40%
- HSR: 82%
- IEP: 200 дней
- Тип спермы: conventional

**Оценка:**
- Категория: Low 21-d PR
- Проблема: Низкий Service Rate (обнаружение охоты)
- Потенциал: +12 п.п. 21-d PR (до 30%)

**Рекомендации:**
1. Обучение персонала по обнаружению охоты (+5 п.п. SR)
2. Внедрение PreSynch-Ovsynch (+10 п.п. SR)
3. Оптимизация IEP до 170 дней

**Экономический эффект:**
- Инвестиции: $5,000
- Прирост 21-d PR: 12 п.п. × $6 = $72/корова/год
- Общий эффект: $72 × 300 = $21,600/год
- ROI: 332%
- Окупаемость: 3 месяца

### Пример 2: Стадо с высокой 21-d PR

**Исходные данные:**
- Коров: 500
- 21-d PR: 33%
- HSR: 88%
- IEP: 150 дней
- Тип спермы: conventional

**Оценка:**
- Категория: High 21-d PR
- Проблема: IEP выше оптимального (должно быть 80 дней)
- Потенциал: оптимизация IEP + переход на sexed+beef

**Рекомендации:**
1. Сократить IEP до 80 дней
2. Перейти на sexed+beef (HSR позволяет)

**Экономический эффект:**
- Инвестиции: $8,000
- Прирост NR: $51/корова/год (sexed+beef) + $20 (IEP) = $71
- Общий эффект: $71 × 500 = $35,500/год
- ROI: 344%
- Окупаемость: 3 месяца

## 5. Ограничения метода

### 5.1. Методологические ограничения

- Упрощённая оценка NR (не заменяет полное экономическое моделирование)
- Средние эффекты от изменений (фактический эффект может отличаться)
- Не учитывает сезонные колебания

### 5.2. Применимость

**Метод применим для:**
- ✅ Голштинских стад с беспривязным содержанием
- ✅ Стад с TMR-кормлением
- ✅ Стад с регулярным ветеринарным мониторингом

**Требует адаптации для:**
- ⚠️ Других пород (Jersey, кроссбриды)
- ⚠️ Привязного содержания
- ⚠️ Экстенсивных систем

**Не применим для:**
- ❌ Стад без системы учёта
- ❌ Стад с хроническими проблемами здоровья
