## 1. ДИАГНОСТИКА КЕТОЗА

### 1.1 Лабораторные методы

#### Кровь: BHB (Золотой стандарт)
| Параметр | Значение |
|----------|----------|
| **Метод** | Фотометрический, портативные анализаторы (Precision Xtra, Nova Vet) |
| **Порог субклинического** | ≥1.0–1.4 mmol/L (вариабельность между исследованиями) |
| **Порог клинического** | ≥3.0 mmol/L |
| **Время** | 5–10 минут |
| **SoTA** | 027, 032, 055, 058, 065, 066, 067 |

**Разногласия в порогах (SoTA):**
| Исследование | Порог BHB | Примечание |
|--------------|-----------|------------|
| LeBlanc 2010 | ≥1.0 mmol/L | Чувствительность |
| Oetzel 2004 | ≥1.2 mmol/L | Баланс чувствительности/специфичности |
| McArt 2011 | ≥1.4 mmol/L | Специфичность |

**Интерпретация (Graef 2025, Lin 2025):**
- BHB в крови — основной предиктор для ML-моделей
- Комбинация с NEFA повышает точность предсказания

#### Кровь: NEFA
| Параметр | Значение |
|----------|----------|
| **Метод** | Ферментативный колориметрический |
| **Порог** | ≥0.3–0.4 mmol/L |
| **Преимущество** | Ранний маркер (предшествует BHB на 2–3 дня) |
| **SoTA** | 055, 065, 066 |

---

#### Молоко: Кетолактация
| Параметр | Значение |
|----------|----------|
| **Метод** | Тест-полоски (KetoCheck, Ketolac) |
| **Порог** | ≥50–100 μmol/L (зависит от теста) |
| **Преимущество** | Неинвазивный, при рутинном доении |
| **SoTA** | 032 (Kowalski 2021) |

### 1.2 Поведенческие методы

#### Мониторинг жвачки (Antanaitis 2020)
| Параметр | Значение |
|----------|----------|
| **Система** | RumiWatch, CowManager, Nedap |
| **Показатель** | Время жвачки, число жеваний |
| **Порог** | Снижение на 30–50% от базовой линии |
| **Преимущество** | Раннее выявление (за 15 дней до клиники) |
| **SoTA** | 027 |

**Ключевые данные:**
- Коровы с SCK: время жвачки в 1.48 раза меньше (p<0.001)
- Число жеваний: в 1.68 раза меньше
- Различия достоверны за 15 дней до диагностики BHB

---

#### Мониторинг активности
| Параметр | Значение |
|----------|----------|
| **Показатели** | Активность, время лежания, время питья |
| **Порог** | Снижение активности на 20–30% |
| **SoTA** | 027 |

## 2. ЛЕЧЕНИЕ КЕТОЗА

### 2.1 Пероральная терапия

#### Глицерол (Kupczyński 2020)
| Параметр | Значение |
|----------|----------|
| **Доза** | 1–3 L однократно или 300–500 мл/день |
| **Механизм** | Глюкогенный прекурсор → повышение глюкозы на 16–25% |
| **Эффект** | Снижение BHB и NEFA через 0.5–2 часа |
| **Длительность** | 3–5 дней |
| **SoTA** | 058 |

**Профилактическое применение:**
- До 10–15% сухого вещества рациона
- Замена зерна (кукурузы) без снижения продуктивности
- Возможное снижение жирности молока

---

#### Пропиленгликоль (PG)
| Параметр | Значение |
|----------|----------|
| **Доза** | 300–500 мл/день |
| **Механизм** | Глюкогенный прекурсор |
| **Сравнение с глицеролом** | Схожая эффективность |
| **SoTA** | 058 |

---

#### Глюкоза/декстроза
| Параметр | Значение |
|----------|----------|
| **Доза** | 500 г в растворе |
| **Механизм** | Прямое повышение гликемии |
| **Ограничение** | Кратковременный эффект |

### 2.3 Профилактика

#### Управление рационом (Drackley 1999)
| Мера | Описание |
|------|----------|
| **Контроль энергии** | Постепенное повышение концентратов |
| **Добавочные жиры** | Осторожность в ранней лактации (риск нарушения баланса) |
| **Переходный период** | Мониторинг DMI, BCS |

---

#### Мониторинг
| Метод | Частота | Целевая группа |
|-------|---------|----------------|
| BHB крови | Еженедельно (1–4 недели) | Все коровы или высокорисковые |
| Жвачка | Непрерывно (автоматика) | Все коровы |
| FTIR | При каждом доении | Все коровы (если доступно) |

## 3. АЛГОРИТМ ДЕЙСТВИЙ

### 3.1 Скрининг (Day 3–14 postpartum)
```
Жвачка снижена? ──→ Да ──→ BHB тест
      ↓                      ↓
     Нет              BHB ≥1.2 mmol/L?
                           ↓
                    Да ──→ Лечение (глицерол/PG)
                           ↓
                          Нет ──→ Мониторинг
```

### 3.2 Лечение
```
BHB 1.2–3.0 mmol/L (SCK):
  → Пероральная терапия (глицерол 1–2 L)
  → Мониторинг через 48 часов

BHB >3.0 mmol/L (Clinical):
  → Глицерол + дексаметазон
  → При отсутствии аппетита: Глюкоза В/В
  → Ветеринарный осмотр
```
