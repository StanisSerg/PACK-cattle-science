## [CS.METHOD.006] Системная оценка кормления при визите на ферму

### Что это такое

Метод полевой оценки кормового блока фермы. Собирает данные из трёх источников — физической среды (стойла, кормушки), учётных систем (удой, рацион) и лаборатории (пробы ТМР) — и связывает их в единую цепочку доказательств.

> **TMR** = total mixed ration = полнорационный кормосмесь.

### Зачем нужен

- Не принимать цифры из программ за реальность без проверки.
- Различать «рассчитали», «заложили» и «скормили».
- Не делать выводы по средним, пока не посмотрели распределение.
- Сформулировать рекомендации на основе evidence, а не впечатлений.

### Что берём в работу

| Показатель | Что это | Обязательно? | Откуда |
|------------|---------|--------------|--------|
| Количество лежачих мест | Физическая ёмкость группы | Да | Визуальный осмотр, план фермы |
| Количество коров в группе | Фактическая загрузка | Да | Учётная система / пересчёт |
| Фронт кормления | Доступ к корму на голову | Да | Замер рулеткой |
| Удой из программы | Publication face продуктивности | Да | AMS / DC305 / DairyComp |
| Фактическое молоко в холодильнике / на площадке | Проверка publication face | Да | Производственные записи |
| Расчётный рацион | WorkPlan / Model claim | Да | Программа нормирования |
| Фактическая выдача корма | Work record | Да | Журнал кормления, весы |
| Проба ТМР | Физический образец смеси | Рекомендуется | Отбор во время выдачи |
| Лабораторный анализ ТМР | Проверка состава | Рекомендуется | Лаборатория кормов |
| Погрешность доильного оборудования | Measurement uncertainty | Да | Калибровка, паспорт |
| Показатели здоровья группы | Confounders | Рекомендуется | Ветеринарные записи |
| Климат / вода / социальная иерархия | Confounders | Рекомендуется | Осмотр, записи |

> **AMS** = automated milking system = автоматическая доильная система.

### Ключевые различия

- **Data vs Episteme:** «47 лежачих мест» — data. «Группе не хватает места» — episteme (description + claim). «Нужно добавить места» — WorkPlan.
- **System vs Publication face:** Система — коровы и молоко в холодильнике. Publication face — цифра удоя на экране. Проверяем второе через первое.
- **WorkPlan vs Work:** Рацион в программе — WorkPlan. Фактически выданный корм и порядок загрузки миксера — Work.
- **Aggregate vs Distribution:** Средний удой группы — aggregate. Распределение удоев (10-й, 50-й, 90-й перцентиль) — distribution. Решения принимаем по distribution.
- **Model vs Reality:** Расчётный состав ТМР — model. Лабораторный анализ пробы — reality check.
- **Justified Decision vs Opinion:** «Снижу CP до 15,5%» — opinion, пока нет источника + прогноза + срока + критерия отката. Обоснованное решение — candidate claim с source, predicted outcome, measurement plan, decision rule.

> **DIM** = days in milk = дни лактации.

## Алгоритм: оценка кормления при визите на ферму

### Фаза 0. Подготовка (до визита)

#### Шаг 0.1. Определи bounded context

| Параметр | Что фиксируем | Пример |
|----------|---------------|--------|
| Группа | Название, DIM-диапазон, парность | Дойные 60–150 DIM, 2+ отёла |
| Период | Даты, сезон, погода | Июнь 2026, THI > 72 |
| Оборудование | Доильная система, весы, миксер | Ротор, весы Trimec, миксер 12 м³ |
| Роли | Кто даёт data, кто принимает решения | Feed Manager + Herd Manager |

> **THI** = temperature-humidity index = индекс температуры и влажности.

#### Шаг 0.2. Запроси предварительные данные

- [ ] Расчётный рацион из программы (WorkPlan).
- [ ] График выдачи и загрузки миксера (Work record).
- [ ] Удой за 7–14 дней по группе и по индивидуальным коровам.
- [ ] Записи о здоровье группы (кетоз, ацидоз, мастит, хромота).
- [ ] История изменений рациона за последние 30 дней.

### Фаза 1. Сбор сырых данных на ферме

#### Шаг 1.1. Физическая среда

| Показатель | Как измеряем | Норма / цель |
|------------|--------------|--------------|
| Лежачие места | Количество / группу | ≥ 100% от головы (идеально ≥ 110%) |
| Фронт кормления | См на голову | ≥ 60–70 см на дойную корову |
| Длина кормушки | Общая / на голову | Не менее нормы для породы |
| Вода | Количество поилок, поток, чистота | ≥ 10 см поилочного фронта на голову |

#### Шаг 1.2. Состояние коров

| Показатель | Что смотрим | Зачем |
|------------|-------------|-------|
| Заполненность стойл | Сколько коров лежит / стоит | Комфорт и доступ к местам |
| Конкуренция у кормушки | Доминантные vs подчинённые | Равномерность потребления |
| Состояние копыт / вымени | Визуально | Confounders для продуктивности |

#### Шаг 1.3. Корм и кормление

| Показатель | Что смотрим | Как фиксируем |
|------------|-------------|---------------|
| Внешний вид ТМР | Однородность, запах, плесень | Фото + описание |
| Остатки корма | Сортировка, длина шнека | Фото + замер |
| Время выдачи | Реальное vs плановое | Запись |
| Фактическая масса выдачи | По весам | Сравнение с WorkPlan |

---

### Фаза 2. Верификация (model vs reality)

#### Шаг 2.1. Проверь publication face

| Что проверяем | Publication face | System | Способ проверки |
|---------------|------------------|--------|-----------------|
| Удой | Цифра в программе | Молоко в холодильнике | Сравнить за период |
| Погрешность | Не указана | Диапазон ±X% | Паспорт калибровки |
| Количество коров | Запись в системе | Фактический пересчёт | Пересчёт в группе |

#### Шаг 2.2. Проверь WorkPlan vs Work

| Элемент | WorkPlan (расчёт) | Work (факт) | Проверка |
|---------|-------------------|-------------|----------|
| Состав ТМР | Ингредиенты, кг СВ | Загрузка в миксер | Журнал + весы |
| Подача | Кг СВ на голову | Выданная масса | Весы / остатки |
| Распределение | Равномерно по кормушке | Визуально / пробоотбор | Осмотр + пробы |

#### Шаг 2.3. Проверь модель состава

| Что | Model claim | Reality check |
|-----|-------------|---------------|
| Сухое вещество | Расчётное из программы | Лаборатория пробы ТМР |
| Протеин / энергия | Расчётное | Лаборатория пробы ТМР |
| Структура / длина частиц | Расчётное / визуально | Ситовой анализ / Penn State Separator |

---

### Фаза 3. Анализ распределения и confounders

#### Шаг 3.1. Распределение продуктивности

| Уровень | Что смотрим | Почему важно |
|---------|-------------|--------------|
| Стадо | Средний удой | Общая картина |
| Группа | Средний + стандартное отклонение | Скрытые проблемы |
| Индивид | 10-й / 50-й / 90-й перцентиль | Конкретные коровы для вмешательства |

#### Шаг 3.2. Confounders

| Фактор | Что проверяем | Как влияет на вывод |
|--------|---------------|---------------------|
| Жара / THI | Температура, влажность, вентиляция | Снижает DMI, не вина рациона |
| Здоровье | Кетоз, ацидоз, мастит, хромота | Снижает удой независимо от корма |
| Вода | Доступ, качество, температура | Ограничивает потребление сухого вещества |
| Социальная иерархия | Доступ к корму и лежакам | Часть группы недополучает рацион |
| DIM и парность | Фаза лактации, первородки | Разные нормы для разных подгрупп |

---

### Фаза 4. Синтез и формулировка выводов

#### Шаг 4.1. Проверь обоснование каждой рекомендации

Кормовик должен уметь назвать источник. Если не может — решение оформляется как candidate claim с планом проверки.

| Уровень обоснования | Что считается valid source | Пример |
|---------------------|----------------------------|--------|
| SoTA / исследование | Рецензируемая статья, NASEM, CNCPS | NASEM 2021: потребность в MP для коровы 600 кг, 30 кг молока |
| Отраслевая норма | Принятый порог или стандарт | Фронт кормления ≥ 60 см / голова |
| Исторические данные фермы | Записи этой же фермы при сопоставимых условиях | В 2025 при CP 16,2% удой был 31 кг vs 30,5 кг при 17,5% |
| Пилотный эксперимент | Явная гипотеза с методом проверки | Снизим CP до 15,5%, измерим удой и BHB через 14 дней |
| Текущие замеры | Данные, показывающие отклонение | BHB > 1,2 ммоль/л у 30% группы — сигнал к корректировке |

**Если source отсутствует — рекомендация оформляется как candidate claim:**

| Элемент | Что фиксируем |
|---------|---------------|
| Гипотеза | «Снижение CP до 15,5% не снизит удой» |
| Ожидаемый результат | «Удой группы сохранится на 30 ± 1 кг» |
| Срок проверки | «14 дней» |
| Измерение | «Ежедневный удой AMS + BHB 10 коров через 7 и 14 дней» |
| Критерий отката | «Удой < 28 кг или BHB > 1,2 ммоль/л → вернуть CP» |

#### Шаг 4.2. Классифицируй language-state каждого вывода

| Вывод | Language-state | Что можно сделать |
|-------|----------------|-------------------|
| «Фронт кормления 55 см на голову» | Data / Measurement | Использовать как evidence |
| «Группе не хватает фронта кормления» | Description (candidate) | Сравнить с нормой |
| «Недостаток фронта снижает DMI низкоранговых коров» | Dynamics (candidate) | Проверить через наблюдение / литературу |
| «Увеличить фронт до 70 см» | WorkPlan | Согласовать с Herd Manager |

#### Шаг 4.2. Построй evidence graph

```
Observation: остатки корма сортированы
        ↓
Data: проба ТМР — высокое содержание мелких частиц
        ↓
Measurement: ситовой анализ показывает недостаток физически эффективного НДФ
        ↓
Hypothesis: коровы сортируют смесь из-за низкой структуры
        ↓
WorkPlan: скорректировать измельчение / добавить грубые волокна
        ↓
Work record: фиксация изменения в журнале кормления
        ↓
Outcome: повторный визит + проба через 7–14 дней
```

### Фаза 5. WorkPlan и follow-up

#### Шаг 5.1. Зафиксируй рекомендации

| Проблема | Рекомендация | Ответственный | Срок | Проверка |
|----------|--------------|---------------|------|----------|
| Фронт кормления < нормы | Увеличить фронт / снизить загрузку группы | Herd Manager | 7 дней | Повторный замер |
| Расхождение расчётного и фактического ТМР | Пересмотреть рецепт / загрузку миксера | Feed Manager | 3 дня | Новая проба |
| Снижение удоя при THI > 72 | Меры охлаждения, кормление в прохладное время | Herd Manager | Немедленно | THI + удой |

#### Шаг 5.2. Назначь повторную проверку

| Показатель | Через сколько проверять | Кто |
|------------|-------------------------|-----|
| Состав ТМР | 7–14 дней после корректировки | Feed Manager + лаборатория |
| Удой группы | 14–21 день | Data Analyst |
| Фронт / лежачие места | Следующий визит | Consultant |
