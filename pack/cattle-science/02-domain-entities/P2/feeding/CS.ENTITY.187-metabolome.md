# Метаболом (Metabolome)

## Определение

**Метаболом** — совокупность всех низкомолекулярных метаболитов (<1500 Да), присутствующих в биологической системе (клетке, ткани, органе, организме или популяции) в определённый момент времени. Представляет собой конечный результат экспрессии генов и активности белков, отражающий функциональное состояние биологической системы и её ответ на внутренние и внешние факторы.

## Контекст SoTA

### Источник
- **Ghaffari et al., 2024** (CS.SOTA.065) — метаболомика у молочных коров

### Значение
Метаболом является:
- Конечным фенотипическим отображением генома
- Интегративным показателем физиологического статуса
- Источником биомаркеров метаболических расстройств
- Основой для персонализированного подхода к животноводству

## Свойства

### Характеристики метаболома
| Парактеристика | Описание |
|----------------|----------|
| Размер | ~5000-25000 различных метаболитов (оценки варьируют) |
| Динамичность | Время отклика: секунды — минуты |
| Концентрационный диапазон | >10¹² (от фМ до мМ) |
| Химическое разнообразие | Липиды, аминокислоты, углеводы, нуклеотиды и др. |
| Зависимость от факторов | Генотип, фенотип, окружающая среда, питание |

### Уровни изучения метаболома
| Уровень | Объект | Методы |
|---------|--------|--------|
| Клеточный | Метаболом клетки | GC-MS, LC-MS культур |
| Тканевый | Метаболом ткани | MALDI-MS imaging, экстракция |
| Организменный | Кровь, моча, молоко | NMR, LC-MS, GC-MS |
| Популяционный | Стадо, порода | Статистика, машинное обучение |

### Классы метаболитов
```
Метаболом
    ├─→ Липиды (30-40% массы)
    │       ├─→ Жирные кислоты
    │       ├─→ Триглицериды
    │       ├─→ Фосфолипиды
    │       └─→ Сфинголипиды
    ├─→ Аминокислоты и производные (20-25%)
    ├─→ Углеводы (10-15%)
    ├─→ Органические кислоты (10-15%)
    ├─→ Нуклеотиды (5-10%)
    └─→ Другие (кофакторы, витамины, ионы)
```

## Физиологическая роль

### Как интегративный показатель
```
Геном → Транскриптом → Протеом → Метаболом → Фенотип
                ↓
        Все уровни регуляции отражаются в метаболоме
                ↓
        Функциональный статус организма
```

### В метаболических исследованиях
- **Переходный период**: Динамика метаболома вокруг отёла
- **НЭБ**: Ранние метаболомные изменения
- **Кетоз**: Специфические метаболомные сигнатуры
- **Молочная продуктивность**: Связь метаболома с удоями

### В молочных коровах
- **Индивидуальная вариабельность**: Метаболомный «отпечаток»
- **Раннее предсказание**: Заболеваний и продуктивности
- **Нутриционный статус**: Отражение питания
- **Состояние здоровья**: Комплексная оценка

## Клиническое и исследовательское значение

### Биомаркерные панели
| Состояние | Ключевые метаболиты | Метод |
|-----------|---------------------|-------|
| Кетоз | β-гидроксибутират, ацетоацетат, НЖК | NMR, LC-MS |
| Жировая инфильтрация печени | Триглицериды, фосфатидилхолины | LC-MS |
| Мастит | Лактоферрин, лизоцим | MS |
| Субклиническая ацидоз | Лактат, VFA | GC-MS |

### Метаболомные сигнатуры
```
Пример: Раннее предсказание кетоза

За 2 недели до клинических признаков:
↓ Глюкоза
↑ НЖК
↑ Ацилкарнитины
↓ Аминокислоты с разветвлённой цепью
↓ Холин, бетаин
```

### В прецизионном животноводстве
- **Индивидуальный мониторинг**: Профилирование каждой коровы
- **Раннее вмешательство**: Предиктивная диагностика
- **Оптимизация питания**: Персонализированные рационы
- **Селекция**: Метаболомные маркеры продуктивности

## Методы исследования

### Ядерный магнитный резонанс (NMR)
| Преимущества | Недостатки |
|--------------|------------|
| Неразрушающий | Низкая чувствительность |
| Количественный | Требует высокой концентрации |
| Воспроизводимый | Ограниченное число метаболитов |
| Не требует предварительной подготовки | |

### Масс-спектрометрия (LC-MS, GC-MS)
| Преимущества | Недостатки |
|--------------|------------|
| Высокая чувствительность | Разрушающий |
| Широкое покрытие | Требует экстракции |
| Идентификация структуры | Матричные эффекты |
| Количественный | Сложность интерпретации |

### Методы анализа данных
- **Многомерная статистика**: PCA, PLS-DA, OPLS-DA
- **Машинное обучение**: Random Forest, SVM, нейронные сети
- **Сетевой анализ**: Метаболические пути
- **Интеграция омикс**: Мультиомиксный подход

## Интерпретации

### Для исследователей
Метаболом — окно в функциональное состояние:
- **Интегративность**: Отражает все уровни регуляции
- **Динамичность**: Быстрый отклик на изменения
- **Сложность**: Требует современных аналитических методов
- **Потенциал**: Биомаркеры и механистические инсайты

### Для практиков
Метаболомика открывает возможности:
- Раннего предсказания заболеваний
- Индивидуального мониторинга
- Оптимизации питания
- Повышения эффективности производства
