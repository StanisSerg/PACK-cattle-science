# Лизосомальная кислая липаза (LAL)

## Определение

**Лизосомальная кислая липаза (LAL, также известная как LIPA)** — лизосомальный фермент, оптимально функционирующий при кислом pH (~4.5-5.0), катализирующий гидролиз триглицеридов и холестериловых эфиров внутри лизосом. Играет критическую роль в внутриклеточном липидном обороте, аутофагии липидов (липофагии) и поддержании липидного гомеостаза.

## Контекст SoTA

### Источник
- **McFadden et al., 2017** (CS.SOTA.039) — обзор липидного метаболизма у молочных коров

### Значение
LAL обеспечивает:
- Внутриклеточный липидный оборот
- Деградацию липидных дроплетов через аутофагию
- Холестериновый гомеостаз
- Альтернативный путь липолиза независимый от цитоплазматических липаз

## Свойства

### Ферментативные характеристики
| Свойство | Значение |
|----------|----------|
| EC Number | 3.1.1.13 |

| Молекулярная масса | ~45 кДа (гликопротеин) |

| Субстраты | Триглицериды, холестериловые эфиры |

### Реакции
```
Триглицериды + H₂O → Диглицериды + НЖК
Холестериловые эфиры + H₂O → Холестерин + Жирная кислота
```

### Биосинтез и транспорт
```
Рибосомы → Транслокация в ЭПР → Гликозилирование
                ↓
        Транспорт в Гольджи
                ↓
        Сортировка маннозо-6-фосфатным путём
                ↓
        Доставка в лизосомы
```

## Физиологическая роль

### Внутриклеточный липидный оборот
- **Базальная утилизация**: Постоянная переработка липидов
- **Липофагия**: Аутофагическое захват и деградация липидных дроплетов
- **Макрофаги**: Критически важна для пенистых клеток

### В печени
- **Липидный гомеостаз**: Регуляция внутриклеточных липидов
- **Холестериновый обмен**: Освобождение холестерина
- **При жировой инфильтрации**: Увеличенная нагрузка на лизосомальный путь

### В молочных коровах
- **Жировая инфильтрация печени**: Роль в переработке накопленных липидов
- **НЭБ**: Возможная компенсаторная роль при перегрузке цитоплазматического липолиза
- **Метаболическая адаптация**: Долгосрочная регуляция липидных запасов

## Клиническое и исследовательское значение

### Наследственные заболевания (человек)
| Заболевание | Мутация | Проявления |
|-------------|---------|------------|
| Болезнь Вольмана | LAL-дефицит | Накопление липидов, гепатоспленомегалия |
| Холестериловый эфир-накопительная болезнь | LAL-дефицит | Аккумуляция холестерила |

### В ветеринарии
- **Исследовательский интерес**: Роль в жировой дистрофии печени
- **Биомаркер**: Активность LAL как индикатор лизосомальной функции
- **Терапевтический потенциал**: Рекомбинантная LAL для липидозов

## Методы измерения

### Активность фермента
- **Принцип**: Гидролиз 4-метилумбеллиферилолеата при pH 4.0-4.5
- **Образец**: Лизосомные фракции или цельные гомогенаты
- **Детекция**: Флуориметрия

### Иммуноанализ
- **Иммуноблоттинг**: Белок LAL
- **Иммуногистохимия**: Локализация в лизосомах

### Клинические тесты
- ** dried blood spot **: Скрининг новорождённых (человек)
- **Генетическое тестирование**: Мутации LIPA

## Интерпретации

### Для исследователей
LAL представляет собой:
- **Лизосомальный путь липолиза**: Альтернатива цитоплазматическим липазам
- **Регулятор гомеостаза**: Контроль внутриклеточного липидного баланса
- **Мишень для терапии**: Заместительная терапия при дефиците

### Для практиков
Понимание LAL помогает:
- Оценить механизмы переработки липидов в печени
- Понять роль аутофагии в метаболических расстройствах
- Рассмотреть альтернативные пути при блокаде цитоплазматического липолиза
