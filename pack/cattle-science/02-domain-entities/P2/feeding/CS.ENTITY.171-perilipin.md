# Перилипин (Perilipin)

## Определение

**Перилипин (PLIN1, также известный как Perilipin A)** — фосфопротеин, являющийся основным белком оболочки липидных дроплетов в адипоцитах. Выступает в роли барьера, защищающего триглицериды от неспецифического гидролиза, и как регуляторная платформа, контролирующая доступ липаз к липидным запасам через гормон-зависимую фосфорилирование.

## Контекст SoTA

### Источник
- **McFadden et al., 2017** (CS.SOTA.039) — обзор липидного метаболизма у молочных коров

### Значение
Перилипин является:
- Ключевым регулятором липолиза
- Защитником липидных запасов от несанкционированного расщепления
- Платформой для сборки липолитического комплекса
- Маркером зрелых липидных дроплетов

## Свойства

### Структурные характеристики
| Свойство | Значение |
|----------|----------|
| Ген | PLIN1 |
| Молекулярная масса | ~60 кДа (Perilipin A), ~46 кДа (Perilipin B) |
| Структура | Не структурированный белок с амфипатичными участками |
| Локализация | Поверхность липидных дроплетов |
| Изоформы | Perilipin A (основная в адипоцитах), Perilipin B |

### Доменная организация
- **N-терминальный домен**: Якорь на липидном дроплете
- **Центральная область**: Сайты фосфорилирования
- **C-терминальный домен**: Взаимодействие с регуляторными белками

### Сайты фосфорилирования (PKA)
- **Ser81, Ser222, Ser276, Ser433, Ser497**: Основные регуляторные фосфосайты
- **Функция**: Конформационные изменения, высвобождение CGI-58

## Физиологическая роль

### В базальном состоянии (без стимуляции)
```
Перилипин (нефосфорилированный)
        ↓
Блокирует доступ ATGL к триглицеридам
        ↓
CGI-58 связан с перилипином (недоступен для ATGL)
        ↓
Минимальный липолиз
```

### При стимуляции (катехоламины)
```
Эпинефрин → β-адренорецептор → cAMP ↑ → PKA активация
                    ↓
            Перилипин фосфорилируется
                    ↓
    ┌───────────────┴───────────────┐
    ↓                               ↓
CGI-58 высвобождается      HSL транслоцируется
    ↓                               ↓
Активация ATGL             Доступ к диглицеридам
    └───────────────┬───────────────┘
                    ↓
            Максимальный липолиз
```

### В молочных коровах
- **Адаптация к НЭБ**: Фосфорилирование перилипина усиливается
- **Индивидуальная вариабельность**: Уровень экспрессии влияет на скорость липолиза
- **Хроническая стимуляция**: Даунрегуляция перилипина при длительном НЭБ

## Клиническое и исследовательское значение

### Регуляция липолиза
| Уровень перилипина | Последствия |
|-------------------|-------------|
| Высокий | Защита запасов, медленный липолиз |
| Низкий | Легкая мобилизация, склонность к кетозу |
| Нефосфорилированный | Блокада липаз |
| Фосфорилированный | Доступ липаз к субстрату |

### Исследовательские приложения
- **Биопсии жировой ткани**: Уровень PLIN1 мРНК/белка
- **Полиморфизмы**: PLIN1 гена и склонность к ожирению/кетозу
- **Нутригеномика**: Влияние питания на экспрессию

## Методы измерения

### Генная экспрессия
- **qPCR**: PLIN1 мРНК
- **Нормализация**: К референсным генам (GAPDH, β-actin)

- **Иммуноблоттинг**: Общий перилипин и фосфоформы
- **Иммунофлуоресценция**: Локализация на липидных дроплетах
- **Кофокальная микроскопия**: Динамика при стимуляции

### Функциональные ассайи
- **Липолиз in vitro**: С коррекцией на уровень перилипина
- **Co-IP**: Взаимодействие с ATGL, HSL, CGI-58

## Интерпретации

### Для исследователей
Перилипин — мастер-регулятор липолиза:
- **Барьерная функция**: Физическая защита липидов
- **Платформенная функция**: Организация липолитического комплекса
- **Регуляторная функция**: Гормональная модуляция через фосфорилирование

### Для практиков
Понимание перилипина объясняет:
- Почему липолиз не происходит постоянно
- Механизм гормонального контроля мобилизации жира
- Потенциальные мишени для модуляции энергетического баланса
- Индивидуальные различия в метаболической адаптации
