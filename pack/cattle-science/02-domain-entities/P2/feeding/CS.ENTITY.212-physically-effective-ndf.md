# CS.ENTITY.212: Физически эффективная NDF (peNDF)

> Показатель способности нейтрально-детергентной клетчатки (NDF) стимулировать жевательную активность и выработку слюны, рассчитываемый как произведение содержания NDF в рационе на физическую эффективность (pef), определяемую размером частиц; критически важный параметр для предотвращения руминального ацидоза и поддержания здоровья рубца.

---

## Определение

**Физически эффективная NDF (peNDF)** — показатель, количественно описывающий способность клетчатки в рационе стимулировать жевательную активность, выработку слюны и формирование рубцовой матрицы, рассчитываемый по формуле: peNDF = NDF x pef.

**Диагностические критерии:**

| Параметр | Формула/значение | Примечание |
|----------|------------------|------------|
| **peNDF** | % от СВ | Произведение NDF x pef |
| **pef (физическая эффективность)** | Безразмерная | Определяется размером частиц |
| **NDF** | % от СВ | Содержание клетчатки в рационе |
| **Размер частиц (референс)** | >4 мм или >1.18 мм | Зависит от метода |

**Определение случая (case definition):**
- **pef (Mertens):** Сумма частиц на сите 1.18 мм и крупнее в PSPS
- **pef (Kononoff):** Сумма частиц на верхнем и среднем сите PSPS (>4 мм + 4-8 мм)
- **peNDF (рационная):** Расчетная величина для всего рациона

## Этиология/Патогенез

### Механизм физической эффективности

```
Грубые частицы (>4 мм)
         ↓
    Стимуляция рецепторов
    ротовой полости
         ↓
    ┌────┴────┐
    ↓         ↓
Пережевывание  Слюноотделение
(жвачка)       (2-6 л/сутки)
    ↓         ↓
    └────┬────┘
         ↓
    ┌────┴────┐
    ↓         ↓
Физическое    Буферный
измельчение   эффект
(>1 мм)       (слюна pH 8.2)
    ↓         ↓
    └────┬────┘
         ↓
    Поддержание pH рубца
    Предотвращение ацидоза
```

### Факторы, влияющие на pef

| Фактор | Влияние на pef | Механизм |
|--------|----------------|----------|
| **Размер частиц** | Крупнее → выше pef | Больше стимуляции жвачки |
| **Длина стебля** | Длиннее → выше pef | Сложнее пережевывать |
| **Тип корма** | Сено > силос > зерно | Структурные различия |
| **Обработка** | Измельчение ↓ pef | Уменьшение размера |
| **Влажность** | Высокая может ↓ pef | Слеживание, изменение текстуры |

## Классификация

### По методу определения pef

| Метод | Расчет pef | Особенности |
|-------|------------|-------------|
| **Mertens** | (% на сите 1.18 мм + % на пане x 0) / 100 | Учет всех частиц >1.18 мм |
| **Kononoff** | (% >19 мм x 1.0 + % 8-19 мм x 0.8 + % 1.18-8 мм x 0.3) / 100 | Взвешенный по размеру |
| **Lammers** | Бинарный (>4 мм = 1, <4 мм = 0) | Упрощенный |

### По уровню peNDF

| Категория | peNDF (% СВ) | Интерпретация | Применение |
|-----------|--------------|---------------|------------|
| **Критически низкая** | <15% | Высокий риск ацидоза | Недопустимо |
| **Низкая** | 15-19% | Повышенный риск | Только с осторожностью |
| **Оптимальная** | 20-24% | Баланс | Стандарт |
| **Высокая** | 25-30% | Консервативная | Риск снижения DMI |
| **Очень высокая** | >30% | Ограничение DMI | Лечебные диеты |

## Диагностика

### Методы оценки

| Метод | Принцип | Точность | Стоимость |
|-------|---------|----------|-----------|
| **PSPS** | Физическое сито | Высокая | Низкая |
| **Wet sieve** | Мокрое просеивание | Очень высокая | Средняя |
| **Laser diffraction** | Лазерное рассеяние | Высокая | Высокая |
| **Визуальная оценка** | Экспертное мнение | Низкая | Нулевая |

### Расчет peNDF

**Алгоритм Kononoff:**
```
pef = (DM retained on 19-mm sieve x 1.0 + 
       DM retained on 8-mm sieve x 0.8 + 
       DM retained on 1.18-mm sieve x 0.3) / Total DM

peNDF = NDF (%) x pef
```

**Пример расчета:**
| Сито | % СВ | Коэффициент | Вклад в pef |
|------|------|-------------|-------------|
| >19 мм | 5% | 1.0 | 0.05 |
| 8-19 мм | 40% | 0.8 | 0.32 |
| 1.18-8 мм | 35% | 0.3 | 0.105 |
| <1.18 мм | 20% | 0 | 0 |
| **Итого pef** | | | **0.475** |
| **NDF рациона** | 35% | | |
| **peNDF** | **16.6%** | | |

## Управление/Применение

### Корректировка peNDF

| Проблема | Решение | Ожидаемый эффект |
|----------|---------|------------------|
| **peNDF < 18%** | Добавить длинноствольное сено | +2-4% peNDF |
| **peNDF < 15%** | Срочная корректировка, увеличить грубый корм | +4-6% peNDF |
| **peNDF > 28%** | Измельчить сено, увеличить концентраты | -3-5% peNDF |
| **Высокая сортировка** | Добавить мелассу, улучшить смешивание | Стабилизация |

### Алгоритм оценки

```
Анализ TMR через PSPS
        ↓
   Расчет pef
        ↓
   Расчет peNDF
        ↓
   ┌────┴────┐
   ↓         ↓
<18%      18-28%      >28%
   ↓         ↓           ↓
Риск      Оптимум      Ограничение
ацидоза               DMI
   ↓                    ↓
Корректировка      Корректировка
```
