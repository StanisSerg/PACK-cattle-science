# Лактатдегидрогеназа (LDH)

## Определение

**Лактатдегидрогеназа (ЛДГ, LDH, EC 1.1.1.27)** — фермент, катализирующий обратимое окислительное восстановление пирувата в лактат с окислением NADH в NAD⁺. Играет критическую роль в анаэробном гликолизе, обеспечивая регенерацию NAD⁺, необходимого для продолжения гликолиза в условиях кислородного дефицита. Является важным диагностическим маркером клеточного повреждения.

## Контекст SoTA

### Источник
- **Aiello et al., 1984** (CS.SOTA.026) — исследование метаболизма у молочных коров

### Значение
ЛДГ является:
- Ключевым ферментом анаэробного гликолиза
- Регенератором NAD⁺ при гипоксии
- Важным диагностическим маркером
- Индикатором тканевого повреждения

## Физиологическая роль

### В анаэробном гликолизе
```
Гликолиза → Пируват + NADH + H⁺
                    ↓
            [Лактатдегидрогеназа]
                    ↓
            Лактат + NAD⁺
                    ↓
            NAD⁺ возвращается в гликолиз
                    ↓
            Продолжение АТФ-продукции без кислорода
```

**Критическое значение:**
- **Регенерация NAD⁺**: Необходима для G3PDH
- **Анаэробное выживание**: Позволяет продолжать гликолиз
- **Коридорный эффект**: Предотвращает накопление NADH

### В различных тканях

#### Мышцы (LDH5)
- **Интенсивная работа**: Продукция лактата
- **Кислородный долг**: Накопление лактата
- **Восстановление**: Реокисление лактата в печени

#### Печень (LDH5)
- **Продукция лактата**: Из глюкозы
- **Глюконеогенез**: Реутилизация лактата (Cori cycle)

#### Сердце (LDH1)
- **Окисление лактата**: Предпочтительный субстрат
- **Аэробный метаболизм**: Эффективное использование лактата

### В молочных коровах
- **Физическая нагрузка**: Продукция лактата
- **Родовая деятельность**: Анаэробный метаболизм матки
- **Метаболический профиль**: Плазменная активность ЛДГ
- **Диагностика**: Повышение при повреждении тканей

## Клиническое и исследовательское значение

### Диагностический маркер
| Состояние | Изменение ЛДГ | Механизм |
|-----------|---------------|----------|
| Миокардит | ↑ LDH1 | Повреждение кардиомиоцитов |
| Гепатит | ↑ LDH5 | Повреждение гепатоцитов |
| Гемолиз | ↑ LDH1 | Разрушение эритроцитов |
| Мышечная травма | ↑ LDH5 | Повреждение миоцитов |

### Метаболические расстройства
- **Лактатный ацидоз**: Накопление лактата
- **Малигнантная гипертермия**: Дефект кальциевой гомеостаза → гиперметаболизм

### В ветеринарии
- **Общая активность**: Неспецифический маркер повреждения
- **Изоформный анализ**: Дифференциальная диагностика
- **Мониторинг**: Травмы, инфекции, метаболические расстройства

## Методы измерения

### Активность фермента
- **Принцип**: Пируват + NADH → Лактат + NAD⁺
- **Детекция**: Снижение абсорбции NADH (340 нм)
- **Образец**: Сыворотка, плазма, тканевые гомогенаты

### Изоформный анализ
- **Электрофорез**: Разделение LDH1-5
- **Хроматография**: Ионообменная
- **Иммунохимия**: Специфичные антитела

### Генетический анализ
- **Секвенирование**: Гены LDHA (M), LDHB (H)
- **Полиморфизмы**: Варианты активности
