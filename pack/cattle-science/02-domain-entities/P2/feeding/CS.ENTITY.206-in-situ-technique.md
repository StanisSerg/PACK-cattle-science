# CS.ENTITY.206: Метод in situ

> Стандартизированный метод оценки руминальной деградации питательных веществ с использованием нейлоновых мешков, помещаемых непосредственно в рубец, позволяющий определять скорость и степень переваривания белка, клетчатки и крахмала в условиях in situ (в естественной среде рубца).

---

## Определение

**Метод in situ** (лат. «в положении», «на месте») — метод исследования руминальной деградации кормов, основанный на инкубации образцов корма в нейлоновых мешках, помещённых непосредственно в рубец животного через фистулу или зонд.

**Диагностические критерии:**

| Параметр | Описание | Типичные значения |
|----------|----------|-------------------|
| **Пористость мешка** | Размер отверстий нейлона | 37-53 мкм (стандарт) |
| **Соотношение образец:объём** | Масса корма к объёму мешка | 10-20 мг/см² |
| **Размер частиц** | Измельчение образца | 2-5 мм (стандарт) |
| **Время инкубации** | Периоды измерения | 0, 2, 4, 8, 16, 24, 48 ч |
| **Количество мешков** | Число повторностей | 2-3 животных × 2 мешка |

**Определение случая (case definition):**
- **Стандартный протокол:** Нейлоновые мешки 10×15 см, пористость 50 мкм, измельчение 2 мм
- **Альтернативный протокол:** Мешки 5×10 см для быстрой деградации
- **Расширенный протокол:** До 72-96 часов для медленно деградируемых компонентов

## Принцип метода

### Механизм инкубации

```
Образец корма (измельчённый)
         ↓
    Нейлоновый мешок
         ↓
    Помещение в рубец
         ↓
    ┌────┴────┐
    ↓         ↓
Микробный   Ферментный
доступ      контакт
    ↓         ↓
    └────┬────┘
         ↓
    Деградация
         ↓
    Извлечение мешка
         ↓
    Промывание + сушка
         ↓
    Расчёт потерь массы
```

### Кинетика деградации

**Уравнение первого порядка:**
- Деградация = a + b × (1 - e^(-ct))
- Где: a — растворимая фракция (%), b — медленно деградируемая фракция (%), c — скорость деградации (%/ч)

**Эффективная деградация:**
- ED = a + b × (c / (c + kp))
- Где kp — скорость прохождения через рубец

| **Долгосрочный** | 72-96 часов | Медленно деградируемые корма |

## Диагностика/Применение

### Стандартный протокол

| Этап | Действие | Параметры |
|------|----------|-----------|
| **Подготовка** | Измельчение, взвешивание | 2 мм, 5 г СВ |
| **Упаковка** | Помещение в мешки | Двойное шитьё |
| **Инкубация** | Помещение в рубец | Перед кормлением |
| **Извлечение** | По графику | 0, 2, 4, 8, 16, 24, 48 ч |
| **Обработка** | Промывание, сушка, взвешивание | Холодная вода, 60°C |

### Расчёт показателей

| Показатель | Формула | Интерпретация |
|------------|---------|---------------|
| **Деградация СВ** | (Начальная масса - Конечная масса) / Начальная × 100% | Общая деградация |
| **Деградация белка** | Аналогично с определением азота | РДС = a + b |
| **Скорость (c)** | Из кривой деградации | Чем выше, тем быстрее |
| **Эффективная деградация** | ED = a + b × c/(c+kp) | С учётом пассажа |
