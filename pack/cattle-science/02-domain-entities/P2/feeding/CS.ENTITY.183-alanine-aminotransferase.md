# Аланинаминотрансфераза (ALT)

## Определение

**Аланинаминотрансфераза (АЛТ, ALT, также известная как глутамин-пируваттрансаминаза, GPT, EC 2.6.1.2)** — фермент, катализирующий обратимую передачу аминогруппы от аланина на α-кетоглутарат с образованием пирувата и глютамата. Является специфичным маркером цитоплазматического повреждения гепатоцитов и одним из наиболее широко используемых показателей функции печени в клинической ветеринарии и медицине.

## Контекст SoTA

### Источник
- **Aiello et al., 1984** (CS.SOTA.026) — исследование метаболизма у молочных коров

### Значение
АЛТ является:
- Ключевым ферментом метаболизма аминокислот
- Специфичным маркером повреждения печени
- Важным показателем в клинической диагностике
- Индикатором функционального состояния гепатоцитов

## Физиологическая роль

### В метаболизме аминокислот
```
Аланин + α-кетоглутарат ⇌ Пируват + Глютамат
        ↓
Глюкоза-аланиновый цикл:
- Мышцы: Аланин → печень
- Печень: Глюконеогенез из аланина
```

**Функции:**
- **Транспорт азота**: Аланин как переносчик
- **Глюконеогенез**: Субстрат для синтеза глюкозы
- **Энергетический метаболизм**: Связь белкового и углеводного обмена

### В печени
- **Детоксикация аммония**: Конверсия в глютамат
- **Глюконеогенез**: Пируват → глюкоза
- **Метаболическая гибкость**: Адаптация к нутритивному статусу

### Глюкоза-аланиновый цикл
```
Мышцы:
Глюкоза → Гликолиз → Пируват + Аминогруппа → Аланин
                                    ↓
                            Транспорт кровью
                                    ↓
Печень:
Аланин → Трансаминирование → Пируват → Глюконеогенез → Глюкоза
                                    ↓
                            Транспорт к мышцам
```

### В молочных коровах
- **Функция печени**: Оценка метаболической нагрузки
- **Переходный период**: Жировая инфильтрация → повреждение
- **Кетоз**: Вторичное повреждение печени
- **Мониторинг**: Ранняя диагностика печёночной дисфункции

## Клиническое и исследовательское значение

### Диагностика печёночных заболеваний
| Состояние | Изменение АЛТ | Механизм |
|-----------|---------------|----------|
| Гепатит | ↑↑↑ (10-100x) | Некроз гепатоцитов |
| Жировая инфильтрация | ↑ (2-5x) | Умеренное повреждение |
| Токсический гепатоз | ↑↑↑ | Химическое повреждение |
| Ишемия печени | ↑↑ | Гипоксическое повреждение |

### Интерпретация в клинике
```
Норма: < 30-50 U/L (зависит от лаборатории)

Умеренное повышение (2-5x): 
- Жировая инфильтрация
- Хроническое повреждение

Значительное повышение (>10x):
- Острый гепатит
- Токсическое повреждение
- Ишемия
```

### В ветеринарии
- **Скрининг**: Рутинный биохимический профиль
- **Мониторинг**: Динамика при лечении
- **Прогноз**: Снижение = улучшение

## Методы измерения

### Спектрофотометрический метод
- **Принцип**: Пируват + NADH + H⁺ → Лактат + NAD⁺ (LDH-связанная)
- **Детекция**: Падение абсорбции NADH (340 нм)
- **Образец**: Сыворотка, плазма

### Автоматизированные анализаторы
- **Стандартизация**: IFCC рекомендации
- **Калибровка**: Сертифицированные стандарты
- **Контроль качества**: Ежедневный контроль

### Факторы, влияющие на результаты
| Фактор | Эффект |
|--------|--------|
| Гемолиз | Ложное повышение (эритроциты содержат АЛТ) |
| Витамин B6 дефицит | Снижение активности (требуется кофактор) |
| Лекарства | Индукция или ингибирование |

## Интерпретации

### Для исследователей
АЛТ — специфичный маркер печёночной цитоплазмы:
- **Высокая специфичность**: Преимущественно печень
- **Цитоплазматическая локализация**: Ранний маркер повреждения
- **Долгий период полураспада**: Подходит для мониторинга

### Для практиков
Понимание АЛТ критично для:
- Ранней диагностики печёночных заболеваний
- Мониторинга функции печени
- Дифференциальной диагностики
- Оценки эффективности терапии
