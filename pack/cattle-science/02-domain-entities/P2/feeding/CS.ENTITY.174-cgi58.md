# CGI-58 (Comparative Gene Identification-58)

## Определение

**CGI-58 (ABHD5, α/β-гидролазный домен-содержащий белок 5)** — белок, являющийся кофактором (коактиватором) ATGL, необходимым для полной активности фермента. Является членом семейства α/β-гидролаз, но лишён липазной активности. Вместо этого функционирует как регуляторный белок, связывающийся с ATGL и увеличивающий его активность примерно в 20 раз.

## Контекст SoTA

### Источник
- **McFadden et al., 2017** (CS.SOTA.039) — обзор липидного метаболизма у молочных коров

### Значение
CGI-58 является:
- Необходимым коактиватором ATGL
- Ключевым регуляторным звеном в липолизе
- Мишенью гормональной регуляции через перилипин
- Геном, мутации которого вызывают болезнь Чанарина-Дорфмана

## Свойства

### Структурные характеристики
| Свойство | Значение |
|----------|----------|
| Ген | ABHD5 |
| Молекулярная масса | ~39 кДа |
| Семейство | α/β-гидролазы (но каталитически неактивен) |
| Локализация | Цитоплазма, поверхность липидных дроплетов |
| Структура | α/β-гидролазный фолд |

### Механизм коактивации
```
CGI-58 связывается с ATGL
        ↓
Индукция конформационного изменения ATGL
        ↓
Увеличение каталитической эффективности ~20-fold
        ↓
Улучшенный доступ к триглицеридам
```

### Регуляция локализации
| Состояние | Локализация CGI-58 | Функция |
|-----------|-------------------|---------|
| Базальное | Связан с перилипином | Инактивен |
| Стимулированное | Свободен, связан с ATGL | Коактивация |

## Физиологическая роль

### В липолитическом комплексе
```
Липидный дроплет
        │
    Перилипин (барьер)
        │
    ┌───┴───┐
    ↓       ↓
CGI-58   ATGL (базальная активность)
    │       ↑
    └───→ (при стимуляции)
            ↓
    ATGL-CGI-58 комплекс
            ↓
    Максимальная липолитическая активность
```

### Гормональная регуляция
1. **Базальное состояние**: CGI-58 связан с перилипином → недоступен для ATGL
2. **PKA активация**: Перилипин фосфорилируется
3. **Высвобождение**: CGI-58 освобождается от перилипина
4. **Коактивация**: CGI-58 связывается с ATGL

### В молочных коровах
- **НЭБ**: Критическая роль для максимального липолиза
- **Генетическая вариабельность**: Полиморфизмы ABHD5 и липолитическая способность
- **Метаболическая адаптация**: Регуляция экспрессии при энергетическом стрессе

## Интерпретации

### Для исследователей
CGI-58 — уникальный регуляторный белок:
- **Коактиватор**: Усиливает ATGL без каталитической активности
- **Переключатель**: Гормональное регулирование липолиза
- **Терапевтическая мишень**: Потенциал для модуляции

### Для практиков
Понимание CGI-58 объясняет:
- Почему ATGL требует активации для полной функции
- Многоуровневую регуляцию липолиза
- Генетические предпосылки к метаболическим расстройствам
