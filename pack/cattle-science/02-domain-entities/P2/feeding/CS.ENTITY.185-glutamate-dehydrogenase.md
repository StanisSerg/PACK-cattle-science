# Глютаматдегидрогеназа (GDH)

## Определение

**Глютаматдегидрогеназа (ГДГ, GDH, EC 1.4.1.3)** — митохондриальный фермент, катализирующий обратимое окислительное дезаминирование глютамата до α-кетоглутарата с высвобождением аммония (NH₄⁺). Играет центральную роль в метаболизме аминокислот, детоксикации аммония и энергетическом метаболизме. Локализована преимущественно в митохондриях печени, почек, мозга и других тканей.

## Контекст SoTA

### Источник
- **Aiello et al., 1984** (CS.SOTA.026) — исследование метаболизма у молочных коров

### Значение
ГДГ является:
- Ключевым ферментом метаболизма аминокислот
- Центральным звеном детоксикации аммония
- Важным диагностическим маркером (особенно в крупном рогатом скоте)
- Регулятором энергетического баланса

## Физиологическая роль

### В метаболизме аминокислот
```
Аминокислоты → Трансаминирование → Глютамат
                                        ↓
                                [ГДГ]
                                        ↓
                            α-кетоглутарат + NH₄⁺
                                        ↓
                            ┌───────────┴───────────┐
                            ↓                       ↓
                        Цикл Кребса           Уреевый цикл
                        (энергия)             (детоксикация)
```

### В печени
- **Уреевый цикл**: Основной источник аммония для синтеза мочевины
- **Детоксикация**: Превращение токсичного NH₄⁺ в безопасную мочевину
- **Энергетический метаболизм**: Связь белкового и углеводного обмена
- **Глюконеогенез**: α-кетоглутарат как субстрат

### В мозге
- **Глутаматергическая передача**: Регуляция уровня глютамата
- **Детоксикация аммония**: Критически важна при печёночной энцефалопатии
- **Энергетика**: Продукция АТФ в анаэробных условиях

### В молочных коровах
- **Метаболизм белка**: Руменовые микроорганизмы → аммония → мочевина
- **Переходный период**: Регуляция азотистого баланса
- **Печёночная функция**: Чувствительный маркер повреждения
- **Молочный белок**: Метаболизм аминокислот для синтеза

## Клиническое и исследовательское значение

### Диагностика в ветеринарии
| Состояние | Изменение ГДГ | Преимущество над АЛТ/АСТ |
|-----------|---------------|-------------------------|
| Острое повреждение печени | ↑↑↑ | Более раннее повышение |
| Жировая инфильтрация | ↑ | Чувствительность к функции |
| Токсический гепатоз | ↑↑↑ | Митохондриальная специфичность |
| Нервные расстройства | ↑ | Печёночная энцефалопатия |

### Преимущества как маркера
```
ГДГ vs. АЛТ/АСТ:

✓ Митохондриальная локализация — ранний маркер
✓ Длительный период полураспада — стабильный показатель
✓ Высокая специфичность к печени
✓ Меньше влияние гемолиза
✗ Меньше доступность в коммерческих лабораториях
```

### Метаболические расстройства
| Состояние | Механизм |
|-----------|----------|
| Гипераммонемия | Дефицит ГДГ или уреевого цикла |
| Печёночная энцефалопатия | Накопление аммония в мозге |
| Синдром Лея | Мутации гена GLUD1 |

## Методы измерения

### Спектрофотометрический метод
- **Принцип**: Глютамат + NAD⁺ + H₂O → α-кетоглутарат + NADH + NH₄⁺
- **Детекция**: Образование NADH (340 нм)
- **Образец**: Сыворотка, плазма (специальная обработка)

### Особенности измерения
- **Нестабильность**: ГДГ быстро теряет активность в сыворотке
- **Требования**: Немедленный анализ или заморозка
- **Интерференции**: Менее подвержена гемолизу

### Генетический анализ
- **GLUD1**: Ген, кодирующий ГДГ
- **Мутации**: Связаны с наследственными расстройствами

## Интерпретации

### Для исследователей
ГДГ — центральный фермент азотистого обмена:
- **Двойная роль**: Катаболизм аминокислот + детоксикация аммония
- **Энергетическая связь**: Регулируется АТФ/АДП
- **Митохондриальная функция**: Индикатор целостности органелл

### Для практиков
Понимание ГДГ важно для:
- Ранней диагностики печёночной дисфункции
- Оценки азотистого баланса
- Мониторинга метаболического статуса
- Интерпретации неврологических симптомов
