# CS.ENTITY.155: Pyruvate (Пируват)

## Определение

Пируват (пировиноградная кислота) — трёхуглеродный кетокарбоновая кислота, конечный продукт гликолиза и ключевой узловой метаболит углеводного метаболизма. Является точкой ветвления между анаэробным и аэробным путями метаболизма глюкозы.

## Структура
- **Химическая формула:** C₃H₄O₃
- **Систематическое название:** 2-оксопропановая кислота
- **Молекулярная масса:** 88.06 г/моль

## Метаболические пути

### 1. Гликолиз (источник)
```
Глюкоза → ... → Фосфоенолпируват (PEP) → Пируват (пируваткиназа)
```

### 2. Возможные судьбы пирувата

| Путь | Фермент | Продукт | Условия |
|------|---------|---------|---------|
| Аэробное окисление | ПДГК | Ацетил-КоА | Достаточно О₂, активна ПДГК |
| Глюконеогенез | ПКП | Оксалоацетат → Фосфоенолпируват | Низкий энергетический статус |
| Лактатный путь | ЛДГ | Лактат | Анаэробиоз, высокий NADH |
| Аланиновый путь | АЛАТ | Аланин | Мышечный метаболизм |

## Регуляция в transition period

### Ранняя лактация
- **Снижение окисления:** инактивация ПДГК (фосфорилирование)
- **Активация глюконеогенеза:** пируваткарбоксилаза активна
- **Лактат как предшественник:** из молочной кислоты (румен) → пируват → глюкоза

### Ключевой узел
Пируват — точка переключения между:
- **Окислением** (ацетил-КоА → АТП)
- **Глюконеогенезом** (глюкоза для молока)

## Клиническое значение

### При кетозе
- Снижение утилизации пирувата в цикл Кребса
- Относительное увеличение лактата
- Маркер метаболического статуса

### При гипогликемии
- Недостаточный субстрат для глюконеогенеза
- Низкий уровень пирувата
- Требуется глюкогенный прекурсор (пропионат, глицерол)
