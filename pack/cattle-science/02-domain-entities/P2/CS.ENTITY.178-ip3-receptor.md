# IP3-рецептор (Инозитол-1,4,5-трисфосфатный рецептор)

## Определение

**IP3-рецептор (IP3R)** — внутриклеточный канал высвобождения ионов кальция, локализованный в мембране эндоплазматического ретикулума. Активируется связыванием вторичного посредника инозитол-1,4,5-трисфосфата (IP3), образующегося при активации G-белок-сопряжённых рецепторов и фосфолипазы C. Является ключевым компонентом кальциевой сигнализации в неэxcitable клетках.

## Контекст SoTA

### Источник
- **McFadden et al., 2017** (CS.SOTA.039) — обзор метаболизма у молочных коров

### Значение
IP3R критически важен для:
- Кальциевой сигнализации в ответ на гормоны
- Регуляции клеточного метаболизма
- Генной экспрессии
- Апоптоза и клеточного стресса

## Физиологическая роль

### Сигнальный каскад
```
Гормон/лиганд → GPCR
        ↓
    Gq активация
        ↓
    Фосфолипаза C (PLC) активация
        ↓
    PIP2 гидролиз → IP3 + DAG
        ↓
    IP3 диффундирует к ЭПР
        ↓
    Связывание с IP3R
        ↓
    Высвобождение Ca²⁺ из ЭПР
        ↓
    Клеточный ответ (секреция, метаболизм, деление)
```

### В печени
- **Гликогенолиз**: Глюкагон → IP3 → Ca²⁺ → активация фосфорилазы киназы
- **Метаболизм**: Регуляция ферментов через Ca²⁺
- **Белковый синтез**: Ca²⁺-зависимая трансляция

### В молочной железе
- **Секреция молока**: Окситоцин → IP3 → Ca²⁺ → миоэпителиальное сокращение
- **Лактация**: Кальциевый сигналинг в альвеолах
- **Инволюция**: Регуляция апоптоза

### В молочных коровах
- **Эндокринная регуляция**: Ответ на метаболические гормоны
- **Метаболическая адаптация**: Ca²⁺-сигналинг в печени
- **Родовая деятельность**: Окситоциновый ответ
- **Мастит**: Воспалительная сигнализация

## Методы измерения

### Функциональные ассайи
- **³H-IP3-биндинг**: Аффинность и плотность рецепторов
- **Ca²⁺-флуоресценция**: Высвобождение из микросом
- **Перфузия ядер**: Одиночные канальные токи

### Генная экспрессия
- **qPCR**: ITPR1, ITPR2, ITPR3
- **Иммуноблоттинг**: Уровень белка

### Визуализация
- **Ca²⁺-imaging**: Динамика в живых клетках
- **FRET**: Конформационные изменения

## Интерпретации

### Для исследователей
IP3R — ключевой канал для гормональной кальциевой сигнализации:
- **Специфичность**: Активируется только IP3
- **Модуляция**: Множественные регуляторные механизмы
- **Изоформное разнообразие**: Тканеспецифические функции

### Для практиков
Понимание IP3R помогает:
- Объяснить механизм окситоцинового ответа
- Понять гормональную регуляцию метаболизма
- Оценить кальциевую сигнализацию в различных тканях
