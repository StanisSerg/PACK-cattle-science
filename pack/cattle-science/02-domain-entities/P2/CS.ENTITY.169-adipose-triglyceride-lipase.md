# Адипозитриацилглицероллипаза (ATGL)

## Определение

**Адипозитриацилглицероллипаза (ATGL, также известная как PNPLA2)** — фермент семейства пататин-подобных фосфолипаз, катализирующий лимитирующую стадию липолиза — гидролиз триглицеридов до диглицеридов. Является основным триглицеридлипазным ферментом в жировой ткани и играет ключевую роль в мобилизации жировых запасов во время отрицательного энергетического баланса.

## Контекст SoTA

### Источник
- **McFadden et al., 2017** (CS.SOTA.039) — обзор липидного метаболизма у молочных коров

### Значение
ATGL является:
- Основным инициатором липолиза в жировой ткани
- Целью регуляции перилипином и CGI-58
- Критическим ферментом для адаптации к НЭБ
- Отличным от HSL механизмом регуляции

## Свойства

### Ферментативные характеристики
| Свойство | Значение |
|----------|----------|
| EC Number | 3.1.1.79 |
| Ген | PNPLA2 |
| Молекулярная масса | ~55 кДа |
| Семейство | Пататин-подобные фосфолипазы (PNPLA) |
| Локализация | Поверхность липидных дроплетов |
| Субстратная специфичность | Триглицериды (TG) → Диглицериды (DG) |

### Структурные особенности
- **Пататин-домен**: Каталитический домен, характерный для семейства
- **Регуляторные участки**: Сайты связывания регуляторных белков
- **Липидная якорь**: Прикрепление к липидным дроплетам

### Кофакторы и регуляторы
| Фактор | Действие |
|--------|----------|
| CGI-58 (ABHD5) | Коактиватор, увеличивает активность ~20-fold |
| Перилипин | Блокирует доступ к субстрату (в базальном состоянии) |
| G0S2 | Ингибитор активности |

## Физиологическая роль

### В липолизе жировой ткани
```
Триглицериды →[ATGL]→ Диглицериды →[HSL]→ Моноглицериды →[MGL]→ Глицерол + НЖК
```

**ATGL отвечает за:**
- ~70% общего липолиза в жировой ткани
- Первую и лимитирующую стадию
- Специфичность к длинноцепочечным жирным кислотам

### Регуляция липолиза
| Состояние | Активность ATGL | Механизм |
|-----------|-----------------|----------|
| Базальное | Низкая | Перилипин блокирует доступ |
| Стимулированное | Высокая | PKA-фосфорилирование перилипина → CGI-58 освобождается |
| Голодание/НЭБ | Максимальная | Эпинефрин → cAMP/PKA-каскад |

### В метаболизме молочных коров
- **Период перехода**: Резкое увеличение активности после отёла
- **НЭБ**: Критический для обеспечения энергией
- **Кетогенез**: Источник НЖК для печени
- **Молочный жир**: Поставка жирных кислот для молока

## Клиническое и исследовательское значение

### Метаболические расстройства
```
Избыточная активность ATGL → Высокий липолиз → ГиперНЖК →
    ├─→ Жировая инфильтрация печени
    ├─→ Кетоз
    └─→ Иммунодепрессия
```

### Применение в исследованиях
- **Биопсии жировой ткани**: Активность ATGL как маркер липолитической способности
- **Генетические полиморфизмы**: PNPLA2 и индивидуальная вариабельность
- **Фармакологические мишени**: Модуляторы для контроля энергетического баланса

## Методы измерения

### Активность фермента
- **Принцип**: Гидролиз радиомеченых триолеинов
- **Образец**: Гомогенаты жировой ткани
- **Единицы**: нмоль/мин/мг белка

### Экспрессия гена
- **qPCR**: PNPLA2 мРНК
- **Иммуноблоттинг**: Белок ATGL
- **Иммуногистохимия**: Локализация на липидных дроплетах

### Косвенные маркеры
- **Соотношение глицерол/НЖК**: ATGL продуцирует больше глицерола относительно НЖК
- **Диглицериды**: Продукт специфичный для ATGL

## Интерпретации

### Для исследователей
ATGL — ключевой фермент инициации липолиза:
- **Лимитирующая стадия**: Определяет общую скорость мобилизации жира
- **Независимая регуляция**: Отличный от HSL механизм контроля
- **Тканеспецифичность**: Доминирует в белой жировой ткани

### Для практиков
Понимание ATGL объясняет:
- Почему липолиз может продолжаться даже при HSL-ингибировании
- Механизм постепенной мобилизации жировых запасов
- Потенциальные мишени для контроля энергетического баланса
- Индивидуальные различия в склонности к кетозу
