# PFKFB (Фосфофруктокиназа-2/фруктозо-2,6-бисфосфатаза)

## Определение

**PFKFB (6-фосфофрукто-2-киназа/фруктозо-2,6-бисфосфосфатаза)** — бифункциональный фермент, катализирующий как синтез, так и деградацию фруктозо-2,6-бисфосфата (F2,6BP) — самого мощного аллостерического активатора фосфофруктокиназы-1 (PFK-1) и ключевого регулятора гликолиза. Содержит две каталитические активности в одном полипептиде, направленные противоположно.

## Контекст SoTA

### Источник
- **Aiello et al., 1984** (CS.SOTA.026) — исследование метаболизма у молочных коров

### Значение
PFKFB является:
- Ключевым регулятором скорости гликолиза
- Точкой интеграции гормональных и нутритивных сигналов
- Мишенью для фармакологической модуляции метаболизма
- Критическим для энергетического баланса

## Свойства

### Бифункциональная природа
| Домен | Реакция | Эффект на метаболизм |
|-------|---------|---------------------|
| Киназный (N-терминал) | F6P + ATP → F2,6BP + ADP | Активация гликолиза |
| Фосфатазный (C-терминал) | F2,6BP + H₂O → F6P + Pi | Ингибирование гликолиза |

### Изоформы PFKFB
| Изоформа | Ген | Ткань | Особенности |
|----------|-----|-------|-------------|
| PFKFB1 | PFKFB1 | Печень, скелетные мышцы | Чувствителен к cAMP/PKA |
| PFKFB2 | PFKFB2 | Сердце | Альтернативный сплайсинг |
| PFKFB3 | PFKFB3 | Убиквитарный, индуцируемый | Высокая киназная активность |
| PFKFB4 | PFKFB4 | Тестис, опухоли | Регулируется pH |

### Регуляция PFKFB1 (печеночная изоформа)
```
Глюкагон/Эпинефрин → cAMP ↑ → PKA активация
            ↓
    PFKFB1 фосфорилируется (Ser32)
            ↓
    Киназная активность ↓↓, Фосфатазная активность ↑↑
            ↓
    F2,6BP ↓↓↓ → PFK-1 ингибируется → Гликолиза ↓
            ↓
    Глюконеогенез стимулируется
```

## Физиологическая роль

### Регуляция гликолиза/глюконеогенеза
| Условие | F2,6BP | PFK-1 | Гликолиза | Глюконеогенез |
|---------|--------|-------|-----------|---------------|
| Приём пищи | Высокий | Активен | Высокая | Низкая |
| Голодание | Низкий | Ингибирован | Низкая | Высокая |
| Инсулин | ↑ | Активен | ↑ | ↓ |
| Глюкагон | ↓↓ | Ингибирован | ↓↓ | ↑↑ |

### В печени
- **Бифункциональный переключатель**: Координирует гликолиз/глюконеогенез
- **Гормональная регуляция**: Инсулин vs. глюкагон
- **Глюкозный сенсор**: Отвечает на уровень субстрата

### В молочных коровах
- **НЭБ**: Снижение F2,6BP направляет глюкозу к глюконеогенезу
- **Лактация**: Поддержание гликолиза для энергии
- **Метаболическая адаптация**: Переключение между путями

## Клиническое и исследовательское значение

### Метаболические расстройства
| Состояние | Механизм |
|-----------|----------|
| Рак | PFKFB3 оверэкспрессия → Warburg эффект |
| Ишемия | PFKFB3 активация → защитная гликолиза |
| Сахарный диабет | Дисрегуляция F2,6BP |

### В ветеринарии
- **Кетоз**: Регуляция глюкозного баланса
- **Молочная лихорадка**: Кальциевый и энергетический метаболизм
- **Переходный период**: Адаптация метаболических путей

## Методы измерения

### Активность фермента
- **Киназная**: F6P + [γ-³²P]ATP → [³²P]F2,6BP
- **Фосфатазная**: F2,6BP → F6P + Pi
- **Образец**: Тканевые гомогенаты

### Уровень F2,6BP
- **Спектрофотометрия**: Связанная система с PFK-1 и aldolase
- **HPLC**: Прямое определение

### Генная экспрессия
- **qPCR**: PFKFB1-4 изоформы
- **Иммуноблоттинг**: Специфичные антитела

## Интерпретации

### Для исследователей
PFKFB — мастер-регулятор гликолитического потока:
- **Бифункциональность**: Две противоположные активности
- **Гормональная интеграция**: cAMP/PKA регуляция
- **Тканеспецифичность**: Разные изоформы для разных функций

### Для практиков
Понимание PFKFB объясняет:
- Почему глюкагон ингибирует гликолиз
- Механизм переключения между гликолизом и глюконеогенезом
- Роль инсулина в стимуляции глюкозного утилизации
- Потенциальные мишени для метаболической модификации
