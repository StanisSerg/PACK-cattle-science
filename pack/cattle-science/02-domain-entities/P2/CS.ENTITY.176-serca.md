# SERCA (Сарко(эндо)плазматический ретикулум Ca²⁺-АТФаза)

## Определение

**SERCA (Sarco/Endoplasmic Reticulum Calcium ATPase)** — семейство мембранных белков-насосов, использующих энергию гидролиза АТФ для активного транспорта ионов кальция (Ca²⁺) из цитоплазмы в просвет саркоплазматического или эндоплазматического ретикулума. Поддерживает низкий цитоплазматический уровень Ca²⁺ (~100 нМ) необходимый для клеточной жизнедеятельности и обеспечивает запас Ca²⁺ для сигнальных каскадов.

## Контекст SoTA

### Источник
- **McFadden et al., 2017** (CS.SOTA.039) — обзор метаболизма у молочных коров

### Значение
SERCA критически важна для:
- Релаксации мышц после сокращения
- Поддержания кальциевого гомеостаза
- Регуляции клеточной сигнализации
- Предотвращения кальциевого стресса

## Свойства

### Изоформы SERCA
| Изоформа | Ген | Основная локализация | Функция |
|----------|-----|---------------------|---------|
| SERCA1 | ATP2A1 | Скелетные мышцы, быстрые | Мышечная релаксация |
| SERCA2a | ATP2A2 | Сердечная мышца | Кардиаконтрактильность |
| SERCA2b | ATP2A2 | Убиквитарная | Общий Ca²⁺-гомеостаз |
| SERCA3 | ATP2A3 | Эндотелий, лимфоциты | Специфические функции |

### Механизм работы
```
Цитоплазматическая сторона:
    2 Ca²⁺ + ATP + E1 → Phosphorylation (E1-P)
                ↓
    Conformational change (E1-P → E2-P)
                ↓
    Luminal side:
    Ca²⁺ release + Pi + E2 → Dephosphorylation
                ↓
    E2 → E1 (reset)
```

### Характеристики транспорта
| Параметр | Значение |
|----------|----------|
| Стохиометрия | 2 Ca²⁺ / 1 ATP |
| Км для Ca²⁺ | ~0.1-1 мкМ |
| Км для ATP | ~0.01-0.1 мМ |
| Ингибиторы | Тапсигаргин, цикlopiazonic acid |

## Физиологическая роль

### В скелетных мышцах
```
Сокращение: Ca²⁺ высвобождается (RyR1) → связывание с тропонином C
                                    ↓
Релаксация: Ca²⁺ захватывается SERCA1 → [Ca²⁺]цитоплазмы падает
                                    ↓
                            Мышца расслабляется
```

### В сердечной мышце
- **SERCA2a**: Основной изоформ
- **Фосфоламбан**: Регуляторный белок (ингибитор)
- **β-адренергическая стимуляция**: Фосфорилирование фосфоламбана → активация SERCA

### В печени и других тканях
- **SERCA2b**: Поддержание Ca²⁺ в ЭПР
- **Белок-связующий Ca²⁺**: Кальретикулин, кальнексин
- **ЭР-стресс**: Дисфункция SERCA нарушает фолдинг белков

### В молочных коровах
- **Мышечная функция**: Особенно важна для родовой деятельности
- **Метаболическая адаптация**: Ca²⁺-сигналинг в печени
- **Гипокальциемия**: Связь с общим кальциевым балансом
- **Мастит**: Ca²⁺-гомеостаз в молочной железе

## Клиническое и исследовательское значение

### Мышечные расстройства
| Состояние | Механизм | Последствия |
|-----------|----------|-------------|
| Брокада | Мутации SERCA1 | Мalignant гипертермия |
| Дилатационная кардиомиопатия | Снижение SERCA2a | Сердечная недостаточность |
| Мышечная дистрофия | Дисрегуляция Ca²⁺ | Дегенерация волокон |

### Метаболические аспекты
- **ЭР-стресс**: Нарушение SERCA → накопление нефолденных белков
- **Апоптоз**: Изменение Ca²⁺-гомеостаза
- **Инсулинорезистентность**: Ca²⁺-сигналинг в мышцах и печени

## Методы измерения

### Активность АТФазы
- **Принцип**: Ca²⁺-зависимое высвобождение Pi из ATP
- **Образец**: Микросомальные фракции
- **Единицы**: мкмоль Pi/мин/мг белка

### Генная экспрессия
- **qPCR**: ATP2A1, ATP2A2, ATP2A3
- **Иммуноблоттинг**: Специфичные антитела к изоформам

### Ca²⁺-имaging
- **Флуоресцентные зонды**: Fura-2, Fluo-4
- **Конфокальная микроскопия**: Динамика Ca²⁺

## Интерпретации

### Для исследователей
SERCA — критический компонент кальциевого гомеостаза:
- **Энергозависимый насос**: Требует АТФ
- **Изоформная специфичность**: Разные ткани — разные изоформы
- **Регулируемость**: Фосфорилирование, взаимодействие с белками

### Для практиков
Понимание SERCA важно для:
- Оценки мышечной функции и метаболизма
- Понимания родовой деятельности
- Диагностики и терапии гипокальциемии
- Оценки стресса и благополучия
