# Рианодиновый рецептор (RyR)

## Определение

**Рианодиновый рецептор (RyR)** — большой тетрамерный канал высвобождения ионов кальция, локализованный в мембране саркоплазматического/эндоплазматического ретикулума. Назван по способности связывать алкалоид рианодин. Является ключевым компонентом электро-механической связи в мышцах и играет центральную роль в высвобождении Ca²⁺ из внутренних запасов.

## Контекст SoTA

### Источник
- **McFadden et al., 2017** (CS.SOTA.039) — обзор метаболизма у молочных коров

### Значение
RyR критически важен для:
- Сокращения скелетных мышц
- Сердечной контрактильности
- Кальциевой сигнализации в нейронах
- Регуляции клеточного метаболизма

## Свойства

### Изоформы RyR
| Изоформа | Ген | Основная локализация | Функция |
|----------|-----|---------------------|---------|
| RyR1 | RYR1 | Скелетные мышцы | Электро-механическая связь |
| RyR2 | RYR2 | Сердечная мышца, нейроны | Кардиостимуляция, нейросигналинг |
| RyR3 | RYR3 | Мозг, гладкие мышцы | Нейромодуляция, сокращение |

### Структурные характеристики
| Параметр | Значение |
|----------|----------|
| Молекулярная масса (мономер) | ~550 кДа |
| Структура | Тетрамер (~2.2 МДа) |
| Размер канала | ~2.2 нм в диаметре |
| Селективность | Ca²⁺ > K⁺, Na⁺ |

### Регуляция
| Активаторы | Ингибиторы | Модуляторы |
|------------|------------|------------|
| Ca²⁺ (CICR) | Рианодин (высокие конц.) | Mg²⁺ (ингибитор) |
| ATP | Рутений красный | Калмодулин |
| Кофеин | Прокаин | Фосфорилирование |
| DHPR (в мышцах) | Тетракаин | Окислительный стресс |

## Физиологическая роль

### В скелетных мышцах (RyR1)
```
Деполяризация Т-тубулы
        ↓
Активация дигидропиридиновых рецепторов (DHPR)
        ↓
Механическая связь (без Ca²⁺-входа)
        ↓
Активация RyR1
        ↓
Массивное высвобождение Ca²⁺ из СР (CICR)
        ↓
Связывание с тропонином C → Сокращение
```

### В сердечной мышце (RyR2)
```
Деполяризация
        ↓
Ca²⁺-вход через L-тип каналы (Ca²⁺-индуцированная Ca²⁺-регургитация)
        ↓
Активация RyR2 (CICR)
        ↓
Бóльшее высвобождение Ca²⁺
        ↓
Сокращение
```

### В молочных коровах
- **Родовая деятельность**: Критически важен для сокращений матки
- **Скелетная мышца**: Двигательная активность, постуральная поддержка
- **Сердечная функция**: Кровообращение, стрессовая адаптация
- **Метаболизм**: Ca²⁺-сигналинг в печени и других тканях

## Клиническое и исследовательское значение

### Малигнантная гипертермия
| Аспект | Описание |
|--------|----------|
| Генетика | Мутации RYR1 |
| Триггер | Летучие анестетики, суксаметоний |
| Механизм | Гиперактивация RyR1 → массивный Ca²⁺-всплеск |
| Клиника | Гипертермия, ригидность, рабдомиолиз |

### Другие патологии
- **Центральное ядро myopathy**: Мутации RYR1
- **Аритмогенная дисплазия правого желудочка**: Мутации RYR2
- **Стресс-индуцированная кардиомиопатия**: Дисфункция RyR2

### В ветеринарии
- **Анестезия**: Скрининг на малигнантную гипертермию
- **Мышечные расстройства**: Диагностика миопатий
- **Родовспоможение**: Регуляция сократительной активности

## Методы измерения

### Функциональные ассайи
- **³H-рианодин-биндинг**: Аффинность и плотность рецепторов
- **Ca²⁺-флуоресценция**: Высвобождение из микросом
- **Электрофизиология**: Одноканальные токи

### Генетический анализ
- **Секвенирование**: Мутации RYR1, RYR2
- **Genotyping**: Предрасположенность к малигнантной гипертермии

### Белковый анализ
- **Иммуноблоттинг**: Уровень экспрессии
- **Иммунофлуоресценция**: Локализация в СР

## Интерпретации

### Для исследователей
RyR — мощный канал высвобождения Ca²⁺:
- **CICR механизм**: Самоусиливающаяся активация
- **Множественная регуляция**: Лиганды, фосфорилирование, окисление
- **Структурная сложность**: Один из крупнейших ионных каналов

### Для практиков
Понимание RyR важно для:
- Безопасной анестезии
- Диагностики мышечных расстройств
- Управления родовой деятельностью
- Оценки сердечной функции
