# Ацетил-КоА-карбоксилаза (ACC)

## Определение

**Ацетил-КоА-карбоксилаза (АКК, ACC, EC 6.4.1.2)** — биотин-зависимый фермент, катализирующий карбоксилирование ацетил-КоА до малонил-КоА — ключевой промежуточный продукт синтеза жирных кислот. Это первая и лимитирующая стадия de novo липогенеза и важнейшая регуляторная точка, контролирующая переключение между синтезом и окислением жирных кислот.

## Контекст SoTA

### Источник
- **Aiello et al., 1984** (CS.SOTA.026) — исследование метаболизма у молочных коров

### Значение
АКК является:
- Лимитирующим ферментом синтеза жирных кислот
- Ключевым регулятором липидного метаболизма
- Мишенью гормональной и нутритивной регуляции
- Точкой интеграции углеводного и липидного обмена

## Свойства

### Изоформы АКК
| Изоформа | Ген | Локализация | Основная функция |
|----------|-----|-------------|------------------|
| ACC1 (ACCα) | ACACA | Цитоплазма | Синтез жирных кислот (липогенез) |
| ACC2 (ACCβ) | ACACB | Митохондрии | Регуляция β-окисления |

### Ферментативные характеристики
| Параметр | Значение |
|----------|----------|
| EC Number | 6.4.1.2 |
| Реакция | Acetyl-CoA + HCO₃⁻ + ATP → Malonyl-CoA + ADP + Pi |
| Кофактор | Биотин (витамин B7) |
| Молекулярная масса | ~265 кДа (ACC1), ~280 кДа (ACC2) |
| Структура | Гомодимер или полимер |

### Двустадийный механизм
```
Стадия 1: Биотин + ATP + HCO₃⁻ → Карбоксибиотин + ADP + Pi
                    ↓ (транслокация)
Стадия 2: Карбоксибиотин + Ацетил-КоА → Малонил-КоА + Биотин
```

## Физиологическая роль

### В синтезе жирных кислот
```
Ацетил-КоА + CO₂ + ATP → Малонил-КоА (АКК)
                                ↓
                        Жирнокислотсинтаза (FAS)
                                ↓
                        Пальмитат (C16:0)
                                ↓
                        Триглицериды, фосфолипиды
```

**Роль АКК:**
- **Лимитирующая стадия**: Определяет общий поток
- **Поставка малонил-КоА**: Строительный блок для FAS
- **Регуляторный переключатель**: Интеграция сигналов

### В регуляции β-окисления
```
ACC2 в митохондриях:
    ↓
Локальный синтез малонил-КоА
    ↓
Ингибирование КПТ1 (аллостерическое)
    ↓
Блокада входа жирных кислот в митохондрии
    ↓
Предотвращение β-окисления при синтезе
```

### В молочных коровах
- **Период сухостоя**: Активация липогенеза
- **Начало лактации**: Ингибирование, переключение на окисление
- **Жировая инфильтрация**: Дисрегуляция при НЭБ
- **Молочный жир**: Субстрат для синтеза

## Клиническое и исследовательское значение

### Метаболические расстройства
| Состояние | Изменение АКК | Последствия |
|-----------|---------------|-------------|
| Ожирение | ↑ Активность | Усиленный липогенез |
| Инсулинорезистентность | Дисрегуляция | Нарушение липидного баланса |
| Жировой гепатоз | ↑ Экспрессия | Накопление триглицеридов |

### В ветеринарии
- **Оценка липогенеза**: Активность АКК в биоптатах
- **Нутригеномика**: Влияние жирных кислот на экспрессию
- **Метаболическая адаптация**: Переходный период

## Методы измерения

### Активность фермента
- **Принцип**: Фиксация ¹⁴CO₂ в малонил-КоА
- **Альтернатива**: Связанная ферментативная система
- **Образец**: Тканевые гомогенаты

### Генная экспрессия
- **qPCR**: ACACA, ACACB
- **Иммуноблоттинг**: ACC1, ACC2 белки
- **Фосфорилирование**: Активная vs. неактивная форма

### Метаболитные маркеры
- **Малонил-КоА**: Прямой продукт
- **Соотношение малонил-КоА/ацетил-КоА**: Индикатор активности

## Интерпретации

### Для исследователей
АКК — мастер-регулятор липидного баланса:
- **Двойная роль**: Липогенез + регуляция окисления
- **Изоформная специфичность**: ACC1 (цитоплазма) vs. ACC2 (митохондрии)
- **Множественная регуляция**: Фосфорилирование, аллостерия, транскрипция

### Для практиков
Понимание АКК объясняет:
- Почему инсулин стимулирует накопление жира
- Механизм переключения между синтезом и окислением
- Роль биотина в метаболизме
- Потенциальные мишени для контроля жировых запасов
