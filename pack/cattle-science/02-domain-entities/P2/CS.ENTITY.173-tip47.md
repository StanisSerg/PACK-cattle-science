# TIP47 (Tail-interacting protein of 47 kDa)

## Определение

**TIP47 (PLIN3, Perilipin-3)** — белок семейства PAT, изначально идентифицированный как фактор, связывающий цитоплазматические хвосты маннозо-6-фосфатных рецепторов. Позже был обнаружен на поверхности липидных дроплетов. Играет роль в транспорте белков между компартментами эндомембранной системы и в регуляции липидного метаболизма.

## Контекст SoTA

### Источник
- **McFadden et al., 2017** (CS.SOTA.039) — обзор липидного метаболизма у молочных коров

### Значение
TIP47 участвует в:
- Везикулярном транспорте между Гольджи и эндосомами
- Формировании липидных дроплетов
- Регуляции липидного обмена
- Связи между эндомембранной системой и липидными дроплетами

## Свойства

### Структурные характеристики
| Свойство | Значение |
|----------|----------|
| Ген | PLIN3 (ранее TIP47) |
| Молекулярная масса | ~47 кДа |
| Семейство | PAT (Perilipin-ADRP-TIP47) |
| Структура | Четырёхспиральная мотив |
| Локализация | Цитоплазма, Гольджи, эндосомы, липидные дроплеты |

### Двойная функция
```
TIP47
    ├─→ Транспорт белков (маннозо-6-фосфатный рецептор)
    └─→ Липидный метаболизм (липидные дроплеты)
```

### Субклеточная локализация
- **Цитоплазматический пул**: Для транспорта
- **Ассоциация с дроплетами**: При липидной перегрузке
- **Гольджи-эндосомный транспорт**: Основная функция

## Физиологическая роль

### В эндомембранном транспорте
- **Mannose-6-phosphate receptor (M6PR)**: Транспорт лизосомальных ферментов
- **Retrograde transport**: Из эндосом в Гольджи
- **Сортировка белков**: В лизосомальном направлении

### В липидном метаболизме
- **Дроплетогенез**: Ранняя стадия формирования дроплетов
- **Малые липидные дроплеты**: Предпочтительная локализация
- **Предшественник**: Возможно, предшествует адипофиллину и перилипину

### В молочных коровах
- **Метаболическая адаптация**: Роль при липидной перегрузке
- **Печень**: Возможное участие в VLDL ассамблее
- **Молочная железа**: Транспорт липидов для синтеза молока

## Интерпретации

### Для исследователей
TIP47 — уникальный PAT-белок с двойной функцией:
- **Транспортная функция**: Эндомембранная система
- **Липидная функция**: Дроплетогенез
- **Ранняя маркер**: Малые липидные дроплеты

### Для практиков
TIP47 менее изучен в контексте молочного скота, но может быть релевантен для:
- Понимания ранних стадий липидного накопления
- Исследования связи между секреторными путями и липидным метаболизмом
- Разработки новых терапевтических подходов
