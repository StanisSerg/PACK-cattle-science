# CS.ENTITY.152: Mitochondria (Митохондрии)

## Определение

Митохондрии — двумембранные органеллы эукариотических клеток, являющиеся основными центрами энергетического метаболизма. Содержат собственную ДНК и обеспечивают синтез АТФ через окислительное фосфорилирование.

## Структура

### Внешняя мембрана
- Проницаемая для малых молекул
- Содержит порины
- Контактные сайты с ВПСР

### Внутренняя мембрана
- Непроницаемая для ионов
- Кристы — складки мембраны
- Электрон-транспортная цепь

### Митохондриальный матрикс
- Митохондриальная ДНК
- Рибосомы
- Ферменты цикла Кребса
- β-окисление жирных кислот

## Функции в печени коровы

### β-окисление жирных кислот
- Основной путь утилизации NEFA
- Производство ацетил-КоА
- Генерация АТП

### Кетогенез
- Образование кетоновых тел при избытке ацетил-КоА
- Альтернативный путь при ограниченном оксалоацетате

### Глюконеогенез
- Связь с циклом Кребса
- Обеспечение предшественниками

## Адаптация в transition period

### Ремоделирование
- Увеличение числа и размера
- Изменение морфологии крист
- Биогенез митохондрий

### Дисфункция
- Неполное β-окисление → ацилкарнитины
- Окислительный стресс
- Митохондриальная пермеабилизация

## Маркеры функции

| Маркер | Значение | Интерпретация |
|--------|----------|---------------|
| Ацилкарнитины | Повышены | Неполное окисление |
| Цитрат/изоцитрат | Снижен | Замедление цикла Кребса |
| β-гидроксиацетил-КоА | Повышен | Активный кетогенез |

## Клиническое значение

### При кетозе
- Перегрузка β-окислением
- Митохондриальная дисфункция
- Терапевтическая цель

### При жировом гепатозе
- Накопление триацилглицеридов
- Инактивация митохондрий
- Восстановление функции
