## Методология заполнения

1. **Авто-извлечение:** Запустить `scripts/extract-entity-mentions.sh`
2. **Ручная доработка:** Проверить контекст, добавить синтез
3. **Валидация:** Убедиться в полноте и консистентности
4. **Обновление матрицы:** Добавить новые связи в 02B9
5. **Обновление графа:** Добавить узлы и рёбра в 02B10 + JSON
