# CS.ENTITY.053: Машинное обучение (Machine learning)

> Машинное обучение (machine learning) — раздел искусственного интеллекта, использующий статистические методы и алгоритмы для анализа данных, выявления паттернов и прогнозирования событий (заболеваний, продуктивности, репродуктивных показателей) в молочном скотоводстве без явного программирования, являющийся ключевой технологией precision dairy farming.

---

## Определение

**Машинное обучение (machine learning)** — метод анализа данных, который автоматически строит модели на основе обучающих данных и использует эти модели для прогнозирования на новых данных.

**Типы машинного обучения:**

| Тип | Описание | Применение в скотоводстве |
|-----|----------|---------------------------|
| **Обучение с учителем** | Модель обучается на размеченных данных | Предсказание заболеваний |
| **Обучение без учителя** | Выявление скрытых паттернов | Кластеризация коров |
| **Обучение с подкреплением** | Оптимизация через пробу и ошибку | Управление кормлением |
| **Глубокое обучение** | Нейронные сети с множеством слоёв | Анализ изображений, спектров |

**Диагностические критерии моделей:**

| Метрика | Описание | Целевое значение |
|---------|----------|------------------|
| **Accuracy** | Общая точность | > 70% |
| **Sensitivity (Recall)** | Чувствительность | > 70% |
| **Specificity** | Специфичность | > 70% |
| **AUROC** | Площадь под ROC-кривой | > 0.70 |
| **Precision** | Точность положительных | > 70% |

## Этиология/Патогенез

### Процесс машинного обучения

```
Сбор данных (молочные спектры, активность, продуктивность)
    ↓
Предобработка (очистка, нормализация, feature engineering)
    ↓
Разделение на обучающую и тестовую выборки
    ↓
Обучение модели (алгоритм: PLS-DA, Random Forest, LSTM)
    ↓
Валидация (кросс-валидация, внешняя валидация)
    ↓
Оптимизация гиперпараметров
    ↓
Развёртывание модели
    ↓
Предсказание на новых данных
```

### Алгоритмы

**Традиционные:**

| Алгоритм | Тип | Применение |
|----------|-----|------------|
| **PLS-DA** | Линейный | Классификация спектров |
| **Random Forest** | Ансамбль | Предсказание заболеваний |
| **SVM** | Линейный/нелинейный | Классификация |
| **Ridge/Lasso** | Линейный | Регрессия |

**Глубокое обучение:**

| Алгоритм | Тип | Применение |
|----------|-----|------------|
| **LSTM** | Рекуррентная сеть | Временные ряды |
| **CNN** | Сверточная сеть | Изображения |
| **Transformer** | Attention-based | Сложные паттерны |

### Применение в скотоводстве

**Предсказание заболеваний (Lin 2025):**

```
Молочные спектры (FTIR)
    ↓
Модель ML (PLS-DA, LSTM)
    ↓
Предсказание риска кетоза/метрита/DA/мастита
    ↓
Раннее предупреждение
    ↓
Вмешательство
```
