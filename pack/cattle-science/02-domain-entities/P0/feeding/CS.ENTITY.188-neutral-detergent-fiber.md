# CS.ENTITY.188: Нейтрально-детергентная клетчатка (NDF)

> Крупнейшая фракция углеводов в рационе молочных коров, определяющая структуру рациона, скорость прохождения корма через пищеварительный тракт, профиль летучих жирных кислот и риск руминального ацидоза.

---

## Определение

**Нейтрально-детергентная клетчатка (NDF)** — комплекс структурных углеводов растительной клеточной стенки, включающий целлюлозу, гемицеллюлозу и лигнин, который не растворяется при обработке нейтральным детергентом. NDF представляет собой полимеры с β-гликозидными связями, устойчивыми к ферментам желудочного сока моногастрических животных, но подвергающимися микробиальной ферментации в рубце жвачных.

**Химический состав NDF:**
| Компонент | Доля в NDF | Ферментируемость | Функция в клеточной стенке |
|-----------|------------|------------------|----------------------------|
| **Целлюлоза** | 40-50% | Умеренная-высокая | Скелетная структура |
| **Гемицеллюлоза** | 25-35% | Высокая | Связывающий материал |
| **Лигнин** | 10-20% | Нулевая | Жесткость, защита |

**Современная номенклатура анализа NDF (по Tylutki 2017):**
- **aNDF** — амилезная NDF (устаревший термин)
- **aNDFom** — амилезная NDF на основе органического вещества (рекомендуемый метод)
- **uNDF** — недоступная NDF (u30, u120, u240 — непереваренная за 30, 120, 240 часов)
- **peNDF** — физически эффективная NDF

**Диагностические критерии:**

| Параметр | Дефицит | Норма | Избыток |
|----------|---------|-------|---------|
| **NDF в рационе** | < 25% | 28-35% | > 40% |
| **NDF из грубых кормов** | < 16% | 19-24% | > 28% |
| **peNDF** | < 22% | 28-32% | > 38% |
| **u240 %BW** | — | 0.30-0.40% | > 0.50% |

## Этиология/Патогенез

### Ферментация NDF в рубце

**Механизм микробиальной деградации:**
```
NDF (целлюлоза, гемицеллюлоза)
         ↓ Целлюлолитические бактерии
    Олигосахариды
         ↓
    Моносахариды (глюкоза, ксилоза)
         ↓
    Пируват
         ↓
    Ацетат + Бутират + CO₂ + CH₄ + Микробная масса
```

**Основные целлюлолитические бактерии:**
| Таксон | Относительная доля | Специализация |
|--------|-------------------|---------------|
| *Fibrobacter succinogenes* | 20-30% | Целлюлоза → ацетат |
| *Ruminococcus albus* | 15-20% | Кристаллическая целлюлоза |
| *Ruminococcus flavefaciens* | 10-15% | Целлюлоза, гемицеллюлоза |
| *Butyrivibrio fibrisolvens* | 10-15% | Гемицеллюлоза → ацетат, бутират |
| *Prevotella* spp. | 15-20% | Гемицеллюлоза, пектин |

**Кинетика ферментации NDF (Tylutki 2017):**
| Пул | Доля в NDF | Скорость деградации | Время разложения |
|-----|------------|---------------------|------------------|
| **Быстрый** | 30-55% | 10-20%/час | 12-24 часа |
| **Медленный** | 15-30% | 2-5%/час | 48-144 часа |
| **Недоступный (uNDF)** | 15-30% | 0%/час | Не ферментируется |

### Влияние на метаболизм

**Формирование профиля VFA:**
- Высокое содержание NDF → повышение ацетата (60-75% VFA)
- Умеренное содержание NDF → баланс ацетат:пропионат (2.5-3.5:1)
- Низкое содержание NDF → сдвиг к пропионату (риск ацидоза)

**Связь с жиром молока:**
| NDF (% СВ) | Ацетат (ммоль/л) | Жир молока (%) |
|------------|------------------|----------------|
| > 35% | 70-80 | 4.0-4.5 |
| 28-35% | 60-70 | 3.5-4.0 |
| 25-28% | 50-60 | 3.0-3.5 |
| < 25% | < 50 | < 3.0 (риск MFD) |

## Классификация

### По физической эффективности

**Физически эффективная NDF (peNDF)** — фракция NDF, стимулирующая жвачку и слюноотделение:

| Фракция | Размер частиц | Эффективность (pef) | Функция |
|---------|---------------|---------------------|---------|
| **Крупная** | > 19 мм | 1.0 | Максимальная стимуляция жвачки |
| **Средняя** | 8-19 мм | 0.5-0.8 | Умеренная стимуляция |
| **Мелкая** | < 8 мм | 0.1-0.3 | Минимальная стимуляция |

**Расчет peNDF:**
```
peNDF = NDF × pef
pef = Σ (% фракции × коэффициент эффективности)
```

### По переваримости

| Категория | NDFD (% NDF) | Примеры кормов |
|-----------|--------------|----------------|
| **Высокая** | > 60% | BMR кукурузный силос, молодая трава |
| **Средняя** | 45-60% | Кукурузный силос, люцерна |
| **Низкая** | < 45% | Солома, зрелое сено, хлопковый жмых |

## Эпидемиология

### Распространенность в рационах

| Тип хозяйства | Средний NDF | Диапазон | Типичные проблемы |
|---------------|-------------|----------|-------------------|
| **Высокопродуктивные** | 28-30% | 25-33% | Риск ацидоза |
| **Средние удои** | 32-36% | 30-38% | Оптимум |
| **Низкие удои/сухостой** | 38-45% | 35-50% | Ограничение DMI |

### Экономический impact

| Параметр | Значение | Механизм |
|----------|----------|----------|
| **Снижение жира при низком NDF** | -0.5-1.0% | Недостаток ацетата |
| **Потери от SARA** | $100-300/корова | Последствия ацидоза |
| **Ограничение DMI при высоком NDF** | -1-3 кг | Fill effect |
| **Оптимизация NDF** | +$50-150/корова | Баланс энергии и здоровья |

## Лечение/Применение/Управление

### Рекомендации по уровню NDF

| Фаза лактации | NDF (% СВ) | NDF из грубых кормов | peNDF |
|---------------|------------|----------------------|-------|
| **Сухостой (ранний)** | 40-50% | 28-35% | 35-42% |
| **Сухостой (поздний)** | 35-42% | 24-30% | 30-36% |
| **Ранняя лактация** | 28-32% | 19-22% | 26-30% |
| **Пик лактации** | 28-33% | 19-23% | 27-32% |
| **Средняя лактация** | 30-36% | 21-26% | 28-34% |

### Стратегии кормления

**При риске ацидоза (увеличение NDF):**
1. Добавить длинную структурную клетчатку (сено 3-5 см)
2. Увеличить долю грубых кормов
3. Использовать BMR гибриды (высокая NDFD)
4. Оптимизировать измельчение сенажа

**При ограничении DMI (оптимизация NDF):**
1. Использовать корма с высокой NDFD
2. Снизить uNDF (недоступную фракцию)
3. Увеличить NFC без снижения peNDF
4. Применить энзимы клетчатки

### Мультипульная модель (CNCPS)

```
NDF = Быстрый пул + Медленный пул + Недоступный пул

Быстрый пул (% NDF) = 100 - u30
Медленный пул (% NDF) = u30 - u240
Недоступный пул (uNDF) = u240

Пример:
u30 = 45%, u240 = 20%
Быстрый = 55%, Медленный = 25%, Недоступный = 20%
```
