# CS.ENTITY.190: pH рубца

> Критический показатель, отражающий баланс между продукцией органических кислот при ферментации и их абсорбцией/нейтрализацией, определяющий здоровье микробиома, эффективность пищеварения и риск метаболических заболеваний.

---

## Определение

**pH рубца** — отрицательный десятичный логарифм концентрации водородных ионов в рубцовой жидкости, являющийся интегральным показателем интенсивности микробиального брожения, скорости абсорбции летучих жирных кислот и буферной емкости рубцового содержимого.

**Физиологические границы:**
| Парамететр | Значение | Клиническое значение |
|------------|----------|---------------------|
| **Физиологический диапазон** | 6.0-7.0 | Нормальная ферментация |
| **Оптимальный диапазон** | 6.0-6.8 | Оптимум для микробиома |
| **Граница стресса** | 5.8-6.0 | Компенсация механизмов |
| **Патологический** | < 5.8 | Субклинический ацидоз |
| **Критический** | < 5.5 | Острый ацидоз |
| **Щелочной сдвиг** | > 7.2 | Редко, при избытке буферов |

**Диагностические критерии (по Zebeli 2008):**

| Состояние | Среднесуточный pH | Время pH < 5.8 | Интерпретация |
|-----------|-------------------|----------------|---------------|
| **Норма** | > 6.16 | < 3 ч/день | Здоровый рубец |
| **Граница** | 5.8-6.16 | 3-5.24 ч/день | Риск SARA |
| **SARA** | < 6.16 | > 5.24 ч/день | Субклинический ацидоз |
| **Острый ацидоз** | < 5.5 | > 8 ч/день | Клинический ацидоз |

---

## Этиология/Патогенез

### Баланс pH в рубце

**Уравнение баланса:**
```
Изменение pH = (Продукция кислот + Буферные способности входа) - (Абсорбция + Нейтрализация + Буферные способности выхода)
```

**Факторы снижения pH:**
| Фактор | Механизм | Интенсивность |
|--------|----------|---------------|
| **Быстрая ферментация крахмала** | Продукция пропионата и лактата | Высокая |
| **Высокие концентраты** | Общее увеличение VFA | Высокая |
| **Мелкое измельчение** | Ускорение ферментации | Умеренная |
| **Низкая буферность** | Недостаток слюны и клетчатки | Умеренная |
| **Редкое кормление** | Пиковая продукция кислот | Умеренная |

**Факторы повышения/стабилизации pH:**
| Фактор | Механизм | Эффективность |
|--------|----------|---------------|
| **Слюна** | Бикарбонатная буферность | Высокая (5-10 л/день) |
| **Структурная клетчатка** | Стимуляция жвачки и слюноотделения | Высокая |
| **Буферные добавки** | Нейтрализация кислот | Умеренная |
| **Абсорбция VFA** | Удаление кислот через сосочки | Высокая |
| **Прохождение содержимого** | Вынос кислот из рубца | Умеренная |

### Динамика pH в течение суток

```
     pH
    7.0 |                      ____
        |                     /    \
    6.5 |    ____            /      \
        |   /    \          /        \
    6.0 |__/      \________/          \____
        |
    5.5 |__________________________________
        +----+----+----+----+----+----+----+----> Время
        0    4    8   12   16   20   24
              ↑        ↑        ↑
           Кормление  Кормление  Кормление
```

| Фаза | Время | pH | Механизм |
|------|-------|-----|----------|
| **Базальный** | Перед кормлением | 6.2-6.8 | Баланс абсорбции/продукции |
| **После кормления** | 1-3 ч | 5.8-6.2 | Пик ферментации |
| **Восстановление** | 3-6 ч | 6.0-6.5 | Адаптация микробиома |
| **Ночной** | Ночь | 6.3-6.8 | Низкая ферментация |

### Патофизиология низкого pH

**При pH < 6.0:**
| Процесс | Изменение | Последствия |
|---------|-----------|-------------|
| **Целлюлолитические бактерии** | Подавление | Снижение переваримости NDF |
| **Протеолитические бактерии** | Активация | Избыток NH3, аминов |
| **Лактатобактерии** | Рост при резком снижении | Острый ацидоз |
| **Абсорбция VFA** | Увеличение | Компенсаторная |
| **Проницаемость стенки** | Повышение | Риск "утечки" |

## Классификация

### По уровню pH

| Категория | Диапазон pH | Характеристика |
|-----------|-------------|----------------|
| **Щелочной** | > 7.0 | Избыток буферов, редко |
| **Нормальный** | 6.0-7.0 | Оптимальная ферментация |
| **Субоптимальный** | 5.8-6.0 | Легкий стресс |
| **Кислый (SARA)** | 5.5-5.8 | Хронический стресс |
| **Острый ацидоз** | 5.0-5.5 | Тяжелое состояние |
| **Критический** | < 5.0 | Опасность для жизни |

### По динамике

| Тип | Характеристика | Пример |
|-----|----------------|--------|
| **Стабильный нормальный** | pH 6.0-6.8, минимальные колебания | Сбалансированный рацион |
| **Волнообразный** | Колебания 5.8-6.5 | Редкое кормление |
| **Хронически низкий** | Постоянно < 6.0 | Высококонцентратный рацион |
| **Острый провал** | Резкое падение < 5.5 | Переедание концентратов |

| **Ламинит** | $500-1000/случай | Хронический низкий pH |
| **Печеночные абсцессы** | $400-800/случай | Последствие ацидоза |

## Лечение/Применение/Управление

### Управление pH через рацион

**Целевые показатели (Zebeli 2008):**
| Параметр | Оптимум | Диапазон |
|----------|---------|----------|
| **Среднесуточный pH** | > 6.2 | > 6.16 |
| **Время pH < 5.8** | < 3 ч/день | < 5.24 ч/день |
| **Минимальный пик** | > 5.8 | > 5.5 |

**Стратегии стабилизации pH:**

| Стратегия | Механизм | Дозировка/Условие | Ожидаемый эффект |
|-----------|----------|-------------------|------------------|
| **Увеличение NDF** | Буферизация, слюна | + 3-5% NDF | + 0.2-0.3 pH |
| **Структурная клетчатка** | Жвачка, слюноотделение | 3-5 см длина | + 0.1-0.2 pH |
| **Бикарбонат натрия** | Буфер | 150-250 г/день | + 0.1-0.2 pH |
| **Карбонат кальция** | Буфер | 150-200 г/день | + 0.05-0.1 pH |
| **Оксид магния** | Буфер | 50-100 г/день | + 0.1-0.15 pH |
| **Йеасты** | Конкуренция за кислоты | 10-30 г/день | Стабилизация |
| **Частое кормление** | Снижение пиков | 3-4 раза/день | Снижение амплитуды |

### Алгоритм коррекции низкого pH

```
Оценка pH рубца
    ↓
├── Средний pH > 6.0 (Норма)
│   └── Поддерживать текущий рацион
│
├── Средний pH 5.8-6.0 (Граница)
│   ├── Увеличить peNDF на 2-3%
│   ├── Добавить буфер 100-150 г/день
│   └── Мониторинг через 1-2 недели
│
├── Средний pH 5.5-5.8 (SARA)
│   ├── Увеличить NDF на 3-5%
│   ├── Буфер 200-300 г/день
│   ├── Снизить крахмал на 5-10%
│   └── Контроль через 3-5 дней
│
└── Средний pH < 5.5 (Острый ацидоз)
    ├── Немедленная коррекция рациона
    ├── Интенсивная буферизация
    ├── Ветеринарное вмешательство
    └── Ежедневный мониторинг
```
