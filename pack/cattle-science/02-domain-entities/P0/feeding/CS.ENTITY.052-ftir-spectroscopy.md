# CS.ENTITY.052: FTIR-спектроскопия (FTIR spectroscopy)

> FTIR-спектроскопия (Fourier-transform infrared spectroscopy) — аналитический метод, использующий инфракрасное излучение для определения химического состава молока (жир, белок, лактоза, соматические клетки), а также для предсказания послеродовых заболеваний и других показателей здоровья коровы на основе анализа молочных спектров с применением методов машинного обучения.

---

## Определение

**FTIR-спектроскопия (Fourier-transform infrared spectroscopy)** — метод анализа, основанный на измерении поглощения инфракрасного излучения молекулами вещества, позволяющий определять химический состав и структуру соединений.

**Принцип работы:**

```
ИК-излучение
    ↓
Прохождение через образец (молоко)
    ↓
Поглощение определённых длин волн
    ↓
Интерферограмма
    ↓
Преобразование Фурье
    ↓
Спектр поглощения
    ↓
Анализ (количественный/качественный)
```

**Диагностические критерии:**

| Параметр | Точность | Применение |
|----------|----------|------------|
| **Жир молока** | ± 0.05% | Стандартный анализ |
| **Белок молока** | ± 0.05% | Стандартный анализ |
| **Лактоза** | ± 0.05% | Стандартный анализ |
| **Соматические клетки** | ± 10% | Скрининг мастита |
| **Мочевина** | ± 5% | Оценка белкового питания |

**Определение случая (case definition):**
- **Нормальный спектр:** Соответствует эталонным значениям
- **Аномальный спектр:** Отклонения в определённых регионах
- **Предиктивный:** Изменения, коррелирующие с заболеваниями

## Этиология/Патогенез

### Физика метода

**Инфракрасные регионы:**

| Регион | Длина волны | Связанные компоненты |
|--------|-------------|----------------------|
| **Средний ИК (MIR)** | 2.5-25 мкм | Основной для молока |
| **Ближний ИК (NIR)** | 0.8-2.5 мкм | Вспомогательный |
| **Дальний ИК (FIR)** | 25-1000 мкм | Редко используется |

**Характеристические поглощения:**

| Компонент | Волновое число (см⁻¹) | Функциональная группа |
|-----------|----------------------|----------------------|
| **Жир** | 1745 | Эфирные связи |
| **Белок** | 1640, 1530 | Амид I, Амид II |
| **Лактоза** | 1040 | C-O связи |
| **Вода** | 3300, 1640 | O-H связи |

### Применение в диагностике

**Анализ состава молока:**

```
Молочный спектр
    ↓
Выделение характеристических пиков
    ↓
Калибровка по эталонным методам
    ↓
Расчёт концентраций компонентов
    ↓
Результат (жир, белок, лактоза)
```

**Предсказание заболеваний (Lin 2025):**

```
Молочный спектр
    ↓
Машинное обучение (PLS-DA, LSTM)
    ↓
Выявление паттернов
    ↓
Предсказание риска заболевания
    ↓
Раннее предупреждение
```
