# CS.ENTITY.051: Тест на беременность (Pregnancy test)

> Тест на беременность (pregnancy test) — диагностический метод определения наличия беременности у молочных коров, включающий различные технологии: ультразвуковое исследование (УЗИ), определение беременностно-ассоциированных гликопротеинов (PAG) в крови или молоке, прогестерона и других маркеров, являющийся критически важным инструментом управления репродукцией стада.

---

## Определение

**Тест на беременность (pregnancy test)** — метод диагностики беременности, основанный на обнаружении специфических маркеров, продуцируемых плодом или плацентой, или визуализации плода.

**Методы диагностики:**

| Метод | Принцип | Время применения | Точность |
|-------|---------|------------------|----------|
| **УЗИ** | Визуализация плода | 28-35 дней | 95-98% |
| **PAG кровь** | Беременностные белки | 28-30 дней | 95-99% |
| **PAG молоко** | Беременностные белки | 35-40 дней | 90-95% |
| **Прогестерон** | Гормон | 18-24 дня | 80-85% |
| **Эстрагон** | Эстрогены | 18-24 дня | 70-80% |

**Диагностические критерии:**

| Метод | Положительный результат | Отрицательный результат |
|-------|------------------------|------------------------|
| **УЗИ** | Визуализация плода, сердцебиение | Отсутствие плода |
| **PAG** | > 0.5-1.0 нг/мл (зависит от теста) | < порогового значения |
| **Прогестерон** | > 2 нг/мл (день 21) | < 2 нг/мл |

**Определение случая (case definition):**
- **Подтверждённая беременность:** Положительный результат двумя методами
- **Подозрение на беременность:** Положительный результат одним методом
- **Не подтверждённая:** Отрицательный результат

## Этиология/Патогенез

### Маркеры беременности

**PAG (Pregnancy-Associated Glycoproteins):**

```
Бинуклеарные гигантские клетки плаценты
    ↓
Секреция PAG
    ↓
Попадание в материнскую кровь (с 25-28 дня)
    ↓
Детекция в крови/молоке
    ↓
Диагностика беременности
```

**Характеристики PAG:**

| Параметр | Описание |
|----------|----------|
| **Источник** | Бинуклеарные гигантские клетки плаценты |
| **Начало секреции** | 25-28 день гестации |
| **Пик концентрации** | 60-100 день |
| **Период полураспада** | 7-10 дней |
| **После аборта** | Снижается в течение 1-2 недель |

### Динамика маркеров

| Маркер | Начало детекции | Пик | После аборта |
|--------|-----------------|-----|--------------|
| **PAG кровь** | 25-28 дней | 60-100 дней | 1-2 недели |
| **PAG молоко** | 35-40 дней | 60-100 дней | 1-2 недели |
| **Прогестерон** | 18-21 день | Весь цикл | 1-3 дня |

## Классификация

### По методу

| Метод | Преимущества | Недостатки | Применение |
|-------|--------------|------------|------------|
| **УЗИ** | Ранняя диагностика, визуализация | Требует оборудования, навыков | 28-35 дней |
| **PAG кровь** | Высокая точность, ранняя | Забор крови, лаборатория | 28-30 дней |
| **PAG молоко** | Неинвазивный, удобный | Позже, чем кровь | 35-40 дней |
| **Прогестерон** | Доступный | Низкая точность | 18-24 дня |

### По времени

| Время | Метод | Точность |

| **18-24 дня** | Прогестерон | 80-85% |
| **25-30 дней** | PAG кровь | 95-99% |
| **28-35 дней** | УЗИ | 95-98% |
| **35-40 дней** | PAG молоко | 90-95% |
| **60-90 дней** | Любой | > 98% |

## Клиническое проявление

### Применение

**Ранняя диагностика (28-35 дней):**
- Выявление небеременных коров
- Раннее повторное осеменение
- Сокращение дней открытой

**Поздняя диагностика (60-90 дней):**
- Подтверждение беременности
- Определение срока
- Планирование сухостоя

### Интерпретация результатов

| Результат | Действие |
|-----------|----------|
| **Положительный** | Подтверждение, планирование |
| **Отрицательный** | Повторное AI, анализ причин |
| **Сомнительный** | Повторный тест через 7-14 дней |
