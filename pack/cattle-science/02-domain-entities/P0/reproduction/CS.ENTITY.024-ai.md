# CS.ENTITY.024: Искусственное осеменение

> Репродуктивная биотехнология введения обработанной и концентрированной спермы в половые пути коровы с помощью специального катетера в оптимальное для оплодотворения время, являющаяся основным методом воспроизводства в современном молочном скотоводстве.

---

## Определение

**Искусственное осеменение (ИО)** — метод репродукции, при котором сперма, собранная от генетическо ценного быка-производителя, обрабатывается, разбавляется, консервируется и вводится в половые пути коровы в период эструса с целью достижения оплодотворения.

### Исторический контекст

| Период | Развитие | Значение |
|--------|----------|----------|
| **1780** | Первые эксперименты (Spallanzani) | Доказана возможность ИО |
| **1930-е** | Разработка разбавителей спермы | Массовое применение |
| **1949** | Криоконсервация (Polge et al.) | Хранение спермы в жидком азоте |
| **1960-е** | Сексированная сперма | Контроль пола потомства |
| **1990-е** | Технологии TAI | Синхронизация и фиксированное время |
| **2000-е** | Геномная оценка быков | Ускоренный генетический прогресс |

### Типы ИО

| Тип | Описание | Применение | Эффективность |
|-----|----------|------------|---------------|
| **По охоте** | Осеменение после визуального/автоматического обнаружения охоты | Традиционное | Зависит от качества ОО |
| **TAI** | Синхронизация + осеменение в фиксированное время | Системные программы | 40-50% стельность |
| **Комбинированное** | TAI + обнаружение охоты | Оптимальное | 45-55% стельность |

### Процесс оплодотворения

| Этап | Время | Описание | Критические факторы |
|------|-------|----------|---------------------|
| **Транспорт спермы** | 0-6 часов | Пассаж через шейку матки | Мотильность, цервикальная слизь |
| **Капацитация** | 2-8 часов | Модификация сперматозоидов | Время, условия в матке |
| **Встреча с ооцитом** | 6-24 часов | В ампуле маточной трубы | Синхронизация овуляции |
| **Оплодотворение** | 12-24 часов | Проникновение в ооцит | Качество гамет |
| **Блок полиспермии** | Мгновенно | Модификация зоны пеллюцида | Нормальная реакция ооцита |

### Оптимальное время осеменения

| Признак охоты | Время до овуляции | Рекомендация по ИО |
|---------------|-------------------|-------------------|
| **Начало охоты** (первые признаки) | 18-24 часа | Слишком рано |
| **Яркая охота** (стоит при осеменении) | 12-18 часов | Раннее ИО |
| **Пик охоты** (активное принятие) | 6-12 часов | **Оптимальное время** |
| **Конец охоты** (слабое принятие) | 0-6 часов | Позднее ИО |
| **После охоты** | Овуляция произошла | Слишком поздно |

**Правило AM-PM:** Корова, обнаруженная в охоте утром → осеменение вечером; вечером → осеменение следующим утром.

| Категория | Показание | Преимущество ИО |
|-----------|-----------|-----------------|

| **Здоровье** | Отсутствие клинических заболеваний | Полное здоровье |
| **Репродуктивный статус** | Нормальный послеродовой период | Полное восстановление |
| **Метаболический статус** | BHB < 1.2 ммоль/л | BHB < 1.0 ммоль/л |

## Протоколы/Техника

### Подготовка спермы

| Шаг | Требования | Контроль качества |
|-----|------------|-------------------|
| **Извлечение из хранилища** | Жидкий азот (-196°C) | Время < 10 секунд |
| **Размораживание** | Вода 35-37°C, 45 секунд | Температура контроль |
| **Осмотр** | Микроскопия (200×) | Мотильность > 30% |
| **Загрузка** | Стерильный шприц | Герметичность |
| **Транспорт** | Температура 35-37°C | Изотермический контейнер |

### Техника осеменения

```
ПОШАГОВАЯ ТЕХНИКА ИО

1. ПОДГОТОВКА
   ├── Ограничение коровы
   ├── Гигиена промежности (мыло + вода)
   ├── Сушка чистым полотенцем
   └── Надевание перчатки

2. ОЦЕНКА ШЕЙКИ МАТКИ
   ├── Ректальное исследование
   ├── Оценка консистенции (мягкая = охота)
   ├── Оценка открытия (1-5 пальцев)
   └── Проверка слизи (прозрачная, вязкая)

3. ВВЕДЕНИЕ КАТЕТЕРА
   ├── Направление под углом 30° вверх
   ├── Прохождение через шейку матки
   ├── Достижение маточного тела
   └── Фиксация катетера

4. ВВЕДЕНИЕ СПЕРМЫ
   ├── Медленное введение (5-10 сек)
   ├── Минимальное давление
   ├── Контроль заполнения матки
   └── Проверка отсутствия рефлюкса

5. ЗАВЕРШЕНИЕ
   ├── Медленное извлечение катетера
   ├── Проверка целостности оборудования
   ├── Документирование
   └── Освобождение коровы
```

### Протоколы синхронизации (TAI)

| Протокол | Схема | Стельность | Применение |
|----------|-------|-----------|------------|
| **Ovsynch** | GnRH → 7 дней → PGF2α → 56 ч → GnRH → 16 ч → TAI | 35-45% | Стандартный |
| **Presynch-Ovsynch** | 2×PGF2α (14 дн) → Ovsynch | 40-50% | Рекомендуемый |
| **Double-Ovsynch** | Ovsynch → 7 дней → Ovsynch | 45-55% | Нулевые коровы |
| **Cosynch** | GnRH + TAI одновременно (72 ч после PGF2α) | 35-40% | Ограниченное применение |
| **5-day CoSynch** | Сокращённый протокол | 45-50% | Экономия времени |

**Схема Ovsynch:**
```
День 0: GnRH (овуляция фолликула + новая волна)
   │
   ├── 7 дней
   │
День 7: PGF2α (лютеолиз)
   │
   ├── 56 часов
   │
День 9: GnRH (овуляция)
   │
   ├── 16 часов
   │
   ↓
TAI (фиксированное время)
```

## Осложнения/Режимы отказа

### Технические ошибки

| Ошибка | Частота | Последствия | Предотвращение |
|--------|---------|-------------|----------------|
| **Неправильное размещение** | 10-20% | Снижение концепции на 20-30% | Обучение, сертификация |
| **Рефлюкс спермы** | 5-15% | Потеря спермы | Правильная техника |
| **Контаминация** | 2-5% | Инфекция, эндометрит | Строгая гигиена |
| **Травма шейки матки** | 1-3% | Боль, спазм, кровотечение | Осторожность, опыт |
| **Неправильное время** | 15-25% | Несинхронизация с овуляцией | Точное обнаружение охоты |

### Биологические режимы отказа

| Режим | Причина | Диагностика | Коррекция |
|-------|---------|-------------|-----------|
| **Ранний эмбриональный смерть** | 20-30% случаев | Повторная охота 18-24 дня | Улучшение метаболизма |
| **Неоплодотворение** | Некачественные гаметы | Постоянная возвратность | Проверка спермы, здоровье |
| **Анэмбриония** | Хромосомные аномалии | УЗИ 30-35 дней | Генетика |
| **Лютеальная недостаточность** | Низкий прогестерон | Профиль P4 | Дополнительный P4 |
