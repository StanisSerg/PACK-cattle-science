# CS.ENTITY.047: Дни открытой (Days open)

> Дни открытой (days open) — период времени от отёла до успешного осеменения, приводящего к беременности, являющийся ключевым показателем репродуктивной эффективности молочного стада, отражающим комплексное влияние множества факторов: здоровья, питания, генетики, менеджмента и экономических потерь.

---

## Определение

**Дни открытой (days open)** — количество дней от момента отёла до момента успешного осеменения, подтверждённого беременностью.

**Формула расчёта:**
```
Days open = Дата успешного осеменения - Дата отёла
```

**Диагностические критерии:**

| Показатель | Целевое значение | Приемлемое | Проблемное |
|------------|------------------|------------|------------|
| **Средние дни открытой** | 90-110 | 110-130 | > 130 |
| **Медиана** | 85-100 | 100-120 | > 120 |
| **% коров > 150 дней** | < 10% | 10-15% | > 15% |
| **% коров > 200 дней** | < 5% | 5-10% | > 10% |

**Определение случая (case definition):**
- **Оптимальные дни открытой:** 90-110 дней
- **Умеренно увеличенные:** 110-130 дней
- **Критически увеличенные:** > 130 дней

## Этиология/Патогенез

### Факторы, влияющие на дни открытой

**Факторы, увеличивающие дни открытой:**

| Фактор | Механизм | Влияние |
|--------|----------|---------|
| **Метрит** | Задержка цикличности | + 20-40 дней |
| **Кетоз** | Нарушение овариальной функции | + 15-30 дней |
| **Мастит** | Иммуносупрессия, стресс | + 10-20 дней |
| **Низкий BCS** | Нарушение цикличности | + 15-25 дней |
| **Тепловой стресс** | Подавление овуляции | + 20-40 дней |
| **Плохая детекция охоты** | Пропущенные охоты | + 20-30 дней |
| **Низкая концепция** | Повторные осеменения | + 30-60 дней |

**Факторы, сокращающие дни открытой:**

| Фактор | Механизм | Влияние |
|--------|----------|---------|
| **TAI (синхронизация)** | Контроль цикла | - 10-20 дней |
| **Хорошее здоровье** | Нормальная цикличность | Оптимальные сроки |
| **Оптимальный BCS** | Нормальная репродукция | Оптимальные сроки |
| **Высокая концепция** | Меньше повторов | - 10-20 дней |

### Взаимосвязь с другими показателями


**Математическая связь:**
```
Интервал между отёлами = Дни открытой + Длительность гестации + Сухостой
                        = Дни открытой + 280 дней + 60 дней
                        = Дни открытой + 340 дней
```

## Клиническое проявление

### Признаки проблем

| Признак | Механизм | Частота |
|---------|----------|---------|
| **Дни открытой > 130** | Проблемы с фертильностью | 20-30% стад |
| **Множественные AI** | Низкая концепция | 30-40% коров |
| **Аноэструс** | Отсутствие циклов | 10-15% коров |

### Клиническое значение

**Индикатор проблем:**
- Отражает общее состояние репродукции
- Сигнализирует о проблемах со здоровьем
- Показатель эффективности менеджмента

**Прогностическое значение:**
- Предиктор экономических потерь
- Индикатор риска выбытия
- Оценка генетического прогресса
