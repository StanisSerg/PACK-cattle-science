# CS.ENTITY.026: Обнаружение охоты

> Система методов идентификации коров в периоде эструса (охоты) по поведенческим, физиологическим и технологическим признакам, являющаяся критически важным элементом репродуктивного менеджмента и определяющая эффективность программ искусственного осеменения.

---

## Определение

**Обнаружение охоты (ОО)** — процесс распознавания физиологического состояния эструса у коровы, характеризующегося повышенной сексуальной активностью и готовностью к спариванию, с целью определения оптимального времени для искусственного осеменения.

### Биологическая основа эструса

| Параметр | Значение | Значимость для ОО |
|----------|----------|-------------------|
| **Длительность эструса** | 12-18 часов (среднее 15 ч) | Окно для обнаружения |
| **Частота приёма** | 0.5-1.5 раз/час в пик | Основной признак |
| **Время приёма** | 1-5 секунд | Кратковременность |
| **Период суток** | 70% случаев: 18:00-06:00 | Ночная активность |
| **Интервал между циклами** | 18-24 дня (среднее 21 день) | Прогнозирование |

### Виды обнаружения охоты

| Тип | Метод | Точность | Трудозатраты |
|-----|-------|----------|--------------|
| **Визуальное** | Наблюдение за поведением | 40-60% | Высокие |
| **Хвостовое мелирование** | Окраска хвоста | 50-70% | Средние |
| **Активность** | Шагомеры, акселерометры | 70-85% | Низкие |
| **Комбинированное** | Несколько методов | 80-95% | Средние |
| **Автоматизированное** | Сенсоры + алгоритмы | 85-95% | Минимальные |

## Показания/Применение

### Критерии проведения ОО

| Категория | Критерий | Действие |
|-----------|----------|----------|
| **Возраст** | > 13-15 месяцев (телки) | Начало ОО |
| **Вес** | > 55-60% взрослой массы | Начало ОО |
| **После отёла** | > 40-50 DIM | Возобновление ОО |
| **Интервал** | > 18 дней с последней охоты | Проверка цикличности |
| **После ИО** | 18-24 дня | Проверка возвратности |

### Факторы, затрудняющие ОО

| Фактор | Влияние | Механизм | Решение |
|--------|---------|----------|---------|
| **Слабая выраженность** | 20-30% коров | Низкий эстрадиол | Усиленное наблюдение |
| **Ночная активность** | 70% случаев | Циркадные ритмы | Автоматизация |
| **Короткий эструс** | 10-15% коров | < 8 часов | Частое наблюдение |
| **Тихая охота** | 10-20% коров | Отсутствие признаков | Технологии ОО |
| **Стадный эффект** | Групповая синхронизация | Феромоны | Использование наездниц |
| **Состояние подстилки** | ↓ Выраженность | Дискомфорт | Улучшение условий |
| **Плотность посадки** | ↓ Приём | Недостаток пространства | Оптимизация |

## Протоколы/Техника

### Методы обнаружения охоты

#### 1. Визуальное наблюдение

| Параметр | Рекомендация | Примечание |
|----------|--------------|------------|
| **Частота** | Минимум 3 раза/день | Идеально: каждые 4-6 часов |
| **Длительность** | 20-30 минут | Достаточно для оценки |
| **Время** | Утро, день, вечер, ночь | Особенно вечер/ночь |
| **Условия** | Хорошее освещение | Искусственное при необходимости |
| **Персонал** | Обученные наблюдатели | Квалификация критична |

**Чек-лист визуального ОО:**
```
□ Приём спаривания (стоит при наезде)
□ Стояние наездницей (наезжает на других)
□ Мукозные выделения (прозрачная, вязкая слизь)
□ Беспокойство (блуждание, обнюхивание)
□ Снижение аппетита (отвлечение от корма)
□ Повышенная vocalization (мычание, рёв)
□ Потёртости на шее/крестце (от наездниц)
□ Припухлость вульвы (отёчность, покраснение)
```

#### 2. Хвостовое мелирование (Tail painting)

| Параметр | Рекомендация |
|----------|--------------|
| **Расположение** | Основание хвоста, крестец |
| **Краска** | Специальная (термочувствительная или обычная) |
| **Обновление** | Каждые 7-14 дней или при выцветании |
| **Интерпретация** | Размазанная краска = активность |

**Оценка интенсивности:**
| Степень | Описание | Действие |
|---------|----------|----------|
| **Не тронута** | Краска целая | Нет активности |
| **Слегка размазана** | Частичное стирание | Ранняя/поздняя охота |
| **Сильно размазана** | Большая часть стёрта | Пик охоты |
| **Полностью стёрта** | Краска отсутствует | Охота подтверждена |

#### 3. Электронные системы

| Тип сенсора | Принцип | Точность | Стоимость |
|-------------|---------|----------|-----------|
| **Активность (шагомер)** | Подсчёт шагов | 75-85% | $50-100/корова |
| **Акселерометр** | Движение во всех плоскостях | 80-90% | $100-150/корова |
| **Жвачка** | Время жвачки | 70-80% | $80-120/корова |
| **Температура** | Изменение температуры | 60-70% | $30-50/корова |
| **Комбинированные** | Несколько параметров | 85-95% | $150-250/корова |

**Алгоритмы обнаружения:**
```
БАЗОВЫЙ АЛГОРИТМ АКТИВНОСТИ

Активность сегодня
      │
      ↓
Сравнение с базовой (средние 7-10 дней)
      │
      ↓
┌─────┴─────┐
↓           ↓
> 150%    < 150%
  активности  активности
    │           │
    ↓           ↓
 ПРЕДУПРЕЖДЕНИЕ  Норма
    │
    ↓
Подтверждение дополнительными признаками
    │
    ↓
ОПОВЕЩЕНИЕ ОБ ОХОТЕ
```

#### 4. Лабораторные методы

| Метод | Показатель | Порог | Применение |
|-------|------------|-------|------------|
| **Прогестерон в молоке** | P4 | < 2 нг/мл | Подтверждение охоты |
| **Прогестерон в крови** | P4 | < 1 нг/мл | Точная диагностика |
| **Эстрадиол** | E2 | > 5 пг/мл | Раннее обнаружение |
| **Тест-полоски** | P4 | Отрицательный | Скрининг |

### Протокол ОО для разных систем

#### Малое стадо (< 100 голов)

| Время | Действие | Персонал |

|-----------|--------------|

#### Крупное стадо (> 500 голов)

| Компонент | Рекомендация |
|-----------|--------------|

| **Поддержка** | TAI для проблемных коров |
| **Интеграция** | Система управления стадом |

## Осложнения/Режимы отказа

### Ошибки обнаружения

| Тип ошибки | Причина | Частота | Последствия |
|------------|---------|---------|-------------|
| **Пропуск охоты** | Ночная активность, короткий эструс | 20-40% | Потеря цикла |
| **Ложноположительная** | Метеэструс, циста, стресс | 5-15% | Премature ИО |
| **Ложноотрицательная** | Тихая охота, слабые признаки | 10-20% | Пропуск цикла |
| **Неправильное время** | Опоздание/раннее обнаружение | 15-25% | Снижение концепции |

### Биологические режимы отказа

| Режим | Механизм | Диагностика | Коррекция |
|-------|----------|-------------|-----------|
| **Аноэструс** | Отсутствие циклов | Нет охот > 50 дней | TAI, CIDR |
| **Субэструс** | Слабые признаки | Неполный приём | Усиленное наблюдение |
| **Циста яичника** | Нарушение цикла | Постоянная/прерывистая охота | Лечение кисты |
| **Сilent heat** | Отсутствие поведения | Только лабораторно | Прогестероновые тесты |
| **Яичниковая недостаточность** | Неответ на гормоны | Отсутствие циклов | Гормональная терапия |
