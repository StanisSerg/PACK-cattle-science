# CS.ENTITY.197: Микробный протеин (Microbial Protein)

> Микробный протеин — высококачественный источник протеина для молочных коров, синтезируемый микробиомом рубца из руминально-деградируемого протеина (RDP) и непротеинового азота, обеспечивающий 50-70% метаболизируемого протеина и обладающий превосходным аминокислотным профилем, близким к потребностям молочной коровы.

---

## Определение

**Микробный протеин (microbial protein, MCP, microbial crude protein, MCP)** — протеин, синтезированный микроорганизмами рубца (бактериями, протозоями, грибами) из источников азота (RDP, NPN) и углерода (ферментируемые углеводы).

**Характеристики MCP:**
| Параметр | Значение | Преимущество |
|----------|----------|--------------|
| **Содержание в МП** | 50-70% | Основной источник |
| **Переваримость в ТК** | 70-80% | Высокая |
| **Аминокислотный профиль** | Сбалансированный | Близок к потребностям |
| **Лизин в протеине** | 7.5-8.0% | Выше, чем в большинстве RUP |
| **Метионин в протеине** | 2.5-2.8% | Адекватен |

**Диагностические критерии синтеза:**

| Параметр | Норма | Оптимум | Патология |
|----------|-------|---------|-----------|
| **Синтез MCP** | 1.5-2.5 кг/день | 2.0-2.8 кг/день | < 1.2 кг/день |
| **Эффективность синтеза** | 0.10-0.15 г MCP/г RDP | 0.13-0.18 | < 0.08 |
| **Доля MCP в MP** | 50-70% | 55-65% | < 40% или > 80% |
| **NH3 в рубце** | 5-15 мг/дл | 8-12 мг/дл | < 5 или > 20 мг/дл |

## Синтез микробного протеина

### Биохимия синтеза

```
Азотистые источники:
- RDP (аминокислоты, пептиды)
- NPN (мочевина, аммиак)
    ↓
Деградация в рубце
    ↓
NH3 (аммиак) + VFA (энергия)
    ↓
Ассимиляция микробами
    ↓
Синтез аминокислот → Синтез белка
    ↓
МИКРОБНЫЙ ПРОТЕИН
```

**Типы микроорганизмов:**
| Группа | Доля в MCP | Особенности |
|--------|------------|-------------|
| **Бактерии** | 70-80% | Основной синтезатор |
| **Протозои** | 15-25% | Утилизация бактерий |
| **Грибы** | 5-10% | Ферментация клетчатки |

### Факторы, влияющие на синтез

**Энергетические факторы:**
| Фактор | Оптимум | Влияние на синтез |
|--------|---------|-------------------|
| **Ферментируемые углеводы** | Адекватно | Лимитирующий фактор |
| **Скорость ферментации** | Умеренная | Быстрая — недостаточное время |
| **Источники углеводов** | Смешанные | Крахмал + клетчатка |

**Азотистые факторы:**
| Фактор | Оптимум | Влияние на синтез |
|--------|---------|-------------------|
| **RDP** | 10-12% СВ | Субстрат для синтеза |
| **Соотношение RDP:энергия** | 20-25 г RDP/МДж | Баланс |
| **NPN (мочевина)** | < 1% СВ | Дополнительный источник NH3 |

## Расчёт и прогнозирование

### Уравнения синтеза MCP

**NRC (2001):**
```
MCP (г/день) = 130 × Доступная энергия (МДж NEL)

Или:
MCP = RDP × Эффективность
Где эффективность = 0.85 (доступность микробов)
```

**CNCPS:**
```
MCP = (FCHO × Kd_CHO + FPROT × Kd_PROT) × 0.13

Где:
- FCHO = ферментируемые углеводы
- Kd = скорость ферментации
- FPROT = ферментируемый протеин
- 0.13 = эффективность конверсии (г MCP/МДж)
```

### Пример расчёта

| Параметр | Значение | Расчёт |
|----------|----------|--------|
| **RDP** | 12% СВ | 2.8 кг RDP/день (при DMI 24 кг) |
| **Эффективность** | 0.85 | Доступность для синтеза |
| **Эффективность конверсии** | 0.13 | г MCP/г RDP |
| **MCP синтез** | — | 2.8 × 0.85 × 0.13 = 2.4 кг/день |

## Практическое применение

### Оптимизация синтеза MCP

| Стратегия | Механизм | Результат |
|-----------|----------|-----------|
| **Адекватный RDP** | 10-12% СВ | Достаточный субстрат |
| **Смешанные углеводы** | Крахмал + клетчатка | Оптимальная энергия |
| **pH рубца 6.0-6.8** | Стабильная среда | Максимальная активность |
| **Время прохождения** | Нормальное | Достаточное время синтеза |
