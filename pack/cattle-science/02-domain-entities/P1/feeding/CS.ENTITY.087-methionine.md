# CS.ENTITY.087: Метионин (Methionine)

> Метионин (methionine) — незаменимая серосодержащая аминокислота, являющаяся первой лимитирующей аминокислотой в рационах молочных коров, ключевым донором метильных групп (через S-аденозилметионин — SAM) для синтеза холина, креатина, нуклеотидов и метилирования ДНК, критически важная для синтеза липопротеинов (VLDL), детоксикации и функции печени.

---

## Определение

**Метионин (methionine)** — незаменимая аминокислота с серосодержащей метильной группой, формула C₅H₁₁NO₂S.

**Структура:**

```
    CH₃ — S — CH₂ — CH₂ — CH — COO⁻
                            |
                           NH₃⁺
```
