# CS.ENTITY.147: Искусственный интеллект (Artificial intelligence)

> Искусственный интеллект (artificial intelligence, AI) в молочном скотоводстве — применение алгоритмов машинного обучения (machine learning — случайный лес, градиентный бустинг, SVM), глубокого обучения (deep learning — CNN для компьютерного зрения, RNN/LSTM для временных рядов), предиктивной аналитики и обработки естественного языка для анализа сложных данных (сенсоры, изображения, тексты), выявления скрытых паттернов, прогнозирования событий (болезни, течка, отел), автоматизации принятия решений и оптимизации процессов, позволяющее превратить большие объёмы данных precision dairy farming в actionable insights с минимальным человеческим вмешательством.

---

## Определение

**Искусственный интеллект (artificial intelligence)** — алгоритмы для анализа данных и принятия решений.

**Типы AI:**

| Тип | Алгоритмы | Применение |
|-----|-----------|------------|
| **Machine Learning** | Random Forest, XGBoost, SVM | Классификация, регрессия |
| **Deep Learning** | CNN, RNN, LSTM | Изображения, временные ряды |
| **NLP** | Transformers, BERT | Тексты, документация |
| **Computer Vision** | CNN, YOLO | Детекция объектов |
