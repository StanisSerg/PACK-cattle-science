# CS.ENTITY.130: Коэффициент замены (Replacement rate)

> Коэффициент замены (replacement rate, RR) — метрика, отражающая процент коров, введённых в стадо в качестве замены выбывших животных за год, с типичными значениями 25-40%, совпадающими с коэффициентом оборачиваемости стада (HTR) при стабильном поголовье, являющаяся ключевым показателем для планирования воспроизводства стада, оценки стоимости замены и управления структурой стада по возрасту, при этом высокий RR указывает на необходимость увеличения воспроизводства или покупки ремонтных телок.

---

## Определение

**Коэффициент замены (replacement rate)** — процент введённых в стадо ремонтных телок за год.

**Формула:**

```
RR (%) = (Количество введённых ремонтных телок / Среднегодовое поголовье) × 100
```

**При стабильном стаде:**

```
RR ≈ HTR (herd turnover rate)
```

**Нормальные значения:**

| Показатель | Диапазон | Интерпретация |
|------------|----------|---------------|
| **Низкий RR** | < 25% | Медленная замена |
| **Оптимальный RR** | 25-35% | Баланс |
| **Высокий RR** | > 35% | Быстрая замена |

### Интерпретация

| RR | Интерпретация | Действие |
|-----|---------------|----------|
| **< 20%** | Низкая замена | Проверка HTR |
| **20-25%** | Ниже оптимума | Мониторинг |
| **25-35%** | Оптимум | Поддержание |
| **35-40%** | Выше оптимума | Анализ |
| **> 40%** | Высокая замена | Планирование воспроизводства |
