# CS.ENTITY.090: Карнитин (Carnitine)

> Карнитин (carnitine, L-карнитин) — условно-заменимое питательное вещество, являющееся essential фактором для транспорта длинноцепочечных жирных кислот из цитоплазмы в митохондрии (через систему карнитин-ацилкарнитинового транспорта), критически важное для β-окисления жирных кислот и энергетического метаболизма, особенно в условиях негативного энергетического баланса у молочных коров.

---

## Определение

**Карнитин (carnitine)** — водорастворимое соединение, производное аминокислоты (лизин + метионин), формула C₇H₁₅NO₃.

**Структура:**

```
    COO⁻
     |
HO — C — CH₂ — CH₂ — N⁺(CH₃)₃
     |
     H
```

**Система карнитинового транспорта:**

```
Цитоплазма                          Митохондрии
    |                                   |
Жирная кислота + КоА                  Ацил-КоА
    |                                   |
    ↓                                   ↑
Ацил-КоА ←—— CPT-I ——→ Ацилкарнитин ←—— CPT-II
    |         (внешняя)   |              (внутренняя)
    |                     |               |
    |                     ↓               |
    |              КАРНИТИН ←—————————————|
    |                     (транспорт)
    ↓
β-окисление → Ацетил-КоА → Энергия / Кетогенез
```

**Диагностические критерии:**

| Параметр | Норма | Нарушение | Клиническое значение |
|----------|-------|-----------|---------------------|
| **Свободный карнитин** | 30-60 мкмоль/l | < 20 | Дефицит |
| **Ацилкарнитин/карнитин** | 0.3-0.6 | > 1.0 | Перегрузка β-окисления |
| **Суммарный карнитин** | 50-100 мкмоль/l | < 30 | Дефицит |
