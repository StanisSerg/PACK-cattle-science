# CS.ENTITY.150: Облачные вычисления (Cloud computing)

> Облачные вычисления (cloud computing) в молочном скотоводстве — модель предоставления вычислительных ресурсов (хранение, обработка, аналитика, приложения) через интернет по требованию, позволяющая фермерам избежать капитальных затрат на IT-инфраструктуру, обеспечивающая масштабируемость (elastic scaling по мере роста данных), доступность из любой точки (мобильные приложения), автоматическое обновление программного обеспечения, интеграцию с сервисами аналитики (Big Data, ML) и безопасное резервное копирование данных, являющаяся критической инфраструктурой для precision dairy farming и цифровой трансформации отрасли.

---

## Определение

**Облачные вычисления (cloud computing)** — предоставление IT-ресурсов через интернет.

**Модели облака:**

| Модель | Описание | Применение |
|--------|----------|------------|
| **IaaS** | Инфраструктура как сервис | Виртуальные серверы |
| **PaaS** | Платформа как сервис | Разработка приложений |
| **SaaS** | Программное обеспечение как сервис | Готовые приложения |

**Преимущества:**

| Преимущество | Описание |
|--------------|----------|
| **Масштабируемость** | Рост ресурсов по мере необходимости |
| **Доступность** | Доступ из любой точки |
| **Экономичность** | Оплата по факту использования |
| **Безопасность** | Резервное копирование, защита |
