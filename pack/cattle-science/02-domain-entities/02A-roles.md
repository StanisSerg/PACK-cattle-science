# Роли в области Cattle Science

> Реестр ролей (функций), участвующих в управлении молочным стадом.

---

## Принцип ролей (FPF A.2 / Role-Centric)

**Различение:** Role ≠ Actor (Holder)
- **Role** = функция, описание "что делать"
- **Actor/Holder** = исполнитель роли (человек, ИИ, организация)

**Нотация:** `Holder#Role:Context@Window`

Примеры:
- `Dr.Smith#Veterinarian:FarmA@2026`
- `KimiCode#KnowledgeEngineer:PACK-cattle-science@Session`

## Роли по категориям

### 1. Ветеринарные роли

| Роль | Описание | Ключевые обязанности | Требуемая квалификация |
|------|----------|---------------------|----------------------|
| **Veterinarian (Ветеринар)** | Диагностика, лечение, профилактика | Репродуктивные программы, хирургия, терапия | DVM/VMD, лицензия |
| **Reproductive Specialist (Репродуктолог)** | Специализация на репродукции | Протоколы синхронизации, сложные случаи | DVM + сертификация ACT/ABVT |
| **Nutrition Veterinarian (Нутрициолог-ветеринар)** | Связь питания и здоровья | Метаболические программы, профилактика | DVM + специализация |
| **Herd Health Consultant (Консультант по здоровью стада)** | Комплексное управление здоровьем | Программы профилактики, аудиты | DVM + опыт >5 лет |

### 2. Зоотехнические роли

| Роль | Описание | Ключевые обязанности | Требуемая квалификация |
|------|----------|---------------------|----------------------|
| **Herd Manager (Заведующий стадом)** | Общее управление стадом | Планирование, контроль показателей, персонал | Опыт >3 лет, образование (желательно) |
| **Breeding Manager (Заведующий воспроизводством)** | Управление репродукцией | Протоколы, AI, диагностика, записи | Опыт AI, знание протоколов |

| **Feed Manager (Заведующий кормлением)** | Управление рационами | TMR, контроль кормов, экономика | Знание нутрициологии |

## Матрица ответственности (RACI)

### Репродуктивный менеджмент

| Задача | Ветеринар | Breeding Manager | AI Tech | Consultant |
|--------|-----------|------------------|---------|------------|
| Протоколы синхронизации | A | R | C | I |
| Выполнение AI | C | A | R | I |
| Диагностика стельности | A/R | C | — | I |
| Анализ 21-d PR | C | R | — | A |
| Обучение персонала | A | R | C | R |

**Легенда:** R = Responsible, A = Accountable, C = Consulted, I = Informed

### Кормление

| Задача | Нутрициолог | Feed Manager | Herd Manager | Vet |
|--------|-------------|--------------|--------------|-----|
| Формулировка рациона | R/A | C | I | C |
| Приготовление TMR | C | R/A | I | — |
| Мониторинг DMI | C | R | A | I |
| Метаболический контроль | C | I | I | A/R |

## Взаимодействие ролей

### Сценарий: Оптимизация репродуктивной программы

```
1. Herd Manager определяет цель (например, 21-d PR > 25%)
   ↓
2. Reproductive Consultant проводит аудит
   - Анализ текущих показателей
   - Сравнение с бенчмарками
   ↓
3. Veterinarian оценяет здоровье факторы
   - Метаболические проблемы
   - Инфекционные риски
   ↓
4. Breeding Manager предоставляет данные по протоколам
   - Compliance
   - Практические ограничения
   ↓
5. Consultant разрабатывает рекомендации
   ↓
6. Herd Manager принимает решение
   ↓
7. Breeding Manager + AI Technician внедряют
   ↓
8. Knowledge Engineer фиксирует в Pack
```

## Развитие ролей

### Траектории карьеры

```
AI Technician → Breeding Manager → Herd Manager
                      ↓
               Reproductive Consultant

Veterinarian → Reproductive Specialist → Herd Health Consultant

Nutritionist → Senior Nutritionist → Independent Consultant
```

### Обучение и сертификация

| Роль | Базовое образование | Продвинутая сертификация |
|------|---------------------|-------------------------|
| Veterinarian | DVM/VMD | ACT, ABVT, ACVIM |
| Nutritionist | MS/PhD Animal Science | ARPAS, ACAN |
| AI Technician | Курсы | NAAB, State licensed |
| Consultant | Опыт >5 лет | Нет единой, репутация |
