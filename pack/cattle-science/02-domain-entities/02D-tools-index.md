# Индекс инструментов Cattle Science

> Реестр инструментов (средств) для работы в области Cattle Science.

---

## Принцип: Method ≠ Tool (FPF A.3.2)

**Method** = способ действия (как делать)  
**Tool** = средство выполнения (чем делать)

Пример:
- Method = "Осеменение по протоколу Ovsynch"
- Tool = "AI gun, semen tank, gloves"
