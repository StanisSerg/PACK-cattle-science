## Назначение

**Не входит:** лечебная ветеринария, генетика, строительство ферм.

## SoTA-аннотация (протокол)

> **КРИТИЧЕСКОЕ ТРЕБОВАНИЕ: ВСЕ SoTA ДОЛЖНЫ БЫТЬ НА РУССКОМ ЯЗЫКЕ**
>
> - Контент: русский (перевод с английского)
> - Метаданные (YAML): английский (как в оригинале)
> - Цитаты: английский (оригинальный текст)
> - Теги: английский (для консистентности)
> - **Аббревиатуры и сокращения: сохранять как в оригинале (английские), НЕ переводить на русский**

### Правило аббревиатур

| Что делать                                                  | Пример                              |
| ----------------------------------------------------------- | ----------------------------------- |
| ✅ Сохранять оригинальную аббревиатуру                      | NEFA, SCFA, VLDL, TAG, BHBA         |
| ✅ Первое упоминание: полное название (англ) + аббревиатура | "non-esterified fatty acids (NEFA)" |
| ✅ Далее использовать только аббревиатуру                   | NEFA                                |
| ❌ НЕ переводить аббревиатуры на русский                    | ~~КЖР, ВНЖК, ОЛПНП~~                |

**Исключения:** Общепринятые русские аббревиатуры (СВС, КРС, ОГАС) можно использовать.

При добавлении новой статьи:

1. **Извлечь метаданные** — авторы, год, журнал, DOI
2. **Определить тип** — observational / experimental / simulation / review / survey
3. **Выделить Key Claims** — 3-7 утверждений с цитатами, evidence, confidence
4. **Оценить методологию** — strengths, limitations, validation status
5. **Указать Revision Criterion** — когда пересматривать
6. **Связать с существующими** — related SoTA entries
7. **Интегрировать в Pack** — methods, entities, work products
8. **Закоммитить** — индекс обновится автоматически (через git hooks)

   - Проверить: `bash scripts/verify-sota-index.sh`

## Ключевые различения (из 01B-distinctions.md)

| Различение                | Применение                                            |
| ------------------------- | ----------------------------------------------------- |
| Input vs. Output          | Здоровье (input) → Репродуктивные показатели (output) |
| Biological vs. Economic   | Биологический потенциал vs. экономический оптимум     |
| Model vs. Reality         | Симуляция требует валидации                           |
| Association vs. Causation | Наблюдательные исследования ≠ причинность             |
| Strategy vs. Tactics      | Выбор программы vs. исполнение                        |
