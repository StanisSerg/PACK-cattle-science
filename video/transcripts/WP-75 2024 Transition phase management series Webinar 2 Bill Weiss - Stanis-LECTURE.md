# Лекция: Минералы и витамины для transition cows

**Источник:** Bill Weiss (Ohio State), WP-75 Webinar 2, 2024  
**Длительность:** 63 мин  
**Дата транскрибации:** 2026-05-24  
**Связанный SoTA:** Minerals & Vitamins Transition Cows (CS.SOTA.XXX)

---

## 1. Фундаментальные принципы transition period

### 1.1. Почему transition критичен

1. **Потребление корма падает** за неделю до отела → абсолютное поступление ВСЕХ нутриентов снижается.
2. **Меняется распределение потребностей:** сухостой → плод → молозиво → молоко.
3. **Резкий рост энергопотребления:** 13–15 → 30–40 Mcal NEₗ/сутки → окислительный стресс.

### 1.2. Ключевое различие: минералы vs витамины

| | Минералы | Витамины |
|---|---|---|
| **Регуляция абсорбции** | Да — корова может усвоить больше % при сниженном потреблении | Нет — меньше съела = меньше усвоила |
| **Практика при сниженном DMI** | AC адаптируется, но медленно | **Нужно увеличивать концентрацию** в рационе |

### 1.3. Правило №1

> Нельзя исправить плохое минерально-витаминное питание за 3 недели pre-fresh. Сухостойные коровы должны получать полноценный рацион с **самого начала сухостоя**.

Цель: поддерживать запасы (печень для минералов, лабильные запасы для витаминов), развивать плод, подготовить иммунную систему.

## 2. Классификация минералов по стратегии кормления

Weiss делит минералы на 3 группы относительно NASEM 2021:

### Группа А: «Близко к NASEM» (избыток вреден)

| Минерал | NASEM | Почему не добавлять | Риск избытка |
|---------|-------|---------------------|--------------|
| **P** | ~0.2% | Ингибирует активацию 1,25(OH)₂D | Нарушение Ca-метаболизма |
| **S** | ~0.2% | Связывает другие минералы, снижает потребление | Cu, Mn, Zn, Se антагонизм |
| **I** | по NASEM | Токсичен для рубцовой микрофлоры | Угнетение микробиомы |
| **Cu** | по NASEM | Риск токсичности, накопление в печени | Печёночный damage |
| **Se** | 0.3 ppm (supplemental) | Регуляторный лимит США | — |

### Группа Б: «+20% к NASEM» (страховой запас)

| Минерал | Логика |
|---------|--------|
| **Ca** | Иронично — имеет наименьший эффект на гипокальциемию. Любая концентрация 0.4–1.0% — ок. Обычно ~0.8% (дешёвый разбавитель энергии) |
| **Na, Cl, K** | Вариабельность потребления, состава кормов |
| **Co, Fe, Zn** | Антагонисты, вариация |

### Группа В: «Значительно выше NASEM»

| Минерал | NASEM | Рекомендация Weiss | Причина |
|---------|-------|-------------------|---------|
| **Mg** | ~0.15% (+20% = 0.18%) | **0.4–0.45%** | K-антагонизм, гипомагниемия → риск молочной лихорадки |
| **Mn** | 40 ppm (+20% = 48 ppm) | **60–80 ppm** | Плод — боттлнек, корова выглядит нормально при дефиците |

## 3. Марганец (Mn) — критичен для плода, не для коровы

**Механизм:** Корова может находиться на дефицитном рационе по Mn и выглядеть абсолютно нормально. Телёнок при этом родится с клиническим дефицитом Mn. **Плод — боттлнек.**

**Данные:**
- Балансовое исследование (intake – feces – urine): нулевой баланс при **48 ppm** (совпадает с NASEM +20%).
- **Рекомендация:** 60–80 ppm для dry и pre-fresh.
- При высоком S (сульфатный DCAD): минимум **70–80 ppm** (S связывает Mn).
- Риск дефицита >> риск токсичности.

---

## 4. Селен (Se) — источник важнее дозы

**Регуляторный контекст (США):** max 0.3 ppm supplemental Se (total diet ~0.4 ppm). Это ниже потребности.

**Потребность:** 6–9 mg Se/день для dry cows. При DMI сухостоя ~10 кг = ~0.6 ppm в диете. Если законно — кормить **0.6 ppm для dry/pre-fresh**, 0.3 для лактирующих.

### Формы Se: не абсорбция, а трансфер

| Форма | Абсорбция | Трансфер в молозиво | Трансфер плоду | Устойчивость к S |
|-------|-----------|---------------------|----------------|------------------|
| **Селенит (Na₂SeO₃)** | Базовая | 1× (база) | Слабый | Подавляется |
| **Se-yeast / селенометионин** | +~20% | **2×** | +50–80% | Менее подавляется |

**Ключевой нюанс:** Различие не в абсорбции (лишь +20%), а в **трансфере в молоко и плод** (в 2 раза лучше).

**Рекомендуемое соотношение:** 2/3 Se-yeast/селенометионин + 1/3 селенит.

| Группа | Рекомендация |
|--------|-------------|
| **Сухостойные / pre-fresh** | Минимум 50% yeast, лучше 2/3–3/4 (трансфер плоду) |
| **Лактирующие** | Yeast vs selenite — эффект на мастит одинаковый. Yeast даёт больше Se в молоко |
| **Высокий S** (вода, distillers) | Больше yeast (менее подавляется S) |

## 5. Магний (Mg) — самый недооценённый минерал при transition

### Проблема NASEM-2021

NASEM: ~0.15% для dry/pre-fresh. +20% = 0.18%. По мнению Weiss — **недостаточно**.

### Механизмы

1. **K антагонизирует Mg абсорбцию.** Начинается даже при дефиците K, усиливается с ростом K. Dry cow diets (сенаж, сено) всегда высокие по K.
2. **MgO — катастрофическая вариабельность качества.** Лучший MgO ~30% доступности, худший — 1–2%. Разница в **10–30 раз**.
3. **Гипомагниемия → риск клинической гипокальциемии.** Кровяной Mg < 1.7 mg/dL → повышенный риск молочной лихорадки.
4. **Мета-анализ:** линейное снижение клинической гипокальциемии при увеличении Mg с 0.1% до 0.45%.

### Исследование NZ (пастбище, очень высокий K)

- Дренчинг 19 г Mg/день (~0.4% в диете)
- Сравнение MgCl₂, MgSO₄, высококачественный MgO
- Абсорбция (по urine Mg): примерно равная для всех трёх
- **Кровяной Ca:** лучше с MgCl₂/MgSO₄
- **Клиническая гипокальциемия:** снижена MgCl₂/MgSO₄

### Практическая рекомендация

| Параметр | Значение |
|----------|----------|
| **Целевая концентрация Mg** | 0.4–0.45% в pre-fresh |
| **Источники** | Часть из **MgCl₂ или MgSO₄** (defined chemicals, стабильная абсорбция ~30%) |
| **MgO** | Можно использовать, но требуется подтверждение качества (доступность) |
| **При высоком K** | Верхняя граница диапазона (0.45%) |

> **Конфаундинг:** В мета-анализе при увеличении Mg часто снижался DCAD (MgCl₂/MgSO₄). Независимый эффект Mg vs DCAD — не до конца ясен, но данные NZ говорят в пользу Mg.

## 6. DCAD (диетический катион-анионный баланс)

### Принцип: всё — компромисс

DCAD эффективен против гипокальциемии, но снижает потребление (особенно у тёлок).

### Ключевые правила

| # | Правило | Обоснование |
|---|---------|-------------|
| 1 | **Мочевая pH < 7 — обязательно** | Если pH > 7 — ноль эффекта на гипокальциемию |
| 2 | **Старт с DCAD ≈ -100 (расчётно), корректировать по мочевой pH** | Расчётный DCAD ≠ фактический (K варьирует, Cl варьирует) |
| 3 | **Целевая мочевая pH: 6.5–6.7** | Компромисс между защитой от гипокальциемии и потреблением |
| 4 | **Смотреть распределение, не только среднее** | Средняя pH = 6.5, но много коров >7 → проблема с сортировкой/кормлением |
| 5 | **Тёлок: НЕ кормить DCAD** | Не получают гипокальциемии, но теряют 1–2 кг потребления. Риск >> польза |
| 6 | **DCAD для всего сухостоя vs только pre-fresh** | Можно, если нет разделения. Потребление −1-2 кг на весь сухостой. Удельная энергия/белок выше. Молоко после отела хуже, чем при pre-fresh-only |

### Корректировки минералов при DCAD

| Компонент | Сульфатный DCAD | Хлоридный DCAD |
|-----------|----------------|----------------|
| **Trace minerals** | +20% сверху (S связывает Cu, Mn, Zn, Se) | Без изменений (Cl не антагонизирует) |
| **Mg** | 0.4–0.45%, часть MgCl₂/MgSO₄ | 0.4–0.45%, часть MgCl₂/MgSO₄ |
| **P** | ~0.2% (NASEM) | ~0.2% (NASEM) |
| **Ca** | 0.4–1.0% — без разницы | 0.4–1.0% — без разницы |
| **Витамины** | Без изменений | Без изменений |

## 7. Источники микроэлементов: органика vs сульфаты vs гидроксиды

### Ключевое замечание

«Органические микроэлементы» — **не однородный класс**. Качество варьирует между производителями. Требуйте конкретных данных.

### Сравнение форм

| Форма | Рубцевая доступность | Эффект на микробиому | Преимущества |
|-------|---------------------|----------------------|--------------|
| **Сульфаты (CuSO₄, ZnSO₄, MnSO₄)** | Очень высокая | Угнетают fiber digestion (свободные ионы токсичны для бактерий) | Дёшево, известная доступность |
| **Гидроксиды** | Низкая в рубце | Нейтральны | Не диссоциируют в рубце, не угнетают микробиому |
| **Органические (хелаты, комплексы)** | Переменная | Нейтральны | Лучший трансфер плоду, лучшее колострум |

### Когда что использовать

| Период | Рекомендация |
|--------|-------------|
| **Late gestation (pre-fresh)** | Бленд органических + гидроксидов. Органика — для трансфера плоду/молозиво |
| **Лактация** | Гидроксиды/сульфаты оправданы. Органика не всегда необходима |
| **Сульфатный DCAD** | Увеличить total trace minerals (+20%). Сульфаты в премиксе — минимизировать |

> **Тенденция:** Переход на гидроксиды и органические формы, сокращение сульфатов.

## 8. Окислительный стресс и антиоксиданты

### Концепция

Окислительный стресс вокруг отела — **невозможно и не нужно полностью устранить**. Некоторые ROS — сигнал к началу родов. Цель — **баланс**, не элиминация.

### Критичное различие: минералы-антиоксиданты vs витамин E

| | Cu, Zn, Se | Витамин E |
|---|---|---|
| **Механизм** | Кофакторы антиоксидантных ферментов | Непосредственный антиоксидант |
| **При избытке** | Становятся **проксидантами** | Продолжает работать как антиоксидант |
| **Prepartum** | **НЕ увеличивать** | **Увеличивать** |

> **Важно:** Увеличивать Cu, Zn prepartum НЕЛЬЗЯ — может усугубить окислительный стресс. Витамин E — наоборот, отвечает на увеличение дозировки.

## 9. Витамины — конкретные рекомендации

### Витамин E

| Группа | NASEM | Weiss | Комментарий |
|--------|-------|-------|-------------|
| **Dry** | 1000 IU/день | ~1200 IU/день | +20% к NASEM |
| **Pre-fresh** | 2000 IU/день | ~2400 IU/день | 2000–4000 IU снижает мастит и внутримамочные инфекции |
| **Fresh** | — | 500–1000 IU | Если нет fresh group |

> **КРИТИЧНО:** Избыток E на **ВЕСЬ сухостой** увеличивает мастит (RR=1.7, 4 из 5 ферм в крупном исследовании). Стратегия: **низкий** на dry → **высокий** на pre-fresh → **низкий** на fresh.

### Витамин A

- Плазменный ретинол падает за неделю до родов (массивный drain в молозиво: **90,000 IU** в первом молозиве = суточная норма коровы).
- Кормление большего количества A **НЕ предотвращает** это падение.

| Условия | Рекомендация |
|---------|-------------|
| **Пастбище** (зелёная трава = бета-каротин) | Supplemental A можно снизить до 25–50% NASEM |
| **Солома/силос** (нет бета-каротина) | ~100,000–110,000 IU/день (+15,000 к NASEM) |

### Витамин D: D₃ vs 25(OH)D₃

| | D₃ (холекальциферол) | 25(OH)D₃ (гидроксихолекальциферол) |
|---|---|---|
| **Метаболизм** | Печень → 25(OH)D → почки → 1,25(OH)₂D | Пропускает шаг печени |
| **Эффективность без DCAD** | Базовая | **Практически нет эффекта** |
| **Эффективность с DCAD** | Базовая | **Усиливает эффект DCAD** |
| **Доза** | Стандартная | **1–3 мг/день** |
| **Токсичность** | Низкая | **6 мг/день ухудшал гипокальциемию** |

**Практика:**
- Использовать **только с DCAD**
- Доза: 1–3 мг/день (только pre-fresh)
- Основной эффект — на **больных** коровах (метрит, др.), не на здоровых
- После отела — переходить на D₃
- Использовать **D₃, не D₂**

### Биотин

- **20 мг/день — на всю жизнь** (сухостой + лактация)
- Эффект на копыта — с лагом в месяцы
- Нет смысла давать только pre-fresh

### Холин

- **13 г active ingredient** (рубец-защищённый)
- Дорого, но обычно выгодно
- Pre-fresh + post-fresh

### Хром

- NASEM: no requirement, но данные положительные
- **0.5 ppm** хрома пропионата (единственный легальный в США)
- Начинать за 2 недели до родов, до 50–60 дней после
- Мета-анализ: снижает NEFA, увеличивает потребление, +~1 кг молока

## 10. Итоговый чек-лист формулирования

### Pre-fresh (главный фокус)

- [ ] **Гипокальциемия:** DCAD (целевая мочевая pH 6.5–6.7) ИЛИ P-binders
- [ ] **P:** ~0.2% (NASEM). Избыток вреден
- [ ] **Mg:** 0.4–0.45%, часть из MgCl₂/MgSO₄
- [ ] **Mn:** 60–80 ppm (при высоком S — 70–80 ppm)
- [ ] **Se:** преимущественно yeast/селенометионин (2/3) + selenite (1/3). Если законно — 0.6 ppm для dry/pre-fresh
- [ ] **Trace minerals (прочие):** бленд органических + гидроксидов. При сульфатном DCAD — +20% сверху
- [ ] **Cu, Zn:** НЕ увеличивать prepartum (проксидантный эффект)
- [ ] **Витамин E:** 2000–4000 IU/день (только pre-fresh, не на весь сухостой)
- [ ] **Витамин A:** пастбище → снижать; солома → 100,000+ IU
- [ ] **25(OH)D₃:** 1–3 мг (только при DCAD, только pre-fresh)
- [ ] **Биотин:** 20 мг/день (постоянно)
- [ ] **Холин:** 13 г (pre + post-fresh)
- [ ] **Хром:** 0.5 ppm (с -14 дней до +60 дней)
- [ ] **Тёлок:** НЕ включать в DCAD-группу

### Fresh (пост-партум)

- [ ] **K:** 40% выше NASEM (рубцовый буфер, водный баланс)
- [ ] **Mg:** 0.35–0.4% (высокий K → высокий Mg)
- [ ] **Витамин E:** 500–1000 IU (если нет fresh group) или оставить высоким
- [ ] **Меньше сульфатов, больше гидроксидов/органики**

## 11. Итоговые тезисы

1. **Transition — не «особые» коровы для всех минералов.** Но Mg, Mn, Se требуют специального подхода.
2. **Минералы ≠ витамины:** минералы адаптируют абсорбцию, витамины — нет. При сниженном DMI увеличивать концентрацию витаминов.
3. **Mn 60–80 ppm** критичен для плода. Корова может выглядеть здоровой при дефиците.
4. **Mg 0.4–0.45%** — недооценённый инструмент профилактики гипокальциемии. Часть из MgCl₂/MgSO₄.
5. **Se-yeast vs selenite:** разница не в абсорбции, а в трансфере в молозиво (2×) и плод.
6. **DCAD:** pH мочи < 7 обязательно. Цель 6.5–6.7. Тёлок — не кормить.
7. **Витамин E:** увеличивать только pre-fresh. Избыток на весь сухостой → мастит.
8. **25(OH)D₃:** работает только с DCAD. Доза 1–3 мг. 6 мг вреден.
9. **Cu, Zn prepartum:** не увеличивать. Минералы-антиоксиданты при избытке = проксиданты.
10. **Нельзя исправить 3 недели pre-fresh.** Полноценный рацион с начала сухостоя.
