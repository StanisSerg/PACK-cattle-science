# Модуль 3: Переходный период

## Цель модуля

Сформировать у технолога понимание метаболических процессов в переходный период и практические навыки профилактики метаболических заболеваний.

| # | Тема | Формат | Длительность | Источник (SoTA) |

| 3.2 | [Глицерол — энергетик и лечение кетоза](./02-lecture-glycerol.md) | Лекция | 40 мин | CS.SOTA.058, CS.SOTA.015, CS.SOTA.016 |

| 3.5 | Итоговый тест | Тест | 30 мин | — |

## Результаты обучения

После модуля слушатель сможет:

1. **Объяснить** изменения метаболизма при отёле
2. **Рассчитать** риск кетоза для конкретной коровы
3. **Применить** протокол профилактики метаболических заболеваний
4. **Оценить** эффективность мероприятий

## Связь с практикой

- **Задача:** Снизить заболеваемость кетозом с 15% до 5%
- **Инструмент:** Протокол оценки риска + чек-лист профилактики
- **Метрика:** % коров с BHB >1.2 ммоль/л в первые 21 день
