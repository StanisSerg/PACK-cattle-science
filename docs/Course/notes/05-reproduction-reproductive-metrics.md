# Лекция 5.2: Метрики воспроизводства

> **Ключевой тезис:** 21-дневная стельность (21-d PR) — это «пульс» репродуктивной программы. Она разлагается на два рычага управления: обнаружение охоты (Service Rate) и качество осеменения (Conception Rate).

---

## Введение для лектора

Эта лекция основана на практических заданиях курса ANS 235 (Dairy Herd Reproduction) и мета-анализах PACK-cattle-science. Студенты работают с реальными цифрами стада 1200 коров, что делает материал осязаемым.

4. Определять узкое место программы: проблема в обнаружении охоты или в фертильности
5. Интерпретировать график open cows по DIM

## 1. Три столпа репродуктивной эффективности (10 мин)

### 1.1. 21-Day Pregnancy Rate (21-d PR)

**Что это:** процент коров, которые стельнеют за 21-дневный период.

```
21-d PR = Service Rate × Conception Rate × 100
```

Или по данным:
```
21-d PR = (Стельные за 21 день / Коровы eligible) × 100
```

**Почему 21 день:** это длина эструсного цикла коровы. Одна «единичная вероятность» за цикл.

**Бенчмарки (CS.SOTA.005, 013):**

| Уровень | 21-d PR | Интерпретация |
|---------|---------|---------------|
| Критично | <15% | Стадо теряет деньги на каждой открытой корове |
| Ниже среднего | 15-25% | Возможно, одна проблема (SR или CR) |
| Хорошо | 25-35% | Конкурентоспособный уровень |
| Отлично | >35% | Топ-10% хозяйств |

### 1.2. Service Rate (SR) — Обнаружение охоты

**Что это:** процент коров, которые были осеменены (или отмечены) за 21 день.

```
Service Rate = (Осеменённые / Коровы eligible) × 100
```

**Важно:** «eligible» = коровы после VWP (Voluntary Waiting Period), не стельные, без ветеринарных запретов.

**Целевые значения (CS.SOTA.005):**

| Система | Минимум | Цель |
|---------|---------|------|
| ED-only (естественное охота) | 70% | 85% |
| TAI (фиксированное время) | 95% | 100% |
| Комбинация | 85% | 95% |

> **Аналогия:** Service Rate — это «сколько дверей мы постучали». Если не постучали — не узнаем, откроется ли.

### 1.3. Conception Rate (CR) — Качество осеменения

**Что это:** процент осеменённых коров, которые застельнелись.

```
Conception Rate = (Стельные / Осеменённые) × 100
```

**Целевые значения:**

| Параметр | Низкий | Средний | Высокий |
|----------|--------|---------|---------|
| CR | <25% | 25-35% | >45% |

**Критический фактор:** BCS при отёле и динамика BCS (CS.SOTA.015):

| ΔBCS (0-21 DIM) | Conception Rate |
|-----------------|-----------------|
| Потеря >0.75 | 25% |
| Потеря 0.25-0.75 | 40% |
| Стабильные | 55% |
| Прирост | 83% |

## 2. Практический расчёт: стадо 1200 коров (20 мин)

### 2.1. Исходные данные

**Current Situation (плохая программа):**
- VWP = 60 дней
- Service Rate = 38%
- Conception Rate = 33%
- Cull после 11 циклов

**Proposed Situation (улучшенная программа):**
- VWP = 80 дней
- Service Rate = 72%
- Conception Rate = 49%
- Cull после 11 циклов

### 2.2. Расчёт 21-d PR

```
Current:  PR = 0.33 × 0.38 = 0.1254 = 12.54%
Proposed: PR = 0.49 × 0.72 = 0.3528 = 35.28%
```

**Вывод:** Proposed программа даёт почти **тройной рост** стельности за цикл.

### 2.3. Распределение коров по циклам

Таблица для Proposed Situation (1200 коров):

| Цикл | Open (старт) | Serviced (72%) | Pregnant (49%) | Open (остаток) | Всего стельных |
|------|-------------|----------------|----------------|----------------|----------------|
| 1 | 1200 | 864 | 588 | 777 | 423 |
| 2 | 777 | 559 | 256 | 503 | 274 |
| 3 | 503 | 362 | 165 | 326 | 177 |
| 4 | 326 | 234 | 107 | 211 | 115 |
| 5 | 211 | 151 | 69 | 137 | 74 |
| ... | ... | ... | ... | ... | ... |
| 11 | 17 | 12 | 5 | 12 | 5 |

**Ключевые цифры:**
- **Всего стельных к 140 DIM:** Proposed = 989 коров (82%), Current = 496 (41%)
- **Остаток открых после 11 циклов:** Proposed = 12 коров (1%), Current = 278 (23%)

### 2.4. График open cows по DIM

**Current:** пик открых коров смещён вправо (большинство коров открыты >200 DIM)
**Proposed:** кривая резко падает — 80%+ коров стельны к 150 DIM

**Практический вывод:**
- Current = «болото» открых коров, каждая из которых теряет $2-5/день
- Proposed = «конвейер» — коровы стельнеют в первые 3-4 цикла

## 3. Диагностика: SR vs CR (15 мин)

### 3.1. Алгоритм разложения

```
21-d PR = Service Rate × Conception Rate

Пример: 20% = 80% × 25%
              ↓       ↓
            норма   низкий
                    → фокус на CR (фертильность)
```

| Комбинация | SR | CR | 21-d PR | Диагноз |
|-----------|----|----|---------|---------|
| Хорошая | 85% | 45% | 38% | Идеал |
| Проблема в обнаружении | 50% | 45% | 22% | Нужны мониторы активности или TAI |
| Проблема в фертильности | 85% | 25% | 21% | Проверять BCS, переходный период, VWP |
| Всё плохо | 50% | 25% | 12% | Два узких места |

### 3.2. Если проблема в Service Rate

**Причины:**
- ED-only: плохое обнаружение охоты (особенно ночью)
- Недостаточно наблюдений (рекомендуется 3×/день по 30 мин)
- Тихие охоты (особенно летом, при стрессе)

**Решения:**
- Внедрение TAI (фиксированное время осеменения) → SR поднимается до 95%+
- Мониторы активности (Heatime, Allflex)
- Обучение персонала (симптомы охоты)

### 3.3. Если проблема в Conception Rate

**Метаболические факторы (CS.SOTA.004, 015):**
- BCS при отёле >3.5 (перекормленные коровы)
- Потеря BCS >0.75 за 21 DIM (недокорм в переходный период)
- NEFA >0.7 ммоль/л (отрицательный энергетический баланс)
- Гипокальция (субклиническая)

**Управленческие факторы:**
- VWP слишком короткий (<50 дней)
- Качество семени / техника AI
- Сезонный стресс (летний стресс снижает CR на 10-20%)

## 4. Average Days Open (ADO) (10 мин)

### 4.1. Что это

Среднее количество дней от отёла до стельности по стаду.

**Формула:**
```
ADO = Σ(Days Open для каждой коровы) / Количество коров
```

**Целевые значения:**
- <100 дней — отлично
- 100-120 — хорошо
- 120-150 — ниже нормы
- >150 — критично (теряем деньги)

### 4.2. Связь ADO и 21-d PR

```
Примерно: ADO ≈ VWP + (1 / 21-d PR) × 21 дней
```

| 21-d PR | Ожидаемое ADO (при VWP=60) |
|---------|---------------------------|
| 12% | ~60 + 8.3×21 = 234 дня |
| 25% | ~60 + 4×21 = 144 дня |
| 35% | ~60 + 2.9×21 = 121 день |

**Вывод:** увеличение 21-d PR с 12% до 35% сокращает ADO на ~100 дней.

### 4.3. Экономика Days Open (CS.SOTA.001)

```
Стоимость 1 дня open = $2-5 / корова
```

**Пример (стадо 500 коров, улучшение ADO с 130 до 110):**
- Экономия: 500 × 20 дней × $3 = **$30,000/год**

## 5. Практические сценарии (5 мин)

### Сценарий 1: «У нас SR 40%, CR 35%»

**21-d PR:** 0.40 × 0.35 = 14% (критично)
**Диагноз:** проблема в обнаружении охоты (SR низкий)
**Решение:** TAI или мониторы активности

### Сценарий 2: «У нас SR 85%, CR 28%»

**21-d PR:** 0.85 × 0.28 = 23.8% (ниже среднего)
**Диагноз:** проблема в фертильности
**Решение:** проверить BCS, переходный период, VWP

## 6. Итоги лекции

### 6.1. Чек-лист для запоминания

- [ ] 21-d PR = Service Rate × Conception Rate
- [ ] Service Rate = обнаружение охоты (цель: >85%)
- [ ] Conception Rate = качество осеменения (цель: >45%)
- [ ] BCS при отёле 2.75-3.0 — критично для CR
- [ ] ADO >150 дней = потери $2-5/день на корову
- [ ] Увеличение 21-d PR с 12% до 35% = +$30k/год на 500 коров

### 6.2. Три ключевых сообщения

**1. «Разложи проблему — найди узкое место»**

Не «у нас плохая репродукция», а «у нас низкий SR» или «у нас низкий CR». Это определяет конкретные действия.

**2. «21-d PR — пульс программы»**

Одна цифра показывает здоровье всей репродуктивной системы. Бенчмарк: >25%.

**3. «Обнаружение охоты дешевле фертильности»**

SR проще поднять (TAI, мониторы), чем CR (требует изменения кормления, BCS, здоровья).
