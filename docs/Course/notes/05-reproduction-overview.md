# Модуль 5: Воспроизводство молочного скота

> **Ключевой тезис:** Воспроизводственная эффективность — это не «биология», а управляемый процесс с измеримыми метриками, бенчмарками и экономическим эффектом.

## Цели модуля

После прохождения модуля технолог сможет:

1. Собирать минимальный dataset для оценки воспроизводства
2. Рассчитывать 21-d PR, Service Rate, Conception Rate
3. Сравнивать метрики с бенчмарками и диагностировать проблемы
4. Приоритизировать вмешательства по ROI
5. Оценивать экономический эффект программ ВОС
