# Руководство по внесению данных в AMTS

> **Модуль 1** | На основе: AMTS LLC White Paper (2014)  
> **Upstream:** PACK-cattle-science

## 1. Принцип работы модели

AMTS.Cattle.Pro — это сложный калькулятор на основе модели CNCPS (Cornell Net Carbohydrate and Protein System).

### Что модель рассчитывает

| Параметр                    | Обозначение | Единицы                  |
| --------------------------- | ----------- | ------------------------ |
| Потребление сухого вещества | DMI         | кг/день                  |
| Энергия метаболизма         | ME          | Mcal/день                |
| Метаболизируемый протеин    | MP          | г/день                   |
| Продуктивность              | —           | кг молока, прирост массы |

### Ключевой принцип

> *"Программа предсказывает хорошо, только если входные данные верны."*
> — Tom Tylutki

Модель принимает **входные данные** → выполняет расчёты → выдаёт **прогнозы**. Если входные данные неточны, прогнозы будут ошибочными.

## 2. Критические входные данные

Приоритет по важности:

| Приоритет | Параметр         | Влияние на результат   | Типичная ошибка            |
| :-------: | ---------------- | ---------------------- | -------------------------- |
| 1         | Живая масса      | ±45 кг → ±5% ME        | Недооценка "на глаз"       |
| 2         | Температура      | 0°C vs 32°C → ±15% DMI | Игнорирование микроклимата |
| 3         | Активность       | 900 м ходьбы → +ME     | Недооценка расстояния      |
| 4         | ADG              | Влияет на MP           | Путаница с BCS             |
| 5         | Параметры кормов | kd → MP из бактерий    | Неправильные лаб. данные   |

---

## 3. Живая масса (критически важно)

### Влияние погрешности

При ошибке ±45 кг (±100 фунтов):

| Параметр           | Изменение |
| ------------------ | --------- |
| ME на поддержание  | ±5%       |
| Прогнозируемое DMI | ±4%       |
| ME-молоко          | ±4%       |
| MP из бактерий     | Снижение  |

### Почему это происходит

Живая масса влияет на:

1. Потребность в энергии на поддержание
2. Скорость прохождения корма через рубец
3. Прогнозируемое потребление корма

### Рекомендации

- ✅ Взвешивать животных регулярно
- ✅ Для групп — использовать среднее взвешенное по дням доения
- ✅ Учитывать стадию лактации (сухостой vs дойные)
- ✅ Для молодых животных — вводить текущий вес, не взрослый

---

## 4. Температура и окружающая среда

### Влияние температуры

| Условия        | DMI     | ME на поддержание |
| -------------- | ------- | ----------------- |
| 0°C (холодно)  | +10-15% | Повышается        |
| 20°C (комфорт) | Базовый | Базовый           |
| 32°C (жарко)   | −10-15% | Незначительно ↓   |

**Важно:** Лактирующие коровы чувствительнее к жаре, чем к холоду (генерируют много тепла от ферментации).

### Активность (расстояние до доильного зала)

| Расстояние         | Влияние            |
| ------------------ | ------------------ |
| 300 м (1000 футов) | Базовый уровень    |
| 900 м (3000 футов) | +ME на поддержание |

### Рекомендации

- ✅ Указывать температуру в помещении, не на улице
- ✅ Учитывать влажность (высокая усиливает жару)
- ✅ Измерять реальное расстояние ходьбы, не "по прямой"
- ✅ Использовать калькулятор активности в AMTS

## 5. Среднесуточный прирост (ADG)

### Различие: ADG (ССП) vs BCS

| Параметр | Что измеряет                   | Пример                         |
| -------- | ------------------------------ | ------------------------------ |
| **ADG**  | Физический рост (кости, мышцы) | Первая лактация: +0.22 кг/день |
| **BCS**  | Изменение жировых запасов      | Потеря 0.5 балла за 90 дней    |

**Корова может одновременно:**

- Терять BCS (худеть визуально)
- Расти по массе (ADG > 0)

### Типичные значения ADG

| Группа                      | ADG, кг/день | Примечание                 |
| --------------------------- | ------------ | -------------------------- |
| Первая лактация (смешанная) | 0.22         | До достижения зрелой массы |
| Лёгкие первотёлки (450 кг)  | 0.52         | Высокие потребности        |
| Взрослые (4+ лактация)      | 0            | Зрелая масса достигнута    |

### Рекомендации

- ✅ Для первотёлок всегда указывать ADG > 0
- ✅ Для взрослых — ADG = 0 (если нет специальных целей)
- ✅ Не путать изменение BCS с физическим ростом

## 6. Параметры кормов

### Ключевые параметры деградации

| Параметр                     | Обозначение | Влияние                   |
| ---------------------------- | ----------- | ------------------------- |
| Скорость деградации крахмала | Starch kd   | Энергия из рубца          |
| Скорость деградации NDF      | CHO B3 kd   | Скорость прохождения, DMI |

### Влияние на результат

При снижении скоростей деградации:

- Меньше энергии из рубца
- Меньше микробного протеина
- Снижение MP-молока (может упасть с 40 до 25 кг)

### Рекомендации

- ✅ Использовать лабораторные анализы с определением kd
- ✅ Не использовать "типовые" значения без проверки
- ✅ Обращать внимание на обработку зерна (плющение, экструдирование)
- ✅ Регулярно перепроверять параметры кормов

---

## 7. Итоговые рекомендации

### Приоритеты при внесении данных

1. **Живая масса** — наибольшее влияние, взвешивать регулярно
2. **Температура** — особенно важна для высокопродуктивных коров
3. **Активность** — часто недооценивается, измерять реальное расстояние
4. **ADG для молодых** — не игнорировать рост первотёлок
5. **Параметры кормов** — использовать реальные лабораторные данные

### Проверка результатов

- Сравнивать прогнозируемое и фактическое DMI
- При расхождении >10% — перепроверить входные данные
- Начинать проверку с массы и температуры
