## Learning Objectives

После прохождения модуля технолог сможет:

1. Определить потребности коровы в энергии (NEL, ME) в зависимости от удоя, массы, стадии лактации
2. Рассчитать потребность в метаболизируемом протеине (MP) и аминокислотах
3. Подобрать оптимальное соотношение кальций:фосфор по фазам лактации
4. Скорректировать рацион под жару, стресс, заболевания
5. Использовать таблицы NASEM 2021 для быстрой оценки рациона
6. Интерпретировать показатели клетчатки (aNDFom, peNDF, uNDF) и учитывать их вариабельность

## Ключевые понятия

| Термин | Определение |
|--------|-------------|
| **NEL** | Net Energy Lactation — чистая энергия лактации, Мкал/кг |
| **MP** | Metabolizable Protein — метаболизируемый протеин, г/день |
| **DMI** | Dry Matter Intake — потребление сухого вещества, кг/день |
| **NDF** | Neutral Detergent Fiber — нейтрально-детергентная клетчатка |
| **MUN** | Milk Urea Nitrogen — мочевина в молоке, индикатор баланса протеина |
| **aNDFom** | ash-free NDF organic matter — стандартный показатель клетчатки |
| **peNDF** | physically effective NDF — фракция, стимулирующая жвачку |
| **uNDF** | undigested NDF — неперевариваемая клетчатка |
