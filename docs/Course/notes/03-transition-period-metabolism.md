# Лекция 3.1: Метаболизм в переходный период

> **Ключевой тезис:** Переходный период (3 недели до до 3 недель после отёла) — критическое окно, когда происходит большинство метаболических нарушений.

---

## Цели лекции (2 мин)

После лекции вы сможете:
1. Объяснить, почему переходный период "интересный" (опасный)
2. Описать изменения липидного и углеводного метаболизма
3. Понять связь между метаболизмом и иммунитетом

## 1. Введение: Почему это важно? (8 мин)

### 1.1. Статистика заболеваемости

| Заболевание | Пик частоты | Связь с метаболизмом |
|-------------|-------------|---------------------|
| Молочная лихорадка | Дни 0-3 | Кальциевый обмен |
| Кетоз | Дни 0-21 | Энергетический дефицит |
| Задержка последа | Дни 0-10 | Иммуносупрессия |
| Метрит | Дни 7-21 | Иммуносупрессия |
| Смещение рубца | Дни 14-60 | Атония, метаболизм |

**Вывод:** Большинство проблем — следствие метаболической адаптации.

### 1.2. Энергетический баланс

```
До отёла:                    После отёла:
Потребление ↑                Потребление ↓↓ (аппетит -30%)
Потребность ↓                Потребность ↑↑ (молоко +40 кг)
Баланс 0 или +               Баланс --- (глубокий минус)
```

**Результат:** Мобилизация жировых запасов → NEFA ↑ → кетоз, жировой гепатоз.

## 2. Липидный метаболизм (15 мин)

### 2.1. Пути метаболизма жирных кислот в печени

```
Вход: NEFA из жировой ткани
        │
        ▼
┌─────────────────┐
│  Активация:     │
│  NEFA → Acyl-CoA│
└────────┬────────┘
         │
    ┌────┴────┬────────────┐
    ▼         ▼            ▼
Митохондрии  Пероксисомы   Этерификация
(β-окисление) (β-окисление)   ↓
    │         │            TAG
    │         │            (накопление)
    ▼         ▼
  Acetyl-CoA  Acetyl-CoA
    │         │
    └────┬────┘
         ▼
    ┌─────────────┐
    │  Цикл Кребса │ ← Oxaloacetate (OAA)
    └─────────────┘
         │
    ┌────┴────┐
    ▼         ▼
  CO₂ + H₂O  Кетоновые тела (если OAA недостаточно)
    (энергия)   (Ацетон, Ацетоацетат, β-OH-butyrate)
```

### 2.2. Контрольные точки

| Фермент | Роль | Регуляция |
|---------|------|-----------|
| **CPT-1** | Транспорт ЖК в митохондрии | Ингибируется малонил-КоА |
| **PPARα** | Индукция пероксисомального окисления | Активируется при голодании |
| **PC (пируваткарбоксилаза)** | Пополнение OAA | ↑ при ограничении корма |

### 2.3. Практический вывод

> **Чем больше NEFA поступает в печень, тем выше риск:**
> - Жирового гепатоза (этерификация → TAG)
> - Кетоза (избыток ацетил-КоА → кетоновые тела)

## 3. Глюконеогенез (12 мин)

### 3.1. Почему глюкоза критична?

- Молочная лактоза = 4.8% молока
- 40 кг молока = 1.9 кг лактозы/день
- Вся лактоза синтезируется **de novo** в молочной железе из глюкозы крови

### 3.2. Субстраты глюконеогенеза

| Субстрат | Вклад | Источник |
|----------|-------|----------|
| **Пропионат** | 60-74% | Рубцовое брожение |
| Аминокислоты | 15-25% | Протеолиз мышц |
| Глицерол | 5-10% | Липолиз |
| Лактат | 5% | Рубец, мышцы |

**Ключевой вывод:** Пропионат — король глюконеогенеза.

### 3.3. Почему коровы не могут просто есть глюкозу?

> **У взрослых коров отсутствует гепатическая глюкокиназа.**
> 
> Даже при внутривенной инфузии глюкозы печень продолжает активный глюконеогенез.

Это эволюционная адаптация:
- Постоянный приток пропионата из рубца
- Нужно утилизировать пропионат (иначе ацидемия)
- Глюконеогенез — единственный путь

## 4. Связь метаболизма и иммунитета (5 мин)

### 4.1. Энергетический дефицит → иммуносупрессия

```
Негативный энергетический баланс
        │
        ▼
    NEFA ↑
        │
        ▼
┌─────────────────┐
│  Жировая        │
│  дистрофия      │
│  печени         │
└────────┬────────┘
         │
    ┌────┴────┐
    ▼         ▼
Метаболические  Иммунные
последствия:    последствия:
- Кетоз         - Снижение фагоцитоза
- Печёночная    - Снижение синтеза
  недостаточность  иммуноглобулинов
- Гипогликемия  - Повышенный риск
                   инфекций
```

### 4.2. Практическое следствие

Корова с кетозом в 5 раз чаще заболевает маститом, метритом.

---

## 5. Итоги и ключевые сообщения (3 мин)

### 5.1. Чек-лист для запоминания

- [ ] Переходный период: 3 нед до → 3 нед после отёла
- [ ] Энергодефицит → мобилизация жира → NEFA ↑
- [ ] Печень — центральный орган метаболизма
- [ ] Пропионат — 60-74% субстратов для глюкозы
- [ ] Кетоз + иммуносупрессия = взрыв заболеваемости

### 5.2. Ключевые сообщения для практики

1. **"Лучшая профилактика кетоза — корм, корм и ещё раз корм"**
   - Максимизация СВС после отёла
   - Стабильный рацион (без резких изменений)

2. **"Кетоз — это не просто переедание жира, это провал глюконеогенеза"**
   - Нужен пропионат, не просто энергия
   - Монензин → пропионат ↑

3. **"Печень не справляется — страдает всё"**
   - Метаболизм + иммунитет + детоксикация
