# Курс по кормлению — Кормление и содержание молочного скота

Базовый курс для технологов молочных ферм. Практические навыки работы с программами кормления, составление рационов, управление переходным периодом.

## Структура

```
Курс по кормлению/
├── README.md     # Этот файл
├── video/        # Видеозаписи лекций, транскрипты, аудио
├── docs/         # Слайды, PDF, методички, задания, таблицы
└── notes/        # Лекции, гайды, конспекты, анализы
```
