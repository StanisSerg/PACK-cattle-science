# ComparativeReviewUnit (E.17.ID.CR): NARRATIVE vs COARSENED

## Критерии сравнения

| Критерий | NARRATIVE.md | COARSENED.md | Примечание |
|----------|--------------|--------------|------------|
| Объём | ~7500 слов, 13 блоков | ~1300 слов, 13 тезисов | Coarsened — сжатие ~6× |
| EntityOfConcern | Сохранён | Сохранён | Оба рендеринга про одно и то же |
| Тайминги | Да, по блокам | Только общие границы | Coarsened теряет точные тайминги |
| Прямая речь | Много цитат | Нет | Coarsened теряет голос спикера |
| Примеры | Детальные (художник, вождение, Boeing, печень) | Сведены к одному предложению | Coarsened declared loss |
| Практикумы | Разобраны | Упомянуты как список | Coarsened теряет детали практик |
| Пригодность для изучения | Высокая | Средняя | Coarsened — только обзор |
| Пригодность для цитирования | Высокая | Низкая | Coarsened не сохраняет точные формулировки |

## Вывод

NARRATIVE.md лучше для погружения, COARSENED.md — для быстрого обзора. Coarsened имеет declared loss mode и narrower admissible use, что соответствует A.6.3.CSC.
