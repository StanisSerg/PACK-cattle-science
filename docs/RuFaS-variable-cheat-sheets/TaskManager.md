# TaskManager

Управление задачами и инициализация симуляции. Сид случайных чисел, общие параметры запуска.

## Примеры regex-фильтров

```regex
^TaskManager\.
```

## Переменные

| Переменная (полное имя для фильтра) | Единицы | Пояснение |
|---|---|---|
| `TaskManager.set_random_seed.random_seed` | unitless | Сид генератора случайных чисел |
