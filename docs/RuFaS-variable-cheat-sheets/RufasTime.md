# RufasTime

Временные переменные симуляции: день, год, календарный год, день симуляции.

## Примеры regex-фильтров

```regex
^RufasTime\.
```

## Переменные

| Переменная (полное имя для фильтра) | Единицы | Пояснение |
|---|---|---|
| `RufasTime.day` | julian day | День года (julian day) |
| `RufasTime.year` | simulation year | Год симуляции |
| `RufasTime.calendar_year` | calendar year | Календарный год |
| `RufasTime.simulation_day` | simulation day | День симуляции |
