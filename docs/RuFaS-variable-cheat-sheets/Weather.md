# Weather

Погодные данные: температура, осадки, солнечная радиация, орошение.

## Примеры regex-фильтров

```regex
^Weather\.
```

## Переменные

| Переменная (полное имя для фильтра) | Единицы | Пояснение |
|---|---|---|
| `Weather.average_annual_temperature` | °C | Среднегодовая температура воздуха |
| `Weather.precipitation` | mm | Суточные осадки (дождь + снег) |
| `Weather.rainfall` | mm | Суточный дождь |
| `Weather.snowfall` | mm | Суточный снег |
| `Weather.maximum_temperature` | °C | Максимальная температура воздуха |
| `Weather.minimum_temperature` | °C | Минимальная температура воздуха |
| `Weather.average_temperature` | °C | Средняя температура воздуха |
| `Weather.radiation` | MJ/m^2 | Солнечная радиация |
| `Weather.irrigation` | mm | Суточное орошение |
