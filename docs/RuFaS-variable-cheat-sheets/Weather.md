# Weather

Погодные данные: температура, осадки, солнечная радиация, орошение.

## Примеры regex-фильтров

```regex
^Weather\.
```

## Переменные

| Переменная (полное имя для фильтра) | Единицы | Пояснение |
|---|---|---|
| `Weather.average_annual_temperature` | °C | Средняя температура |
| `Weather.precipitation` | mm | Погодный показатель |
| `Weather.rainfall` | mm | Количество осадков в виде дождя |
| `Weather.snowfall` | mm | Количество осадков в виде снега |
| `Weather.maximum_temperature` | °C | Погодный показатель |
| `Weather.minimum_temperature` | °C | Погодный показатель |
| `Weather.average_temperature` | °C | Средняя температура |
| `Weather.radiation` | MJ/m^2 | Погодный показатель |
| `Weather.irrigation` | mm | Погодный показатель |
