#### 4.4.2. Пересчёт N Efficiency на ФАКТИЧЕСКОЕ молоко

AMTS считает Productive N на основе **Input Milk = 25 кг**. Но фактическое молоко другое. Пересчитаем:

**Предположения:**
- Молочный белок (Crude) = 3.39%
- Азот в белке = 16%
- N на беременность + прирост (AMTS): ~2.24 + 24.30 = ~26.5 г (примерно одинаково)

| Группа | Факт молоко, кг | Молочный белок, кг | Молочный N, г | Продуктивный N (молоко + прочее), г | N Intake (факт DMI=19.3) | **N Efficiency** |
|--------|----------------|--------------------|---------------|--------------------------------------|--------------------------|------------------|
| **Гр.15** | 23 | 0.78 | ~125 | ~151 | ~578* | **~26.1%** |
| **Гр.1** | 32.1 | 1.09 | ~174 | ~200 | ~578* | **~34.6%** |

\* N Intake при DMI=19.3 кг: 482.23 × (19.3/16.1) = **~578 г**

**Выводы по N Efficiency:**
1. **N Efficiency гр.1 (34.6%) > гр.15 (26.1%)** — высокопродуктивные коровы используют азот эффективнее (больше молока на грамм азота).
2. **Новый рацион даёт больше N Intake (+21 г при 16.1 кг, ~+116 г при 19.3 кг)** — это риск роста мочевого N без роста молока.
3. **MUN** — отличный индикатор:
   - Старый: **6.94 мг/дл** (низкий, <8 — возможен дефицит руменного NH₃)
   - Новый: **8.23 мг/дл** (оптимальная зона 8–12)
   - При факт DMI=19.3 кг MUN будет ещё выше: **~9.5–10 мг/дл** (хорошо)
4. **Rumen N Balance** +84.88 г — избыток азота для микробов. Это позитивно для синтеза МП, но при высоком MUN (>12) может сигнализировать о потерях.
