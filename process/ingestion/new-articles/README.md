## Структура папок

```
new-articles/
├── README.md              # Этот файл
├── [старые PDF]           # Уже обработанные (25 шт.)
├── W12/                   # Новые статьи для недели W12
```

---

## Процесс

1. **Добавление:** PDF в `W12/` или `W13/`
2. **Обработка:** По шаблону `TEMPLATES/SOTA-TEMPLATE.md`
3. **Выход:** `06-sota/[area]/CS.SOTA.XXX-author-year.md`
4. **Метрики:** Фиксация времени на каждую статью
