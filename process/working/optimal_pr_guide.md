# Оптимальный 21-дневный коэффициент оплодотворения (21-d PR)

**Целевые показатели репродуктивной эффективности молочного стада**

---

## Ключевой вывод

**Оптимальный 21-d PR зависит от контекста хозяйства**

---

## Оптимальный 21-d PR по контекстам

| Контекст | Оптимальный 21-d PR | Обоснование |
|:---------|:-------------------:|:------------|
| Минимально приемлемый | >=20% | Ниже — дефицит телок (Lauber 2025) |
| Целевой (средний) | 25-30% | Средний показатель по отрасли |
| Высокий (топ 25%) | 30-35% | Лучшие хозяйства |
| Отличный (топ 10%) | >=35% | Максимальная эффективность |

---

## Экономика разных уровней 21-d PR

| 21-d PR | Статус | Доходность | Оптимальный IEP |
|:-------:|:-------|:-----------|:----------------|
| 20% | Низкий | Базовый ($0) | 170 дней |
| 25% | Ниже среднего | +$10-30/кор/год | 140 дней |
| 30% | Средний | +$30-60/кор/год | 110 дней |
| 35% | Выше среднего | +$60-90/кор/год | 95 дней |
| 40% | Высокий | +$90-120/кор/год | 80 дней |

<p class="note">
<b>Важно:</b> Каждый +1% 21-d PR = +$2-7/кор/год дополнительной прибыли (Lauber 2025)
</p>

---

## Практические рекомендации по уровням 21-d PR

### Если 21-d PR < 20% (Критический уровень)

- **Проблема:** Недостаток телок для замены
- **Действие:** Срочно улучшать репродукцию или увеличить выживаемость телок
- **IEP:** Максимальный (170+ дней) — сохранять коров дольше

### Если 21-d PR 20-25% (Низкий уровень)

- **IEP:** 140-170 дней (conventional семя)
- **Приоритет:** Увеличение 21-d PR до 30%+
- **Семя:** Только conventional (не sexed+beef — риск дефицита)

---

### Если 21-d PR 25-35% (Средний уровень)

- **IEP:** 110-140 дней
- **Семя:** Можно переходить на sexed+beef при HSR > 85%

### Если 21-d PR >= 35% (Высокий уровень)

- **IEP:** 80-110 дней (короткий период!)
- **Семя:** Sexed+beef оптимален (+$51/кор/год)
- **Стратегия:** Максимизировать доход от молока, быстрая замена

---

## Формула расчёта

| Компонент | Описание | Типичное значение |
|:----------|:---------|:------------------|
| Service Rate | % коров, осеменённых за 21 день | 60-80% |
| P/AI (Conception) | % оплодотворения на осеменение | 35-45% |
| 21-d PR | Service Rate × P/AI × 100 | 20-40% |

**Пример:** Service Rate = 70%, P/AI = 40% → 21-d PR = 0.70 × 0.40 × 100 = **28%**
