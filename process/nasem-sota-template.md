### 3.3 Основные части

Каждая глава NASEM делится на **логические части**:

```markdown
## Часть 1. [Название — физиология/основы]

### 1.1 [Подраздел]
[Текст с объяснением механизмов]

### 1.2 [Подраздел]
[Таблицы с нормами, но БЕЗ конкретных рецептов рационов]
```

**Что включать:**
- ✅ Физиологические механизмы
- ✅ Нормы питательности (NEL, CP, NDF, минералы)
- ✅ Формулы расчёта
- ✅ Таблицы с референсными значениями
- ✅ Алгоритмы принятия решений

**Что НЕ включать:**
- ❌ Конкретные рецепты рационов (% силоса, сена, концентрата)
- ❌ Данные без подтверждения из NASEM
- ❌ Непроверенные нормы (отметить "требует уточнения")

### 3.4 Практические кейсы (3-4 штуки)

**Структура каждого кейса:**

```markdown
### Кейс [N]: [Краткое название]

**СИТУАЦИЯ:**
- Ферма: [размер], [регион]
- Проблема: [что было]
- Предыдущий подход: [что делали]

**АНАЛИЗ:**
- [Фактор 1]
- [Фактор 2]

**РЕШЕНИЕ:**

**Шаг 1:** [Действие]
- Результат: [что изменилось]

**Шаг 2:** [Действие]
- Результат: [что изменилось]

**РЕЗУЛЬТАТ:**
- [Показатель 1]: [было] → [стало]
- [Показатель 2]: [было] → [стало]

**УРОК:**
> [Ключевой вывод для практики]
```

**Типы кейсов:**
1. **Внедрение с нуля** — полный цикл изменений
2. **Решение проблемы** — типичная ошибка и фикс
3. **Связь факторов** — как одно влияет на другое
4. **Пошаговый расчёт** — учебный пример с цифрами

## 4. Правила оформления

### 4.1 Терминология

| Не использовать | Использовать |
|-----------------|--------------|
| Свежие коровы | **Новотельные коровы** |
| Палатабельность | **Привлекательность рациона** |
| Молочная лихорадка | **Клиническая гипокальция** (или "молочная лихорадка" в скобках) |

### 4.2 Единицы измерения

| Параметр | Единица |
|----------|---------|
| Деньги | **$ (USD)** — пересчитывать по курсу |
| Энергия | Mcal/кг или MJ/кг |
| Белок | % СВ или г/кг СВ |
| Минералы | % СВ или мг/кг СВ |
| Кальций в крови | mmol/L |
| NEFA | mmol/L |
| BHBA | mmol/L |

### 4.3 Инфобоксы

Использовать для акцента:

```markdown
**⚠️ ВАЖНО:**
> [Критическое предупреждение]

**💡 ПРАКТИЧЕСКИЙ СОВЕТ:**
> [Полезный совет из практики]
```

### 4.4 Данные требующие уточнения

Если данные взяты не из NASEM или требуют проверки:

```markdown
> **Примечание:** Требует уточнения из первоисточника.
```

Или убрать до подтверждения (как с ионизированным кальцием).
