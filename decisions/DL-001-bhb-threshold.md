# DL-001: Порог BHB 1.2 — смена стратегии с симптоматики на систему

> **Источник:** [CASE-001: BHB 1.6](../cases/CASE-001-bhb-threshold.md)  
> **Дата принятия:** 2026-03-20 (анализ после CASE-001)

## Что именно было решением?

**Ситуация (IF):**
```
- BHB > 1.2 mmol/L (в CASE-001: BHB 1.6)
- Низкий DMI (<80% от нормы)
- Потеря BCS (>0.5 с отёла)
- Дни отёла < 30 (ранний постпартум)
- Клинические признаки минимальны (нет ацетона)
```

**Действие (THEN):**
```
- НЕ применять гепатопротекторы как основное/единственное решение
- НЕ использовать симптоматическую терапию в изоляции
- Приоритет: коррекция системы (рацион, менеджмент)
- Разделение групп ДО/сухостойных
- Снижение энергетической плотности ДО с увеличением объёма
```

**Механизм (BECAUSE):**
```
Метаболический дефицит не устранён — субстраты уходят на:
1. Базальные функции (поддержание температуры, жизнедеятельность)
2. Продукцию молока (лактационный синтез)
3. Компенсацию энергетического дефицита

Печень не имеет ресурсов для детоксикации через:
- Глюконеогенез (нет субстрата)
- Уреогенез (переваривание аммиака)
- Желчный синтез

Гепатопротекторы без устранения дефицита — симптоматическое 
лечение без адресации первопричины.
```

## Вердикт

**Решение:**
При BHB > 1.2 + низкий DMI + потеря BCS → отказ от стандартной 
симптоматической терапии в пользу системной коррекции.

**Альтернативы, которые были:**
| Вариант | Почему отклонён |
|---------|-----------------|
| Продолжить гепатопротекторы | Нет эффекта в CASE-001 (10 дней, BHB 1.6→1.4) |
| Увеличить дозу/длительность | Экономически неэффективно, не адресует причину |
| Добавить гормоны/стимуляторы | Риск углубления дефицита, побочные эффекты |
| Перевести в сухостой | Экономически невыгодно (2-я лактация, доильная) |

## Валидация

**Результат:** ✅ Подтвердилось

**Метрики CASE-001:**
- До: BHB 1.6, АСТ 142, DMI 14.2
- После: BHB 0.4, АСТ 89, DMI 19.2
- Прирост молока: +6 л/сут
- Экономия: отказ от неэффективной терапии

**Вывод:**
Стратегия работает. Порог BHB 1.2 — точка принятия решения о смене подхода.

## Ограничения (LIMITS)

**Где работает:**
- Ранний постпартум (<30 дней)
- Низкий DMI (<80% нормы)
- Потеря BCS
- Клинические признаки минимальны или отсутствуют

**Где НЕ работает / требует осторожности:**
- Выраженный клинический кетоз (ацетон в дыхании, атаксия) — требуется экстренная терапия
- Поздний постпартум (>60 дней) — другой механизм
- Сопутствующие инфекции — требуется комплексное лечение
- Тяжёлая печёночная дисфункция (АСТ >500) — первично поражение печени

**Требует проверки:**
- Другие породы (Holstein vs Jersey)
- Другие системы содержания (роботы vs привязь)
- Разный климат (жарко/холодно)
